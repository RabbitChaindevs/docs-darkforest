Быстрый старт за 5 минут

Предварительные требования​

  • Python 3.10+
  • Активная цивилизация Dark Forest
  • Приватный ключ вашего кошелька (для подписи транзакций агентом)

Шаг 1: Установка агента​

git clone https://github.com/dark-forest/agent.git
cd agent
pip install -r requirements.txt

Шаг 2: Импорт приватного ключа​

python -m dark_forest.cli key import

Приватный ключ хранится в зашифрованном виде, никогда не в открытом тексте.

⚠️ Настоятельно рекомендуется использовать выделенный игровой кошелёк — не используйте ваш основной кошелёк со значительными активами.

Шаг 3: Авторизация агента​

В интерфейсе игры установите разрешения для адреса кошелька вашего агента:

  • Отметьте необходимые разрешения операций
  • Установите политические ограничения (частота атак, резервы ресурсов и т.д.)

Шаг 4: Выбор стратегии​

# Список доступных стратегий
python -m dark_forest.cli strategy list

# Использование сбалансированной стратегии (рекомендуется)
python -m dark_forest.cli strategy select balanced

Четыре предустановленные стратегии:

СтратегияСтильЛучше всего для
aggressiveАгрессивный атакующийОчень активные игроки
balancedСбалансированная атака/защитаРекомендуется, подходит большинству игроков
defensiveКонсервативная защитаИгроки, часто находящиеся офлайн
farmerФокус на сбореМирное развитие

Шаг 5: Запуск​

# Запуск (по умолчанию сбалансированная стратегия)
python -m dark_forest.cli run

# Указать стратегию
python -m dark_forest.cli run --strategy aggressive

Шаг 6: Настройка уведомлений (опционально)​

Telegram​

python -m dark_forest.cli config set notify.telegram.token ВАШ_ТОКЕН_БОТА
python -m dark_forest.cli config set notify.telegram.chat_id ВАШ_ID_ЧАТА

Discord​

python -m dark_forest.cli config set notify.discord.webhook ВАШ_URL_ВЕБХУКА

Напоминания по безопасности​

  1. 🔐 Используйте выделенный игровой кошелёк с разумными лимитами резерва DFT/энергии
  2. 📉 Открывайте разрешения постепенно — начинайте только с COLLECT + REPAIR, затем добавьте ATTACK, когда убедитесь в стабильности
  3. 🔄 Регулярно проверяйте — просматривайте журналы и отчёты агента
