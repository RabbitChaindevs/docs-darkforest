# DFT 토큰 개요

**DFT (Dark Forest Token)**는 Dark Forest 생태계의 핵심 토큰으로, 표준 ERC-20 토큰으로 구현되었습니다.

---

## 기본 정보

| 속성 | 값 |
|------|-----|
| 컨트랙트 주소 | `0x1266e922fe34459efda34e7ee5caf327fbf138d7` |
| 토큰 이름 | DarkForest |
| 심볼 | DFT |
| 소수점 | 18 |
| 총 공급량 상한 | **4,206,900,000,000 DFT** (~4.2조) |
| 배분 기간 | ~10년 (3,650일) |

---

## DFT 사용처

| 사용처 | 설명 |
|-------|------|
| 🚀 **스페이스 점프** | `10 + 10 × √(점프 횟수)` DFT, 상한 1,000 |
| 🔧 **시스템 업그레이드** | 앵커 공식 기반 업그레이드 비용 |
| 🔄 **문명 재건** | `100 + 10 × 환생 횟수` DFT, 상한 2,000 |
| 🏛️ **동맹 탈퇴 벌금** | `100 + 10 × 멤버 수` DFT |
| ⚡ **에너지 시장** | 오더북을 통해 DFT로 에너지 구매 |

---

## 토크노믹스

### 공급량

- **하드 캡**: 4,206,900,000,000 DFT (~4.2조)
- **일일 발행량**: `4,206,900,000,000 / 3,650 ≈ 1,152,575,342 DFT` (일정한 일일 발행)
- **일일 배분**: DailyMinter를 통해 활성 플레이어에게 배분

### 거래세

DEX(PancakeSwap)에서 DFT가 거래될 때:

| 세금 | 비율 | 목적 |
|-----|------|------|
| 개발 수수료 | 1% | 팀 |
| 마케팅 수수료 | 1.5% | 마케팅 |
| **총 거래세** | **2.5%** | |

세금은 **등록된 유동성 풀**(LP)에서 거래할 때만 적용됩니다. 일반 전송에는 세금이 부과되지 않습니다.

## 지갑에서 DFT 확인

MetaMask에 DFT를 추가하려면:

1. MetaMask 열기 → **"Import Tokens"**
2. 컨트랙트 주소: `0x1266e922fe34459efda34e7ee5caf327fbf138d7`
3. 심볼이 **DFT**로 자동 표시됩니다
4. **"Add"**를 클릭하세요
