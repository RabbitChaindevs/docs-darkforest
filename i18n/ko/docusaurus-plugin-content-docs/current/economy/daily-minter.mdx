# 데일리 민트

**DailyMinter** 컨트랙트는 매일 고정된 양의 DFT를 발행하여 활성 문명들에게 균등하게 배분합니다. **당일에 수령하지 않으면 소멸됩니다** — 미수령 금액은 누적되지 않습니다.

---

## 수령 방법

### 요구사항

- 수령일 **UTC 자정 이전**에 문명을 생성해야 함
- 해당 일에 `distribute()`가 호출되어야 함 (누구나 호출 가능)
- 각 주소는 하루에 한 번만 수령 가능

### 작동

```
claim()
```

트랜잭션을 확인하세요 (가스비만 필요).

---

## 발행 흐름

### 1단계: 배분 (Distribute)

누구나 `distribute()`를 호출할 수 있습니다:

1. 컨트랙트가 마지막으로 배분되지 않은 에포크를 확인
2. 현재 활성 문명 수를 스냅샷
3. `플레이어당 지분 = 일일 발행량 / 활성 문명 수` 계산
4. 총 DFT를 컨트랙트로 발행
5. `일일 발행량 = 1,152,575,342 DFT` (고정)

### 2단계: 수령 (Claim)

플레이어가 `claim()`을 호출합니다:

1. 이 에포크 이전에 문명을 생성했는지 확인
2. 이미 수령하지 않았는지 확인
3. 지분을 지갑으로 전송

### 3단계: 잔여분 소각

다음 `distribute()` 호출 시, 이전 에포크에서 수령되지 않은 DFT는 **소각**됩니다 (유통 공급량 감소).

---

## 타임라인

```
에포크 0      에포크 1      에포크 2
   |            |            |
UTC 00:00    UTC 00:00    UTC 00:00
   ↑문명 생성    ↑distribute  ↑distribute
                ↑claim       ↑claim
                              ↑미수령 소각
```

- 에포크는 UTC 자정을 기준으로 나뉩니다
- **가장 최근에 배분된** 에포크만 수령할 수 있습니다
- 에포크를 놓치면 영원히 수령할 수 없습니다

---

## 주요 파라미터

| 파라미터 | 값 |
|---------|-----|
| 일일 발행량 | **1,152,575,342 DFT** (고정) |
| 총 배분 기간 | **3,650일** (~10년) |
| 총 공급량 상한 | **4,206,900,000,000 DFT** (~4.2조) |
| 수령 기간 | 각 에포크 내 (일반적으로 1일) |

---

## 매일 수령해야 하는 이유

1. 🆓 **무료 DFT** — 가스비만 필요하며 매일 이용 가능
2. 📉 **미수령 시 소각** — 놓치면 사라집니다
3. 📈 **플레이어가 적을수록 1인당 지분 증가** — 활성 플레이어가 줄어들면 각자의 지분이 늘어납니다

> 💡 UTC 시간에 맞춰 매일 수령 알림을 설정하세요. 해당 일에 `distribute()`가 아직 호출되지 않았다면 직접 호출할 수 있습니다 — 가스비만 듭니다.
