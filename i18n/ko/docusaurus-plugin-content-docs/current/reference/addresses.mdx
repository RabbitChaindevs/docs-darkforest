## 컨트랙트 아키텍처

```
DarkForest (DELEGATECALL 진입점)
├── DELEGATECALL → DarkForestGameplay   (문명 생성, 시스템 업그레이드)
├── DELEGATECALL → DarkForestBattle      (공격, 방어막 수리, 방어막 재생)
├── DELEGATECALL → DarkForestMovement    (스페이스 점프, 추적 점프, 크루즈)
└── DELEGATECALL → DarkForestAdmin       (수집, 수리, 재건, 토템, 출금)

DarkForestStorage (공유 저장소 레이아웃)
→ 모든 위성 컨트랙트가 DarkForestStorage를 상속하여 일관된 레이아웃 유지
```
