# 토템 시스템

토템은 동맹 전체에 적용되는 버프 시스템입니다. 멤버들이 에너지를 기부하여 토템을 업그레이드하면 모든 멤버가 방어 보너스를 받습니다.

---

## 작동 방식

1. **에너지 기부**: 동맹 멤버들이 토템 풀에 에너지를 기부합니다
2. **임계치 도달 시 업그레이드**: 토템의 에너지가 업그레이드 비용에 도달하면 리더가 업그레이드를 실행합니다
3. **보너스 수령**: 모든 멤버가 방어 보너스를 얻습니다

---

## 에너지 기부

### 작동

```
donateToTotem(allianceId, amount)
```

- 각 기부는 기부 타이머를 초기화합니다 (48시간 만료)
- 유효한 기부를 한 멤버만 토템 보너스의 혜택을 받습니다

### 기부 요구사항

- 각 기부는 최소 간격을 두고 이루어져야 합니다 (스팸 방지)
- 기부는 48시간 후 만료됩니다 (활성 상태를 유지하려면 재기부 필요)

---

## 토템 업그레이드

### 비용 공식

```
다음 업그레이드 비용 = 20,000,000 × N × (N + 5) / 10
여기서 N = 현재 토템 레벨 + 1
```

| 레벨 | 필요 에너지 |
|------|-----------|
| Lv 0 → 1 | 12,000,000 |
| Lv 1 → 2 | 42,000,000 |
| Lv 2 → 3 | 96,000,000 |
| Lv 10 → 11 | 330,000,000 |

### 작동

**동맹 리더**만 업그레이드를 실행할 수 있습니다:

```
upgradeTotem(allianceId)
```

업그레이드는 토템 풀에서 에너지를 소비하고 토템 레벨을 1 증가시킵니다.

---

## 토템 효과

```
동맹 방어 보너스 = (전체 멤버 수 - 1) × 8 × (1 + 토템 레벨 × 0.5%)
```

| 토템 레벨 | 보너스 계수 |
|-----------|------------|
| Lv 0 | 0% (보너스 없음) |
| Lv 10 | +5% |
| Lv 20 | +10% |
| Lv 100 | +50% |

예: 레벨 20 토템을 가진 50명 동맹:
```
보너스 = (50-1) × 8 × (1 + 20 × 0.5%) = 392 × 1.1 = 431.2 방어력
```

## 전략 팁

1. **멤버는 정기적으로 기부하세요** — 소액의 잦은 기부가 한 번에 큰 금액을 기부하는 것보다 효과적입니다
2. **리더는 신속히 업그레이드하세요** — 토템 풀이 임계치에 도달하자마자 업그레이드하세요
3. **높은 토템 레벨 + 많은 동맹원 = 상당한 방어 이점**
