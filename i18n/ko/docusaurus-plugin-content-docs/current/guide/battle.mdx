# 공격 및 방어

## 공격 조건

### 누구를 공격할 수 있나요?

- 대상이 존재하고 파괴되지 않은 상태여야 함 (폐허 상태가 아님)
- 대상이 **24시간 신규 플레이어 보호** 기간 중이 아니어야 함
- 대상이 **레이더 스캔 범위** 내에 있어야 함 (거리 ≤ 레이더 범위)
- 자신을 공격할 수 없음
- 같은 동맹의 멤버를 공격할 수 없음 (먼저 탈퇴해야 함)
- 지난 **24시간** 이내에 동맹을 탈퇴한 사람을 공격할 수 없음 (상호 쿨다운)

### 소프트 게이트

공격 조건이 충족되지 않으면, 시도는 **자원을 소모하지 않으며** — 공격이 실패한 이유를 알려줍니다. 에너지 낭비 걱정 없이 자유롭게 시도해볼 수 있습니다.

---

## 공격 비용

성공적인 공격은 **초당 1회**로 제한되며, 다음을 소모합니다:

| 자원 | 비용 |
|------|------|
| **에너지** | `50,000 + 50,000 × 무기 레벨` |
| **무기 내구도** | 1포인트 (무기 내구도 > 0일 때) |
| **공격 토큰** | 1토큰 |

---

## 공격 토큰 시스템

공격 토큰은 **토큰 버킷 알고리즘**을 사용하여 공격 빈도를 제한합니다:

| 파라미터 | 공식 | 설명 |
|---------|------|------|
| **최대 토큰** | `3 + 무기 레벨 / 10`, 상한 **10** | 시간이 지남에 따라 자동 회복 |
| **회복 간격** | `(300 - 무기 레벨 × 10) / 100`초, 최소 **1초** | 간격당 1토큰 |
| **비용** | 공격당 1토큰 | |

> 💡 무기 레벨이 높을수록 공격 토큰 회복 속도가 빠르고 최대 토큰 수가 높아집니다.

## 데미지 계산

### 공격자

```
기본 공격력 = 900 + 10 × 공격자 무기 레벨²
무기 내구도가 0이면 공격력 절반
최종 공격력 = 기본 공격력 × (1 + 전투 부스트 / 10000)
```

### 방어자

```
기본 방어력 = 540 + 6 × 방어자 방어막 레벨²
동맹 방어 보너스 = (동맹원 수 - 1) × 8 × (1 + 토템 레벨 × 0.5%)
최종 방어력 = 기본 방어력 × (1 + 전투 부스트 / 10000) + 동맹 보너스
```

### 최종 데미지

```
방어막 데미지 = 공격력 + 200 - 방어력 (0보다 클 경우)
체력 데미지 = (공격력 - 방어력) × 2 (0보다 클 경우)

방어막 데미지가 남은 방어막 HP보다 클 경우:
  오버플로우 × 3이 체력 데미지로 전환
```

## 약탈

전투에서 승리한 후, 대상의 에너지를 약탈합니다:

```
약탈 비율 = 5% + 무기 레벨 × 0.5%
약탈된 에너지 = 대상의 현재 에너지 × 약탈 비율
```

약탈된 에너지는 **보류 중인 전투 에너지** 풀에 저장되며, 수동으로 청구해야 합니다.

---

## 패배 및 파괴

### 패배

대상의 체력이 0이 되면:

1. **남은 모든 에너지가 약탈**되어 보류 풀에 저장됩니다
2. 대상의 무작위 시스템 하나가 **다운그레이드**됩니다 (BSC: 가장 높은 레벨의 시스템 다운그레이드; 비콘 체인 랜덤니스가 있는 체인: 수집기/무기/방어막/엔진 중 무작위)
3. 다운그레이드된 시스템의 레벨이 여전히 ≥ 1이면 문명은 생존하고 내구도가 50%로 복구됩니다
4. 모든 시스템이 레벨 1이면 (더 이상 다운그레이드 불가), 문명은 **폐허**가 됩니다
5. 처치 약탈은 일반 공격 약탈과 **중첩**됩니다

### 폐허

폐허가 된 문명은:
- 활성 문명 목록에서 제거됩니다
- 공격할 수 없습니다
- **재건** 기능을 통해 부활할 수 있습니다

### 전투 부스트 (환생 부스트)

재건할 때마다 **+1200 bps** (12%) 전투 부스트가 부여되며, 최대 **30000 bps** (3배)까지 중첩됩니다.
공격과 방어 모두 이 부스트의 혜택을 받습니다.

## 방어막 재생

### 자동 재생

```
재생량 = 50 + 방어막 레벨²
에너지 비용 = 재생량 (1:1 비율)
```

### 수동 수리

```
에너지 비용 = 수리량 × 4
```

### 동맹 지원

```
에너지 비용 = 수리량 × 1 (동맹이 지불)
동맹원끼리 서로 방어막을 수리할 수 있음
```

---

## 방어 전략

1. **에너지 보유량을 일일 유지비의 7배 이상 유지** — 방어막 재생을 중단시키는 에너지 고갈 방지
2. **정기적으로 내구도 수리** — 내구도가 0인 시스템은 매우 취약함
3. **방어막 레벨 업그레이드** — 방어막 HP와 방어력 모두 레벨 제곱에 비례
4. **동맹에 가입하고 토템 업그레이드** — 동맹 방어 보너스가 상당함
