# 첫 번째 문명 생성하기

## 사전 요구사항

- MetaMask가 설치되어 있고 BSC 메인넷에 연결되어 있어야 함
- 지갑에 **0.01 ~ 0.05 BNB** 보유 (입장료, 최소값에서 최대값까지 시간에 따라 선형 증가)
- 최소 **1 DFT** 보유 (이후 업그레이드용)

## 생성 절차

### 1단계: 게임 접속

**[darkforest.uk](https://darkforest.uk)**를 열고 우측 상단의 **"Connect Wallet"**을 클릭하세요.

### 2단계: 문명 이름 정하기

문명 이름은 **1~32바이트**로, 문자, 숫자, 기호를 지원합니다.

### 3단계: 초대 코드 입력 (선택 사항)

초대자의 지갑 주소가 있다면 여기에 입력할 수 있습니다. 본인과 초대자 모두 **150 에너지**를 보상으로 받습니다.

### 4단계: 생성 확인

**"Create Civilization"**을 클릭하면 MetaMask가 트랜잭션 확인 창을 띄웁니다. 트랜잭션에는 다음이 포함됩니다:

- **입장료**: 0.01 BNB (최소, 365일 동안 0.05 BNB까지 선형 증가)
- **가스비**: 소량의 BNB

확인하고 트랜잭션이 온체인에서 완료될 때까지 기다리세요 (약 3~5초).

## 초기 상태

생성 성공 후 문명은 다음 상태로 시작합니다:

| 속성 | 초기값 |
|------|--------|
| 에너지 | **2,000** |
| 체력 | **3,000** |
| 스캔 범위 | **1,000** |
| 모든 시스템 레벨 | **Lv 1** (수집기, 무기, 방어막, 레이더, 엔진) |
| 방어막 HP | **3,615** (3600 + 15 × 1²) |
| 위치 | 결정론적 좌표 (주소와 논스에서 파생) |

## 좌표 할당

문명은 **결정론적 좌표** `(x, y, z)`를 할당받으며, 이는 지갑 주소, 생성 시간, 총 문명 수의 해시를 통해 계산됩니다. 각 문명의 좌표는 고유합니다.

## 신규 플레이어 보호

> 🛡️ 새로 생성된 문명은 **24시간** 보호 기간이 있습니다. 이 기간 동안 다른 플레이어가 공격할 수 없습니다.
