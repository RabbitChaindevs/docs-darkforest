# 지갑 설치하기

Dark Forest를 플레이하려면 먼저 디지털 자산을 관리하고 온체인 트랜잭션에 서명할 수 있는 Web3 지갑이 필요합니다.

## Web3 지갑이란?

Web3 지갑은 탈중앙화 애플리케이션(dApp)에 접근하는 열쇠입니다. 지갑을 통해 다음을 할 수 있습니다:

- **디지털 자산 보관 및 관리** — DFT, BNB 등
- **트랜잭션 서명** — 게임 내 모든 행동(건설, 이동, 전투)에는 지갑 서명이 필요합니다
- **신원 인증** — 지갑 주소가 Dark Forest에서의 신원입니다

## 추천 지갑

### MetaMask (추천)

MetaMask는 가장 인기 있는 Web3 지갑으로, 브라우저 확장 프로그램 및 모바일 앱으로 제공됩니다.

#### 브라우저 설치

1. [MetaMask 웹사이트](https://metamask.io)를 방문하세요
2. "Download"를 클릭하여 브라우저 확장 프로그램을 다운로드하세요
3. 브라우저를 선택하세요 (Chrome / Firefox / Edge / Brave)
4. "Add to Browser"를 클릭하세요
5. 안내에 따라 새 지갑을 생성하세요

:::tip
설치 후, 빠른 접근을 위해 MetaMask 확장 프로그램을 브라우저 툴바에 고정하세요.
:::

#### 지갑 생성

1. MetaMask 아이콘을 클릭하고 "Create a Wallet"을 선택하세요
2. **강력한 비밀번호**를 설정하세요 (최소 8자, 문자와 숫자 조합)
3. **시드 구문을 안전하게 백업하세요** — 이것이 지갑을 복구할 수 있는 유일한 방법입니다!

:::danger 시드 구문 보안
- **절대** 시드 구문을 타인과 공유하지 마세요
- 스크린샷을 찍거나 클라우드에 저장하지 **마세요**
- 물리적 매체(종이와 펜)를 사용하여 오프라인 백업하세요
- 시드 구문을 묻는 사람은 모두 사기꾼입니다
:::

4. 시드 구문을 확인하여 지갑 생성을 완료하세요

### Trust Wallet

Trust Wallet은 바이낸스의 공식 모바일 지갑입니다.

1. App Store 또는 Google Play에서 "Trust Wallet"을 검색하여 설치하세요
2. 앱을 열고 "Create a New Wallet"을 선택하세요
3. 시드 구문을 안전하게 백업하세요
4. 지갑 내 DApp 브라우저를 활성화하세요
