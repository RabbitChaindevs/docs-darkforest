# توثيق Dark Forest

**Dark Forest** هي لعبة استراتيجية حضارات بين النجوم (SLG) ثلاثية الأبعاد تعمل بالكامل على السلسلة، منشورة على **BNB Smart Chain**. جميع منطق اللعبة يعمل على العقود الذكية، مع بنية DELEGATECALL للترقيات المعيارية.
