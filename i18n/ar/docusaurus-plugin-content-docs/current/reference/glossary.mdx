# قاموس المصطلحات

| المصطلح | الوصف |
|------|-------------|
| **الحضارة (Civilization)** | هوية اللاعب داخل اللعبة بمستوياتها وخصائصها وإحداثياتها الخاصة |
| **الطاقة (Energy)** | المورد الأساسي داخل اللعبة المستخدم في الترقيات والهجمات والحركة والإصلاحات |
| **DFT** | رمز Dark Forest، رمز ERC-20 يستخدم للقفزات والترقيات وغرامات الخروج من التحالف والمزيد |
| **الإحداثيات (Coordinates)** | موقع الحضارة في الفضاء ثلاثي الأبعاد `(x, y, z)`، يتم تعيينه عبر خوارزمية حتمية |
| **المستوى الإجمالي (Total Level)** | مجموع مستويات جميع الأنظمة؛ يؤثر على تكاليف الصيانة اليومية |
| **نطاق المسح (Scan Range)** | نصف قطر الكشف الخاص بك: `1000 + 150×lv + 5×lv²`؛ يحدد من يمكنك مهاجمته أو تعقبه |
| **قوة الهجوم (Attack Power)** | `900 + 10 × المستوى²`؛ السمة الأساسية لنظام السلاح |
| **الدفاع (Defense)** | `540 + 6 × المستوى²`؛ السمة الأساسية لنظام الدرع |
| **نقاط حياة الدرع (Shield HP)** | `3600 + 15 × المستوى²`؛ يمتص الضرر الوارد |
| **سرعة المحرك (Engine Speed)** | `10 + 5 × المستوى`؛ سرعة الانطلاق |
| **معدل الجمع (Collection Rate)** | `3 + 10 × √(المستوى - 1)` طاقة/ثانية |
| **المتانة (Durability)** | حالة عمل كل نظام، تُستهلك بمرور الوقت أو من خلال الاستخدام |
| **الصيانة اليومية (Daily Upkeep)** | `2,000 × المستوى الإجمالي / يوم`؛ تُخصم بشكل متناسب مع مرور الوقت |
| **رسوم الدخول (Entry Fee)** | BNB تُدفع عند إنشاء حضارة، تتراوح من 0.01 إلى 0.05 BNB بنمو خطي |
| **معادلة المرساة (Anchor Formula)** | معادلة حساب تكاليف الترقية بناءً على الإنتاج اليومي وأيام المرساة |
| **أيام المرساة (Anchor Days)** | المستوى 1–100: 1–30 يوماً؛ 100–1000: 30–60 يوماً؛ 1000+: 60 يوماً (الحد الأقصى) |
| **الانطلاق (Cruise)** | طريقة الحركة القياسية — التحرك نحو إحداثيات الهدف بسرعة ثابتة |
| **القفز الفضائي (Space Jump)** | انتقال فوري إلى إحداثيات جديدة، يكلف طاقة + DFT، تنخفض الدرع إلى 10% |
| **القفز التعقبي (Tracking Jump)** | قفز فوري إلى موقع الهدف؛ يتطلب رادار Lv20+ والهدف ضمن النطاق |
| **رمز الهجوم (Attack Token)** | خوارزمية دلو الرموز للحد من تواتر الهجوم؛ الحد الأقصى 3–10 رموز |
| **نسبة النهب (Plunder Ratio)** | `5% + مستوى السلاح × 0.5%`؛ جزء من طاقة الهدف يُنهب عند النصر |
| **حماية اللاعب الجديد (New Player Protection)** | لا يمكن الهجوم عليك خلال أول 24 ساعة بعد الإنشاء |
| **الخراب (Ruins)** | حضارة لا يمكن تخفيض مستواها أكثر بعد الهزيمة؛ يمكن إحياؤها عبر إعادة البناء |
| **إعادة البناء / الولادة الجديدة (Rebuild / Rebirth)** | إحياء حضارة مدمرة، يكلف طاقة + DFT، يمنح تعزيز قتال |
| **تعزيز القتال (Combat Boost)** | +12% لكل إعادة بناء، الحد الأقصى 300% (30000 نقطة أساس) |
| **التحالف (Alliance)** | تنظيم لاعبين للتعاون، الحد الأقصى 100 عضو |
| **فترة تبريد المغادرة (Leave Cooldown)** | فترة عدم اعتداء متبادلة لمدة 24 ساعة بعد مغادرة التحالف |
| **الطوطم (Totem)** | نظام تعزيز على مستوى التحالف؛ تبرع بالطاقة للترقية للحصول على مكافآت دفاعية |
| **الحقبة (Epoch)** | وحدة وقت DailyMinter، من منتصف الليل UTC إلى منتصف الليل UTC التالي |
| **DELEGATECALL** | نمط بنية العقود حيث يفوض العقد الرئيسي المنطق إلى عقود تابعة |
| **البوابة اللينة (Soft Gate)** | عندما لا تستوفى شروط الهجوم، تظهر اللعبة السبب دون استهلاك موارد |
