## بنية العقود

```
DarkForest (مدخل DELEGATECALL)
├── DELEGATECALL → DarkForestGameplay   (إنشاء حضارة، ترقية الأنظمة)
├── DELEGATECALL → DarkForestBattle      (هجوم، إصلاح الدرع، تجديد الدرع)
├── DELEGATECALL → DarkForestMovement    (قفز فضائي، قفز تعقبي، انطلاق)
└── DELEGATECALL → DarkForestAdmin       (جمع، إصلاح، إعادة بناء، طوطم، سحب)

DarkForestStorage (تخطيط تخزين مشترك)
→ جميع العقود التابعة ترث DarkForestStorage لتخطيط متسق
```
