# ما هو الوكيل الذكي

الوكيل الذكي (AI Agent) هو **برنامج آلي خارج السلسلة** يقوم بعمليات اللعبة نيابة عن حضارتك. يتم تفويضه من خلال **قناع أذونات (permission bitmask)** ويعمل ضمن **قيود السياسات**.

---

## ما يمكن للوكيل فعله

يتم التحكم في قدرات الوكيل بواسطة **الأذونات المصرح بها**:

| الإذن | القيمة | الإجراء |
|-----------|-------|--------|
| COLLECT | 1 | جمع الطاقة |
| REPAIR | 2 | إصلاح الأنظمة |
| CLAIM_COMBAT | 8 | المطالبة بطاقة القتال |
| ATTACK | 32 | الهجوم |
| UPGRADE | 64 | ترقية الأنظمة |
| JUMP | 128 | القفز الفضائي |
| REBUILD | 256 | إعادة بناء الحضارة |
| DONATE | 512 | التبرع للطوطم |

يمكنك منح أي مجموعة من الأذونات للوكيل (قناع بت).

---

## قيود السياسات

بالإضافة إلى الأذونات، يمكنك تعيين **قيود سياسات** للوكيل:

| القيد | الوصف |
|-----------|-------------|
| `minEnergyReserve` | الحد الأدنى لاحتياطي الطاقة (لن ينفق الوكيل أقل من هذا) |
| `minDftReserve` | الحد الأدنى لاحتياطي DFT |
| `maxAttackPerHour` | الحد الأقصى للهجمات في الساعة |
| `maxUpgradePerDay` | الحد الأقصى للترقيات في اليوم |
| `maxSingleUpgradeCost` | الحد الأقصى لتكلفة DFT لكل ترقية |
| `autoCollect` | الجمع التلقائي للطاقة |
| `autoRepair` | الإصلاح التلقائي للأنظمة |
| `autoShieldRepair` | الإصلاح التلقائي للدرع |
| `autoUpgrade` | الترقية التلقائية |
| `autoAttack` | الهجوم التلقائي |

## آليات الأمان

### 1. التسجيل على السلسلة

سياسة الوكيل مخزنة في عقد `DarkForestAgentRegistry.sol` ولا يمكن التلاعب بها بمجرد تعيينها.

### 2. فصل الأذونات

- **التفويض**: تفوض الوكيل للتصرف نيابة عنك
- **قناع الأذونات**: تحكم دقيق في كل عملية
- **قابل للإلغاء**: `revokeAgent()` يسري مفعوله فوراً

### 3. قواطع الدائرة

```
حد تواتر الهجوم: الحد الأقصى N في الساعة
حد تواتر الترقية: الحد الأقصى N في اليوم
الحد الأدنى للطاقة: لا تقل عن X
الحد الأدنى لـ DFT: لا يقل عن Y
```

### 4. القائمة السوداء / القائمة البيضاء

- **القائمة السوداء**: منع مهاجمة لاعبين محددين
- **القائمة البيضاء**: السماح فقط بمهاجمة لاعبين محددين (يتم تجاهل القائمة السوداء عند تنشيط القائمة البيضاء)
