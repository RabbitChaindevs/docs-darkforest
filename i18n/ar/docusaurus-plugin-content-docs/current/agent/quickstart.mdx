# بداية سريعة في 5 دقائق

## المتطلبات الأساسية

- Python 3.10+
- حضارة Dark Forest نشطة
- المفتاح الخاص لمحفظتك (لكي يوقع الوكيل المعاملات)

---

## الخطوة 1: تثبيت الوكيل

```bash
git clone https://github.com/dark-forest/agent.git
cd agent
pip install -r requirements.txt
```

## الخطوة 2: استيراد مفتاحك الخاص

```bash
python -m dark_forest.cli key import
```

يتم تخزين المفتاح الخاص مشفراً، وليس كنص عادي أبداً.

> ⚠️ يُوصى بشدة باستخدام **محفظة ألعاب مخصصة** — لا تستخدم محفظتك الرئيسية التي تحتوي على أصول كبيرة.

## الخطوة 3: تفويض الوكيل

في واجهة اللعبة، قم بتعيين الأذونات لعنوان محفظة الوكيل:

- حدد أذونات التشغيل التي يحتاجها الوكيل
- عيّن قيود السياسات (تواتر الهجوم، احتياطيات الموارد، إلخ.)

## الخطوة 4: اختيار استراتيجية

```bash
# عرض الاستراتيجيات المتاحة
python -m dark_forest.cli strategy list

# استخدام الاستراتيجية المتوازنة (موصى بها)
python -m dark_forest.cli strategy select balanced
```

أربع استراتيجيات مسبقة:

| الاستراتيجية | الأسلوب | الأنسب لـ |
|----------|-------|----------|
| `aggressive` | هجومي | اللاعبون النشطون جداً |
| `balanced` | هجوم/دفاع متوازن | **موصى بها**، تناسب معظم اللاعبين |
| `defensive` | دفاعي محافظ | اللاعبون غير المتصلين غالباً |
| `farmer` | التركيز على الجمع | التطوير السلمي |

## الخطوة 5: التشغيل

```bash
# بدء (الاستراتيجية المتوازنة افتراضياً)
python -m dark_forest.cli run

# تحديد استراتيجية
python -m dark_forest.cli run --strategy aggressive
```

## الخطوة 6: تكوين الإشعارات (اختياري)

### Telegram

```bash
python -m dark_forest.cli config set notify.telegram.token YOUR_BOT_TOKEN
python -m dark_forest.cli config set notify.telegram.chat_id YOUR_CHAT_ID
```

### Discord

```bash
python -m dark_forest.cli config set notify.discord.webhook YOUR_WEBHOOK_URL
```

## تذكيرات الأمان

1. 🔐 **استخدم محفظة ألعاب مخصصة** مع حدود معقولة لاحتياطي DFT/الطاقة
2. 📉 **افتح الأذونات تدريجياً** — ابدأ بـ COLLECT + REPAIR فقط، ثم أضف ATTACK بمجرد التأكد من استقرارها
3. 🔄 **تحقق بانتظام** — راجع سجلات الوكيل وتقاريره

**الخطوة التالية الموصى بها: [قوالب الاستراتيجيات →](/docs/agent/strategies)**
