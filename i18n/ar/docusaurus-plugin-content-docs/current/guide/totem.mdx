# نظام الطوطم

الطوطم هو نظام تعزيز على مستوى التحالف. يتبرع الأعضاء بالطاقة لترقيته، مما يمنح مكافآت دفاعية لجميع الأعضاء.

---

## كيف يعمل

1. **تبرع بالطاقة**: يتبرع أعضاء التحالف بالطاقة لمجمع الطوطم
2. **التراكم حتى الحد**: بمجرد وصول طاقة الطوطم إلى تكلفة الترقية، يقوم القائد بتشغيل الترقية
3. **الحصول على المكافأة**: يحصل جميع الأعضاء على مكافأة دفاعية

---

## التبرع بالطاقة

### العملية

```
donateToTotem(allianceId, amount)
```

- كل تبرع يعيد ضبط مؤقت التبرع الخاص بك (تنتهي صلاحيته بعد 48 ساعة)
- فقط الأعضاء الذين لديهم تبرعات سارية يستفيدون من مكافأة الطوطم

### متطلبات التبرع

- يجب أن يكون كل تبرع مفصولاً بفاصل زمني أدنى (مكافحة البريد العشوائي)
- ينتهي التبرع بعد 48 ساعة (يجب إعادة التبرع للبقاء نشطاً)

---

## ترقية الطوطم

### معادلة التكلفة

```
تكلفة الترقية التالية = 20,000,000 × N × (N + 5) / 10
حيث N = مستوى الطوطم الحالي + 1
```

| المستوى | الطاقة المطلوبة |
|-------|----------------|
| المستوى 0 ← 1 | 12,000,000 |
| المستوى 1 ← 2 | 42,000,000 |
| المستوى 2 ← 3 | 96,000,000 |
| المستوى 10 ← 11 | 330,000,000 |

### العملية

فقط **قائد التحالف** يمكنه تشغيل الترقيات:

```
upgradeTotem(allianceId)
```

تستهلك الترقية الطاقة من مجمع الطوطم وتزيد مستوى الطوطم بمقدار 1.

---

## تأثير الطوطم

```
مكافأة دفاع التحالف = (إجمالي الأعضاء - 1) × 8 × (1 + مستوى الطوطم × 0.5%)
```

| مستوى الطوطم | معامل المكافأة |
|-------------|-------------------|
| المستوى 0 | 0% (لا مكافأة) |
| المستوى 10 | +5% |
| المستوى 20 | +10% |
| المستوى 100 | +50% |

مثال: تحالف من 50 عضواً مع طوطم مستوى 20:
```
المكافأة = (50-1) × 8 × (1 + 20 × 0.5%) = 392 × 1.1 = 431.2 دفاع
```

## نصائح استراتيجية

1. **يجب على الأعضاء التبرع بانتظام** — التبرعات الصغيرة المتكررة أكثر فعالية من مبلغ كبير واحد
2. **يجب على القادة الترقية فوراً** — قم بالترقية بمجرد وصول مجمع الطوطم إلى الحد المطلوب
3. **مستوى طوطم مرتفع + حلفاء كثيرون = ميزة دفاعية كبيرة**
