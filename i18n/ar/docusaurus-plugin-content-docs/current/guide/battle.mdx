# الهجوم والدفاع

## شروط الهجوم

### من يمكنك مهاجمته؟

- يجب أن يكون الهدف موجوداً وغير مدمر (ليس في حالة خراب)
- يجب ألا يكون الهدف في فترة **حماية اللاعب الجديد (24 ساعة)**
- يجب أن يكون الهدف ضمن **نطاق مسح الرادار الخاص بك** (المسافة ≤ نطاق الرادار الخاص بك)
- لا يمكنك مهاجمة نفسك
- لا يمكنك مهاجمة أعضاء تحالفك (يجب أن تغادر أولاً)
- لا يمكنك مهاجمة شخص غادر تحالفك في آخر **24 ساعة** (فترة تبريد متبادلة)

### البوابة اللينة (Soft Gate)

إذا لم يتم استيفاء شرط الهجوم، فإن المحاولة **لا تستهلك أي موارد** — بل تخبرك ببساطة لماذا فشل الهجوم. هذا يتيح لك التجربة بحرية دون القلق بشأن الطاقة المهدرة.

---

## تكلفة الهجوم

كل هجوم ناجح يقتصر على **هجوم واحد في الثانية**، ويستهلك:

| المورد | التكلفة |
|----------|------|
| **الطاقة** | `50,000 + 50,000 × مستوى السلاح` |
| **متانة السلاح** | نقطة واحدة (عندما تكون متانة السلاح > 0) |
| **رمز الهجوم** | رمز واحد |

---

## نظام رموز الهجوم

تستخدم رموز الهجوم **خوارزمية دلو الرموز (Token Bucket)** للحد من تواتر الهجوم:

| المعامل | المعادلة | الوصف |
|-----------|---------|-------------|
| **الحد الأقصى للرموز** | `3 + مستوى السلاح / 10`، الحد الأقصى **10** | يتعافى تلقائياً بمرور الوقت |
| **فترة الاسترداد** | `(300 - مستوى السلاح × 10) / 100` ثانية، الحد الأدنى **1ث** | رمز واحد لكل فترة |
| **التكلفة** | رمز واحد لكل هجوم | |

> 💡 كلما ارتفع مستوى سلاحك، زادت سرعة استرداد رموز الهجوم وارتفع الحد الأقصى.

## حساب الضرر

### المهاجم

```
الهجوم الأساسي = 900 + 10 × مستوى سلاح المهاجم²
إذا كانت متانة السلاح = 0، يتم تخفيض الهجوم إلى النصف
الهجوم النهائي = الهجوم الأساسي × (1 + تعزيز القتال / 10000)
```

### المدافع

```
الدفاع الأساسي = 540 + 6 × مستوى درع المدافع²
مكافأة دفاع التحالف = (الحلفاء - 1) × 8 × (1 + مستوى الطوطم × 0.5%)
الدفاع النهائي = الدفاع الأساسي × (1 + تعزيز القتال / 10000) + مكافأة التحالف
```

### الضرر النهائي

```
ضرر الدرع = الهجوم + 200 - الدفاع (إذا كان > 0)
ضرر الصحة = (الهجوم - الدفاع) × 2 (إذا كان > 0)

إذا كان ضرر الدرع > نقاط حياة الدرع المتبقية:
  الفائض × 3 يتحول إلى ضرر للصحة
```

## النهب

بعد الفوز في معركة، تنهب الطاقة من الهدف:

```
نسبة النهب = 5% + مستوى السلاح × 0.5%
الطاقة المنهوبة = طاقة الهدف الحالية × نسبة النهب
```

تذهب الطاقة المنهوبة إلى **مجموعة طاقة قتال معلقة**، والتي يجب عليك المطالبة بها يدوياً.

---

## الهزيمة والتدمير

### الهزيمة

عندما تصل صحة الهدف إلى الصفر:

1. **يتم نهب كل الطاقة المتبقية** في مجموعتك المعلقة
2. يتم **خفض مستوى نظام عشوائي** للهدف (على BSC: يخفض أعلى نظام مستوى؛ على السلاسل مع عشوائية beacon chain: يختار عشوائياً من الجامع/السلاح/الدرع/المحرك)
3. إذا كان مستوى النظام المخفض لا يزال ≥ 1، تبقى الحضارة على قيد الحياة وتُستعاد متانتها إلى 50%
4. إذا كانت جميع الأنظمة في المستوى 1 (لا يمكن تخفيضها أكثر)، تصبح الحضارة **خراباً**
5. يتراكم نهب القتل **فوق** نهب الهجوم العادي

### الخراب

الحضارة في حالة خراب:
- تُزال من قائمة الحضارات النشطة
- لا يمكن مهاجمتها
- يمكن إحياؤها عبر وظيفة **إعادة البناء**

### تعزيز القتال (تعزيز إعادة الميلاد)

كل عملية إعادة بناء تمنح **+1200 نقطة أساس (12%)** تعزيز قتال، يتراكم حتى **30000 نقطة أساس (مضاعف 3×)**.
يستفيد كل من الهجوم والدفاع من هذا التعزيز.

## تجديد الدرع

### التجديد التلقائي

```
كمية التجديد = 50 + مستوى الدرع²
تكلفة الطاقة = كمية التجديد (نسبة 1:1)
```

### الإصلاح اليدوي

```
تكلفة الطاقة = كمية الإصلاح × 4
```

### المساعدة من التحالف

```
تكلفة الطاقة = كمية الإصلاح × 1 (يدفعها الحليف)
يمكن للحلفاء إصلاح الدروع لبعضهم البعض
```

## استراتيجية الدفاع

1. **احتفظ باحتياطي طاقة > 7 أضعاف الصيانة اليومية** — يمنع استنفاذ الطاقة الذي قد يوقف تجديد الدرع
2. **أصلح المتانة بانتظام** — الأنظمة ذات المتانة الصفرية ضعيفة للغاية
3. **ارفع مستوى الدرع** — نقاط حياة الدرع والدفاع يتناسبان مع مربع المستوى
4. **انضم إلى تحالف وارفع مستوى الطوطم** — مكافآت دفاع التحالف كبيرة
