# نظام الطاقة

الطاقة هي المورد الأكثر أساسية في Dark Forest. تقريباً كل عملية — الترقيات، الهجمات، الحركة، والإصلاحات — تتطلب طاقة.

---

## كسب الطاقة

### 1. جامع الطاقة (الإنتاج السلبي)

```
معدل الجمع = 3 + 10 × √(مستوى الجامع - 1)   طاقة/ث
```

لكل لاعب دعوته، يزداد المعدل أكثر.

**متانة الجامع**: يفقد الجامع نقطة متانة واحدة في الثانية.

| مستوى الجامع | الإنتاج في الثانية | أقصى متانة للجامع |
|----------------|-------------------|--------------------------|
| المستوى 1 | 3 | يوم واحد |
| المستوى 10 | ~33 | يوم واحد + 18 ساعة |
| المستوى 50 | ~73 | يوم واحد + 4 أيام |
| المستوى 100 | ~103 | يوم واحد + 8.3 أيام |

> ⚠️ عندما تنفد متانة الجامع، يتوقف الإنتاج. تحتاج إلى إصلاحه لاستئناف الجمع.

### 2. النهب القتالي

بعد الفوز في معركة ضد حضارة أخرى، تنهب جزءاً من طاقتهم:

```
نسبة النهب = 5% + مستوى السلاح × 0.5%
```

تذهب الطاقة المنهوبة إلى **مجموعة معلقة** (pendingCombatEnergy)، والتي يجب عليك المطالبة بها يدوياً.

### 3. مكافآت الدعوة

لكل لاعب جديد تدعوه بنجاح (يدخل عنوانك عند إنشاء حضارته):

- **أنت** تتلقى 150 طاقة
- **المدعو** يتلقى 150 طاقة

### 4. سوق الطاقة

انظر [سوق الطاقة](/docs/economy/energy-market).

## استهلاك الطاقة

| حالة الاستخدام | التكلفة |
|----------|------|
| **الصيانة اليومية** | `2,000 × المستوى الإجمالي` / يوم |
| **ترقيات الأنظمة** | معادلة المرساة (انظر دليل الحضارة) |
| **الهجوم** | `50,000 + 50,000 × مستوى السلاح` |
| **القفز الفضائي** | `200,000 + 200,000 × √(عدد القفزات)`، الحد الأقصى 16.5M |
| **إصلاح الجامع** | `1 طاقة / ثانية` |
| **إصلاح السلاح** | `3 طاقة / نقطة` |
| **إصلاح الدرع** | `4 طاقة / نقطة` |
| **إصلاح المحرك** | `5 طاقة / نقطة` |
| **تجديد الدرع** | `كمية التجديد × 1` (تجديد تلقائي) |
| **إصلاح الدرع يدوياً** | `كمية الإصلاح × 4` (إصلاح يدوي) |
| **مساعدة درع التحالف** | `كمية الإصلاح × 1` (يدفعها الحليف) |
| **التبرع للطوطم** | مبلغ مخصص |
| **إعادة البناء** | أساس 500,000، يزداد مع عدد الولادات الجديدة |

## إدارة الطاقة

### الصيانة اليومية

يتم خصم الصيانة بشكل متناسب مع مرور الوقت كلما قمت بأي إجراء:

```
الصيانة الحالية = 2,000 × المستوى الإجمالي × (الوقت منذ آخر إجراء / 86400)
```

### الطاقة المحجوزة

الطاقة الموضوعة في أوامر بيع في سوق الطاقة تكون مقفلة (`energyReserved`). تتوقف الصيانة عن الخصم بمجرد وصول `طاقة - محجوز` إلى الصفر.

### نصائح

1. **احتفظ بطاقة > صيانة شهر واحد** — يمنع النضوب المتكرر من الإجراءات الروتينية
2. **أولوية ترقية الجامع** — الجامع عالي المستوى هو مفتاح الدخل السلبي
3. **احسب التكاليف قبل الهجوم** — تأكد من أن النهب يستحق الاستثمار
4. **أصلح متانة الجامع بانتظام** — تجنب انقطاعات الجمع

**الخطوة التالية الموصى بها: [نظام التحالف →](/docs/guide/alliance)**
