# Documentation de Dark Forest

Dark Forest est un jeu de stratégie (SLG) de civilisation interstellaire 3D entièrement on-chain déployé sur **BNB Smart Chain**. Toute la logique du jeu fonctionne via des smart contracts, avec une architecture DELEGATECALL pour des mises à niveau modulaires.
