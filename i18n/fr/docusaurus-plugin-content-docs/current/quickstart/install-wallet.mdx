# Installer un portefeuille

Avant de pouvoir jouer à Dark Forest, vous avez besoin d'un portefeuille Web3 pour gérer vos actifs numériques et signer des transactions on-chain.

## Qu'est-ce qu'un portefeuille Web3 ?

Un portefeuille Web3 est votre clé pour les applications décentralisées (dApps). Il vous permet de :

- **Stocker et gérer** des actifs numériques comme DFT et BNB
- **Signer des transactions** — chaque action dans le jeu (construire, se déplacer, combattre) nécessite la signature de votre portefeuille
- **Authentifier votre identité** — l'adresse de votre portefeuille est votre identité dans Dark Forest

## Portefeuilles recommandés

### MetaMask (Recommandé)

MetaMask est le portefeuille Web3 le plus populaire, disponible en extension de navigateur et application mobile.

#### Installation sur navigateur

1. Visitez le [site web de MetaMask](https://metamask.io)
2. Cliquez sur "Télécharger" pour obtenir l'extension de navigateur
3. Choisissez votre navigateur (Chrome / Firefox / Edge / Brave)
4. Cliquez sur "Ajouter au navigateur"
5. Suivez l'assistant pour créer un nouveau portefeuille

:::tip
Après l'installation, épinglez l'extension MetaMask à la barre d'outils de votre navigateur pour un accès rapide.
:::

#### Création d'un portefeuille

1. Cliquez sur l'icône MetaMask et sélectionnez "Créer un portefeuille"
2. Définissez un **mot de passe fort** (au moins 8 caractères, mélange de lettres et de chiffres)
3. **Sauvegardez votre phrase de récupération en lieu sûr** — c'est le seul moyen de récupérer votre portefeuille !

:::danger Sécurité de la phrase de récupération
- **Ne partagez jamais** votre phrase de récupération avec quiconque
- **Ne faites pas** de captures d'écran et ne la stockez pas dans le cloud
- Utilisez un support physique (papier et stylo) pour la sauvegarde hors ligne
- Quiconque vous demande votre phrase de récupération est un arnaqueur
:::

4. Confirmez votre phrase de récupération pour terminer la création du portefeuille

### Trust Wallet

Trust Wallet est le portefeuille mobile officiel de Binance.

1. Recherchez "Trust Wallet" sur l'App Store ou Google Play et installez-le
2. Ouvrez l'application et sélectionnez "Créer un nouveau portefeuille"
3. Sauvegardez votre phrase de récupération en toute sécurité
4. Activez le navigateur DApp dans le portefeuille

## Prochaines étapes

Une fois votre portefeuille installé, passez à [Se connecter à BSC](/docs/quickstart/connect-bsc) et préparez-vous à entrer dans Dark Forest.
