# Obtenir des tokens DFT

**DFT (Dark Forest Token)** est le token ERC-20 central de Dark Forest, avec un plafond d'émission total d'environ **~4,2 billions (4 206 900 000 000)**.

**Utilisations du DFT :**
- 🚀 Sauts spatiaux (10–1 000 DFT par saut)
- 🔧 Améliorations de systèmes
- 🔄 Reconstruction d'une civilisation
- 🏛️ Amendes de sortie d'alliance
- ⚡ Transactions sur le marché de l'énergie

---

## Méthode 1 : Acheter sur un DEX

DFT est listé sur PancakeSwap.

### Étapes

1. Ouvrez [PancakeSwap](https://pancakeswap.finance/swap)
2. Connectez votre portefeuille
3. Dans le champ "Vers", entrez l'adresse du contrat DFT :

```
0x1266e922fe34459efda34e7ee5caf327fbf138d7
```

4. Entrez le montant et cliquez sur **"Swap"**

> 💡 **Note** : Les transactions LP sur PancakeSwap entraînent une **taxe de transaction de 2,5 %** (1 % de frais de développement + 1,5 % de frais de marketing).

---

## Méthode 2 : Minage quotidien (Gratuit)

Réclamez un montant gratuit de DFT chaque jour :

1. Ouvrez [darkforest.uk](https://darkforest.uk)
2. Connectez votre portefeuille
3. Cliquez sur **"Réclamation quotidienne"**
4. Si `distribute()` n'a pas encore été appelé pour la journée, vous pouvez le déclencher vous-même
5. Confirmez la transaction (frais de gaz uniquement)

> ⚠️ **Réclamez-le le jour même ou perdez-le.** Le DailyMinter n'accumule pas les DFT non réclamés — les restes sont brûlés lors de la distribution du jour suivant.

## Voir votre solde DFT

Pour ajouter DFT à MetaMask :

1. Ouvrez MetaMask → **"Importer des tokens"**
2. Adresse du contrat : `0x1266e922fe34459efda34e7ee5caf327fbf138d7`
3. Le symbole s'affichera automatiquement comme **DFT**, avec **18** décimales
4. Cliquez sur **"Ajouter"**
