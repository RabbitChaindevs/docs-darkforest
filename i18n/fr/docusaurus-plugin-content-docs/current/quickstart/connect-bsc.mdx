# Se connecter à BSC

Dark Forest est déployé sur **BNB Smart Chain (BSC)**. Vous devez basculer votre portefeuille sur le réseau BSC pour interagir avec le jeu.

## Qu'est-ce que BSC ?

BNB Smart Chain est une blockchain lancée par Binance, offrant :

- **Frais de transaction faibles** — chaque opération ne coûte que quelques fractions de centime
- **Débit élevé** — conçu pour des interactions de jeu à haute fréquence
- **Compatible EVM** — fonctionne avec MetaMask et d'autres portefeuilles courants

## Ajout automatique du réseau BSC

Le moyen le plus simple est d'utiliser Chainlist :

1. Visitez [Chainlist](https://chainlist.org)
2. Recherchez "BNB Smart Chain"
3. Cliquez sur "Connect Wallet" et autorisez
4. Cliquez sur "Add to MetaMask"
5. Confirmez l'ajout

## Configuration manuelle

Vous pouvez également ajouter le réseau BSC manuellement :

### Ajout de BSC à MetaMask

1. Cliquez sur le menu déroulant du réseau dans le coin supérieur droit de MetaMask
2. Cliquez sur "Ajouter un réseau"
3. Remplissez les informations suivantes :

| Paramètre | Valeur |
|-----------|--------|
| **Nom du réseau** | BNB Smart Chain |
| **URL RPC** | `https://bsc-dataseed.binance.org` |
| **ID de chaîne** | `56` |
| **Symbole de la monnaie** | `BNB` |
| **Explorateur de blocs** | `https://bscscan.com` |

4. Cliquez sur "Enregistrer"

### Nœuds RPC de secours

Si le RPC principal est congestionné, vous pouvez utiliser des nœuds de secours :

| Fournisseur | URL RPC |
|-------------|---------|
| Binance Officiel | `https://bsc-dataseed1.binance.org` |
| Ankr | `https://rpc.ankr.com/bsc` |
| PublicNode | `https://bsc.publicnode.com` |

## Obtenir du BNB

Chaque opération sur BSC nécessite du BNB pour les frais de gaz. Voici comment en obtenir :

1. **Échange centralisé** — achetez du BNB sur Binance, OKX, etc. et retirez-le sur le réseau BSC
2. **Pont inter-chaînes** — transférez du BNB d'autres réseaux vers BSC via un pont

:::caution
Lors du retrait, **assurez-vous de sélectionner le réseau BSC (BEP-20)** sinon vous pourriez perdre vos fonds !
:::

## Vérifier votre connexion

Après la configuration, basculez vers le réseau BNB Smart Chain dans votre portefeuille. Si l'ID de chaîne affiche `56` et que votre solde affiche le montant correct de BNB, tout est prêt.

Une fois votre portefeuille et le réseau prêts, la prochaine étape est d'[Obtenir des tokens DFT](/docs/quickstart/get-dft).
