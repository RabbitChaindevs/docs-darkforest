## Architecture des contrats

```
DarkForest (entrée DELEGATECALL)
├── DELEGATECALL → DarkForestGameplay   (créer civ, améliorer systèmes)
├── DELEGATECALL → DarkForestBattle      (attaquer, réparer bouclier, régén. bouclier)
├── DELEGATECALL → DarkForestMovement    (saut spatial, saut poursuite, croisière)
└── DELEGATECALL → DarkForestAdmin       (collecter, réparer, reconstruire, totem, retirer)

DarkForestStorage (disposition de stockage partagée)
→ Tous les contrats satellites héritent de DarkForestStorage pour une disposition cohérente
```
