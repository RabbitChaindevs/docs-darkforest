# Démarrage rapide en 5 minutes

## Prérequis

- Python 3.10+
- Une civilisation Dark Forest active
- La clé privée de votre portefeuille (pour que l'agent signe les transactions)

---

## Étape 1 : Installer l'agent

```bash
git clone https://github.com/dark-forest/agent.git
cd agent
pip install -r requirements.txt
```

## Étape 2 : Importer votre clé privée

```bash
python -m dark_forest.cli key import
```

La clé privée est stockée chiffrée, jamais en texte clair.

> ⚠️ Il est fortement recommandé d'utiliser un **portefeuille de jeu dédié** — n'utilisez pas votre portefeuille principal qui contient des actifs importants.

## Étape 3 : Autoriser l'agent

Dans l'interface du jeu, définissez les permissions pour l'adresse du portefeuille de votre agent :

- Cochez les permissions d'opération dont l'agent a besoin
- Définissez les contraintes de politique (fréquence d'attaque, réserves de ressources, etc.)

## Étape 4 : Choisir une stratégie

```bash
# Lister les stratégies disponibles
python -m dark_forest.cli strategy list

# Utiliser la stratégie équilibrée (recommandée)
python -m dark_forest.cli strategy select balanced
```

Quatre stratégies prédéfinies :

| Stratégie | Style | Idéal pour |
|-----------|-------|------------|
| `aggressive` | Attaquant agressif | Joueurs très actifs |
| `balanced` | Attaque/défense équilibrée | **Recommandée**, convient à la plupart des joueurs |
| `defensive` | Défense conservatrice | Joueurs souvent hors ligne |
| `farmer` | Concentré sur la collecte | Développement pacifique |

## Étape 5 : Lancer

```bash
# Démarrer (stratégie équilibrée par défaut)
python -m dark_forest.cli run

# Spécifier une stratégie
python -m dark_forest.cli run --strategy aggressive
```

## Étape 6 : Configurer les notifications (Optionnel)

### Telegram

```bash
python -m dark_forest.cli config set notify.telegram.token VOTRE_TOKEN_BOT
python -m dark_forest.cli config set notify.telegram.chat_id VOTRE_ID_CHAT
```

### Discord

```bash
python -m dark_forest.cli config set notify.discord.webhook VOTRE_URL_WEBHOOK
```

---

## Rappels de sécurité

1. 🔐 **Utilisez un portefeuille de jeu dédié** avec des limites raisonnables de réserve DFT/énergie
2. 📉 **Ouvrez les permissions graduellement** — commencez avec seulement COLLECT + REPAIR, puis ajoutez ATTACK une fois que vous êtes confiant de sa stabilité
3. 🔄 **Vérifiez régulièrement** — examinez les journaux et rapports de l'agent

**Prochaine étape recommandée : [Modèles de stratégie →](/docs/agent/strategies)**
