# Système d'énergie

L'énergie est la ressource la plus fondamentale dans Dark Forest. Presque toutes les opérations — améliorations, attaques, déplacements et réparations — nécessitent de l'énergie.

---

## Gagner de l'énergie

### 1. Collecteur d'énergie (Production passive)

```
Taux de collecte = 3 + 10 × √(niveau du collecteur - 1)   énergie/s
```

Pour chaque joueur que vous avez invité, le taux augmente davantage.

**Durabilité du collecteur** : Le collecteur perd 1 de durabilité par seconde.

| Niveau du collecteur | Production par seconde | Durabilité max du collecteur |
|---------------------|----------------------|------------------------------|
| Nv 1 | 3 | 1 jour |
| Nv 10 | ~33 | 1 jour + 18 heures |
| Nv 50 | ~73 | 1 jour + 4 jours |
| Nv 100 | ~103 | 1 jour + 8,3 jours |

> ⚠️ Quand la durabilité du collecteur s'épuise, la production s'arrête. Vous devez le réparer pour reprendre la collecte.

### 2. Pillage de combat

Après avoir gagné un combat contre une autre civilisation, vous pillez une partie de son énergie :

```
Ratio de pillage = 5 % + niveau d'arme × 0,5 %
```

L'énergie pillée va dans une **réserve en attente** (pendingCombatEnergy), que vous devez réclamer manuellement.

### 3. Récompenses d'invitation

Pour chaque nouveau joueur que vous invitez avec succès (il entre votre adresse lors de la création de sa civilisation) :

- **Vous** recevez 150 énergie
- **L'invité** reçoit 150 énergie

### 4. Marché de l'énergie

Voir [Marché de l'énergie](/docs/economy/energy-market).

## Dépenser de l'énergie

| Utilisation | Coût |
|-------------|------|
| **Entretien quotidien** | `2 000 × niveau total` / jour |
| **Améliorations de systèmes** | Formule d'ancrage (voir guide Civilisation) |
| **Attaque** | `50 000 + 50 000 × niveau d'arme` |
| **Saut spatial** | `200 000 + 200 000 × √(nombre de sauts)`, max 16,5 M |
| **Réparation du collecteur** | `1 énergie / seconde` |
| **Réparation de l'arme** | `3 énergie / point` |
| **Réparation du bouclier** | `4 énergie / point` |
| **Réparation du moteur** | `5 énergie / point` |
| **Régénération du bouclier** | `montant de régénération × 1` (régénération auto) |
| **Réparation du bouclier** | `montant de réparation × 4` (réparation manuelle) |
| **Assistance bouclier d'alliance** | `montant de réparation × 1` (payé par l'allié) |
| **Don au totem** | Montant personnalisé |
| **Reconstruction** | Base 500 000, augmente avec le nombre de renaissances |

## Gestion de l'énergie

### Entretien quotidien

L'entretien est déduit proportionnellement au fil du temps lorsque vous effectuez une action :

```
Entretien actuel = 2 000 × niveau total × (temps depuis la dernière action / 86400)
```

### Énergie réservée

L'énergie placée dans des ordres de vente sur le marché de l'énergie est verrouillée (`energyReserved`). L'entretien cesse d'être déduit une fois que `énergie - réservée` atteint zéro.

### Conseils

1. **Gardez l'énergie > 1 mois d'entretien** — évite un épuisement fréquent lors des actions de routine
2. **Priorisez les améliorations du collecteur** — un collecteur de haut niveau est la clé du revenu passif
3. **Calculez les coûts avant d'attaquer** — assurez-vous que le pillage vaut l'investissement
4. **Réparez régulièrement la durabilité du collecteur** — évitez les interruptions de collecte
