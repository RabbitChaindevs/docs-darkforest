# Aperçu du token DFT

**DFT (Dark Forest Token)** est le token central de l'écosystème Dark Forest, implémenté comme un token ERC-20 standard.

---

## Informations de base

| Propriété | Valeur |
|-----------|--------|
| Adresse du contrat | `0x1266e922fe34459efda34e7ee5caf327fbf138d7` |
| Nom du token | DarkForest |
| Symbole | DFT |
| Décimales | 18 |
| Plafond d'émission total | **4 206 900 000 000 DFT** (~4,2 billions) |
| Période de distribution | ~10 ans (3 650 jours) |

---

## Utilisations du DFT

| Cas d'utilisation | Description |
|-------------------|-------------|
| 🚀 **Saut spatial** | `10 + 10 × √(nombre de sauts)` DFT, max 1 000 |
| 🔧 **Améliorations de systèmes** | Coûts d'amélioration basés sur la formule d'ancrage |
| 🔄 **Reconstruction de civilisation** | `100 + 10 × nombre de renaissances` DFT, max 2 000 |
| 🏛️ **Amende de sortie d'alliance** | `100 + 10 × nombre de membres` DFT |
| ⚡ **Marché de l'énergie** | Acheter de l'énergie via le carnet d'ordres avec du DFT |

---

## Tokenomics

### Offre

- **Plafond strict** : 4 206 900 000 000 DFT (~4,2 billions)
- **Minage quotidien** : `4 206 900 000 000 / 3 650 ≈ 1 152 575 342 DFT` (émission quotidienne constante)
- **Distribution quotidienne** : via DailyMinter aux joueurs actifs

### Taxe de transaction

Quand DFT est échangé sur un DEX (PancakeSwap) :

| Taxe | Taux | Objectif |
|------|------|----------|
| Frais de développement | 1 % | Équipe |
| Frais de marketing | 1,5 % | Marketing |
| **Taxe de transaction totale** | **2,5 %** | |

La taxe s'applique uniquement lors des transactions dans les **pools de liquidité enregistrés** (LP). Les transferts réguliers ne sont pas taxés.

## Voir DFT dans votre portefeuille

Pour ajouter DFT à MetaMask :

1. Ouvrez MetaMask → **"Importer des tokens"**
2. Adresse du contrat : `0x1266e922fe34459efda34e7ee5caf327fbf138d7`
3. Le symbole s'affichera automatiquement comme **DFT**
4. Cliquez sur **"Ajouter"**

**Prochaine étape recommandée : [Marché de l'énergie →](/docs/economy/energy-market)**
