# Minage quotidien

Le contrat **DailyMinter** crée un montant fixe de DFT chaque jour et le distribue équitablement entre les civilisations actives. **Réclamez-le le jour même ou perdez-le** — les montants non réclamés ne s'accumulent pas.

---

## Comment réclamer

### Conditions requises

- Vous devez avoir créé une civilisation **avant minuit UTC** le jour de la réclamation
- `distribute()` doit avoir été appelé pour ce jour (n'importe qui peut le déclencher)
- Chaque adresse ne peut réclamer qu'une fois par jour

### Opération

```
claim()
```

Confirmez la transaction (frais de gaz uniquement).

---

## Flux de minage

### Étape 1 : Distribuer

N'importe qui peut appeler `distribute()` :

1. Le contrat vérifie la dernière époque non distribuée
2. Prend un instantané du nombre actuel de civilisations actives
3. Calcule `part par joueur = émission quotidienne / civilisations actives`
4. Crée le DFT total vers le contrat
5. `Émission quotidienne = 1 152 575 342 DFT` (constante)

### Étape 2 : Réclamer

Les joueurs appellent `claim()` :

1. Vérifie que vous avez créé une civilisation avant cette époque
2. Vérifie que vous n'avez pas déjà réclamé
3. Transfère votre part vers votre portefeuille

### Étape 3 : Brûler le reste

Lors du prochain `distribute()`, tout DFT de l'époque précédente qui n'a pas été réclamé est **brûlé** (réduisant l'offre en circulation).

---

## Chronologie

```
Époque 0      Époque 1      Époque 2
   |            |            |
UTC 00:00    UTC 00:00    UTC 00:00
   ↑créer civ   ↑distribuer   ↑distribuer
                ↑réclamer     ↑réclamer
                              ↑non réclamé brûlé
```

- Les époques sont divisées par minuit UTC
- Vous ne pouvez réclamer que l'**époque la plus récemment distribuée**
- Manquez une époque et vous ne pourrez jamais la réclamer

---

## Paramètres clés

| Paramètre | Valeur |
|-----------|--------|
| Émission quotidienne | **1 152 575 342 DFT** (constante) |
| Période totale de distribution | **3 650 jours** (~10 ans) |
| Plafond d'émission total | **4 206 900 000 000 DFT** (~4,2 billions) |
| Fenêtre de réclamation | Dans chaque époque (généralement 1 jour) |

---

## Pourquoi réclamer quotidiennement

1. 🆓 **DFT gratuit** — ne coûte que du gaz, disponible tous les jours
2. 📉 **Non réclamé est brûlé** — manquez-le et il disparaît
3. 📈 **Moins de joueurs = plus par personne** — si les joueurs actifs diminuent, chaque part augmente

> 💡 Définissez un rappel quotidien vers l'heure UTC pour réclamer. Si `distribute()` n'a pas encore été appelé pour la journée, vous pouvez le déclencher vous-même — cela ne coûte que du gaz.
