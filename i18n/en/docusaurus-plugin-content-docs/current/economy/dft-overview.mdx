# DFT Token Overview

**DFT (Dark Forest Token)** is the core token of the Dark Forest ecosystem, implemented as a standard ERC-20 token.

---

## Basic Information

| Property | Value |
|----------|-------|
| Contract Address | `0x1266e922fe34459efda34e7ee5caf327fbf138d7` |
| Token Name | DarkForest |
| Symbol | DFT |
| Decimals | 18 |
| Total Supply Cap | **420,690,000,000,000 DFT** (420.69 trillion) |
| Distribution Period | ~10 years (3,650 days) |

---

## DFT Uses

| Use Case | Description |
|----------|-------------|
| 🚀 **Space Jump** | `10 + 10 × √(jump count)` DFT, cap 1,000 |
| 🔧 **System Upgrades** | Anchor formula-based upgrade costs |
| 🔄 **Rebuild Civilization** | `100 + 10 × rebirth count` DFT, cap 2,000 |
| 🏛️ **Alliance Exit Fine** | `100 + 10 × member count` DFT |
| ⚡ **Energy Market** | Buy energy via order book using DFT |

---

## Tokenomics

### Supply

- **Hard cap**: 420.69 trillion DFT
- **Daily mint**: `total supply / 3,650`
- **Daily distribution**: via DailyMinter to active players

### Transaction Tax

When DFT is traded on a DEX (PancakeSwap):

| Tax | Rate | Purpose |
|-----|------|---------|
| Development Fee | 1% | Team |
| Marketing Fee | 1.5% | Marketing |
| **Total Transaction Tax** | **2.5%** | |

Tax only applies when trading in **registered liquidity pools** (LP). Regular transfers are untaxed.

## Viewing DFT in Your Wallet

To add DFT to MetaMask:

1. Open MetaMask → **"Import Tokens"**
2. Contract address: `0x1266e922fe34459efda34e7ee5caf327fbf138d7`
3. The symbol will auto-display as **DFT**
4. Click **"Add"**
