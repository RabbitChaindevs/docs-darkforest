# AI ایجنٹ کیا ہے

AI ایجنٹ ایک **آف چین خودکار پروگرام** ہے جو آپ کی تہذیب کی طرف سے گیم کے آپریشنز انجام دیتا ہے۔ یہ **اجازت بٹ ماسک** کے ذریعے مجاز ہوتا ہے اور **پالیسی کی حدود** میں کام کرتا ہے۔

---

## ایجنٹ کیا کر سکتا ہے

ایجنٹ کی صلاحیتیں **مجاز اجازتوں** کے ذریعے کنٹرول کی جاتی ہیں:

| اجازت | قدر | عمل |
|-----------|-------|--------|
| COLLECT | 1 | توانائی جمع کریں |
| REPAIR | 2 | نظاموں کی مرمت کریں |
| CLAIM_COMBAT | 8 | جنگی توانائی کلیم کریں |
| ATTACK | 32 | حملہ کریں |
| UPGRADE | 64 | نظاموں کو اپ گریڈ کریں |
| JUMP | 128 | خلائی چھلانگ |
| REBUILD | 256 | تہذیب کی تعمیر نو |
| DONATE | 512 | ٹوٹم کو عطیہ کریں |

آپ ایجنٹ کو اجازتوں کا کوئی بھی مجموعہ دے سکتے ہیں (بٹ ماسک)۔

---

## پالیسی کی حدود

اجازتوں کے علاوہ، آپ ایجنٹ کے لیے **پالیسی کی حدود** مقرر کر سکتے ہیں:

| حد | تفصیل |
|-----------|-------------|
| `minEnergyReserve` | کم از کم توانائی کا ذخیرہ (ایجنٹ اس سے نیچے خرچ نہیں کرے گا) |
| `minDftReserve` | کم از کم DFT کا ذخیرہ |
| `maxAttackPerHour` | فی گھنٹہ زیادہ سے زیادہ حملے |
| `maxUpgradePerDay` | فی دن زیادہ سے زیادہ اپ گریڈ |
| `maxSingleUpgradeCost` | فی اپ گریڈ زیادہ سے زیادہ DFT لاگت |
| `autoCollect` | خودکار توانائی جمع کریں |
| `autoRepair` | خودکار نظاموں کی مرمت |
| `autoShieldRepair` | خودکار ڈھال کی مرمت |
| `autoUpgrade` | خودکار اپ گریڈ |
| `autoAttack` | خودکار حملہ |

## سیکیورٹی میکانزم

### 1. آن چین رجسٹریشن

ایجنٹ کی پالیسی `DarkForestAgentRegistry.sol` کنٹریکٹ پر محفوظ ہوتی ہے اور ایک بار سیٹ ہونے کے بعد اس کے ساتھ چھیڑ چھاڑ نہیں کی جا سکتی۔

### 2. اجازت کی علیحدگی

- **اختیار دینا**: آپ ایجنٹ کو اپنی طرف سے کام کرنے کا اختیار دیتے ہیں
- **اجازت بٹ ماسک**: ہر عمل پر باریک بینی سے کنٹرول
- **منسوخ قابل**: `revokeAgent()` فوری طور پر اثر انداز ہوتا ہے

### 3. سرکٹ بریکرز

```
حملہ فریکوئنسی کی حد: زیادہ سے زیادہ N فی گھنٹہ
اپ گریڈ فریکوئنسی کی حد: زیادہ سے زیادہ N فی دن
توانائی کی منزل: X سے کم نہ ہو
DFT کی منزل: Y سے کم نہ ہو
```

### 4. بلیک لسٹ / وائٹ لسٹ

- **بلیک لسٹ**: مخصوص کھلاڑیوں پر حملہ منع کریں
- **وائٹ لسٹ**: صرف مخصوص کھلاڑیوں پر حملہ کرنے کی اجازت دیں (وائٹ لسٹ فعال ہونے پر بلیک لسٹ کو نظر انداز کیا جاتا ہے)
