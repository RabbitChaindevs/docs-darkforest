# 5 منٹ کا فوری آغاز

## شرائط

- Python 3.10+
- ایک فعال ڈارک فاریسٹ تہذیب
- آپ کے والٹ کی پرائیویٹ کی (ایجنٹ کے لین دین پر دستخط کرنے کے لیے)

---

## مرحلہ 1: ایجنٹ انسٹال کریں

```bash
git clone https://github.com/dark-forest/agent.git
cd agent
pip install -r requirements.txt
```

## مرحلہ 2: اپنی پرائیویٹ کی درآمد کریں

```bash
python -m dark_forest.cli key import
```

پرائیویٹ کی خفیہ کردہ محفوظ کی جاتی ہے، کبھی سادہ متن میں نہیں۔

> ⚠️ یہ انتہائی تجویز کیا جاتا ہے کہ **ایک مخصوص گیم والٹ** استعمال کریں — اپنا مرکزی والٹ استعمال نہ کریں جس میں اہم اثاثے ہوں۔

## مرحلہ 3: ایجنٹ کو اختیار دیں

گیم انٹرفیس میں، اپنے ایجنٹ کے والٹ ایڈریس کے لیے اجازتیں سیٹ کریں:

- وہ آپریشن اجازتیں منتخب کریں جن کی ایجنٹ کو ضرورت ہے
- پالیسی کی حدود سیٹ کریں (حملہ فریکوئنسی، وسائل کے ذخائر، وغیرہ)

## مرحلہ 4: حکمت عملی منتخب کریں

```bash
# دستیاب حکمت عملیوں کی فہرست
python -m dark_forest.cli strategy list

# متوازن حکمت عملی استعمال کریں (تجویز کردہ)
python -m dark_forest.cli strategy select balanced
```

چار پری سیٹ حکمت عملیاں:

| حکمت عملی | انداز | بہترین برائے |
|----------|-------|----------|
| `aggressive` | جارحانہ حملہ آور | انتہائی فعال کھلاڑی |
| `balanced` | متوازن حملہ/دفاع | **تجویز کردہ**، زیادہ تر کھلاڑیوں کے لیے موزوں |
| `defensive` | قدامت پسند دفاع | وہ کھلاڑی جو اکثر آف لائن ہوتے ہیں |
| `farmer` | جمع کرنے پر توجہ | پرامن ترقی |

## مرحلہ 5: لانچ کریں

```bash
# شروع کریں (پہلے سے طے شدہ متوازن حکمت عملی)
python -m dark_forest.cli run

# ایک حکمت عملی متعین کریں
python -m dark_forest.cli run --strategy aggressive
```

## مرحلہ 6: اطلاعات ترتیب دیں (اختیاری)

### Telegram

```bash
python -m dark_forest.cli config set notify.telegram.token YOUR_BOT_TOKEN
python -m dark_forest.cli config set notify.telegram.chat_id YOUR_CHAT_ID
```

### Discord

```bash
python -m dark_forest.cli config set notify.discord.webhook YOUR_WEBHOOK_URL
```

## حفاظتی یاد دہانیاں

1. 🔐 **مخصوص گیم والٹ استعمال کریں** معقول DFT/توانائی ذخیرہ حدود کے ساتھ
2. 📉 **آہستہ آہستہ اجازتیں کھولیں** — صرف COLLECT + REPAIR سے شروع کریں، پھر ATTACK شامل کریں جب آپ کو یقین ہو جائے کہ یہ مستحکم ہے
3. 🔄 **باقاعدگی سے چیک کریں** — ایجنٹ کے لاگز اور رپورٹس کا جائزہ لیں

**تجویز کردہ اگلا مرحلہ: [حکمت عملی ٹیمپلیٹس →](/docs/agent/strategies)**
