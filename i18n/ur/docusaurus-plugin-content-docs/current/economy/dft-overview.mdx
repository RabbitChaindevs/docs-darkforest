# DFT ٹوکن کا جائزہ

**DFT (ڈارک فاریسٹ ٹوکن)** ڈارک فاریسٹ ایکو سسٹم کا بنیادی ٹوکن ہے، جو ایک معیاری ERC-20 ٹوکن کے طور پر نافذ کیا گیا ہے۔

---

## بنیادی معلومات

| خاصیت | قدر |
|----------|-------|
| کنٹریکٹ ایڈریس | `0x1266e922fe34459efda34e7ee5caf327fbf138d7` |
| ٹوکن کا نام | DarkForest |
| علامت | DFT |
| اعشاریہ | 18 |
| کل سپلائی کی حد | **4,206,900,000,000 DFT** (~4.2 ٹریلین) |
| تقسیم کی مدت | ~10 سال (3,650 دن) |

---

## DFT کے استعمال

| استعمال کا معاملہ | تفصیل |
|----------|-------------|
| 🚀 **خلائی چھلانگ** | `10 + 10 × √(چھلانگوں کی تعداد)` DFT، حد 1,000 |
| 🔧 **سسٹم اپ گریڈ** | اینکر فارمولا پر مبنی اپ گریڈ کے اخراجات |
| 🔄 **تہذیب کی تعمیر نو** | `100 + 10 × پنر جنم کی تعداد` DFT، حد 2,000 |
| 🏛️ **اتحاد چھوڑنے کا جرمانہ** | `100 + 10 × اراکین کی تعداد` DFT |
| ⚡ **توانائی مارکیٹ** | DFT کا استعمال کرتے ہوئے آرڈر بک کے ذریعے توانائی خریدنا |

---

## ٹوکن اکنامکس

### سپلائی

- **سخت حد**: 4,206,900,000,000 DFT (~4.2 ٹریلین)
- **یومیہ منٹ**: `4,206,900,000,000 / 3,650 ≈ 1,152,575,342 DFT` (مسلسل یومیہ اخراج)
- **یومیہ تقسیم**: DailyMinter کے ذریعے فعال کھلاڑیوں کو

### ٹرانزیکشن ٹیکس

جب DFT DEX (PancakeSwap) پر تجارت کیا جاتا ہے:

| ٹیکس | شرح | مقصد |
|-----|------|---------|
| ڈیولپمنٹ فیس | 1% | ٹیم |
| مارکیٹنگ فیس | 1.5% | مارکیٹنگ |
| **کل ٹرانزیکشن ٹیکس** | **2.5%** | |

ٹیکس صرف **رجسٹرڈ لیکویڈیٹی پولز** (LP) میں تجارت کرتے وقت لاگو ہوتا ہے۔ باقاعدہ ٹرانسفر بلا ٹیکس ہیں۔

## اپنے والٹ میں DFT دیکھنا

MetaMask میں DFT شامل کرنے کے لیے:

1. MetaMask کھولیں → **"Import Tokens"**
2. کنٹریکٹ ایڈریس: `0x1266e922fe34459efda34e7ee5caf327fbf138d7`
3. علامت خود بخود **DFT** ظاہر ہوگی
4. **"Add"** پر کلک کریں

**تجویز کردہ اگلا مرحلہ: [توانائی مارکیٹ →](/docs/economy/energy-market)**
