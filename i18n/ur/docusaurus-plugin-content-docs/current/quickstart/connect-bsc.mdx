# Connect to BSC

Dark Forest is deployed on **BNB Smart Chain (BSC)**. You need to switch your wallet to the BSC network to interact with the game.

## What Is BSC?

BNB Smart Chain is a blockchain network launched by Binance, offering:

- **Low transaction fees** — each operation costs just fractions of a cent
- **High throughput** — built for high-frequency game interactions
- **EVM compatible** — works with MetaMask and other主流 wallets

## Auto-Add BSC Network

The easiest way is to use Chainlist:

1. Visit [Chainlist](https://chainlist.org)
2. Search for "BNB Smart Chain"
3. Click "Connect Wallet" and authorize
4. Click "Add to MetaMask"
5. Confirm the addition

## Manual Configuration

You can also add the BSC network manually:

### Adding BSC to MetaMask

1. Click the network dropdown in the top-right corner of MetaMask
2. Click "Add Network"
3. Fill in the following details:

| Setting | Value |
|---------|-------|
| **Network Name** | BNB Smart Chain |
| **RPC URL** | `https://bsc-dataseed.binance.org` |
| **Chain ID** | `56` |
| **Currency Symbol** | `BNB` |
| **Block Explorer** | `https://bscscan.com` |

4. Click "Save"

### Backup RPC Nodes

If the main RPC is congested, you can use backup nodes:

| Provider | RPC URL |
|----------|---------|
| Binance Official | `https://bsc-dataseed1.binance.org` |
| Ankr | `https://rpc.ankr.com/bsc` |
| PublicNode | `https://bsc.publicnode.com` |

## Getting BNB

Every operation on BSC requires BNB for gas fees. Here's how to get some:

1. **Centralized exchange** — buy BNB from Binance, OKX, etc. and withdraw to the BSC network
2. **Cross-chain bridge** — transfer BNB from other networks to BSC via a bridge

:::caution
When withdrawing, **make sure to select the BSC network (BEP-20)** or you may lose your funds!
:::

## Verify Your Connection

After configuration, switch to the BNB Smart Chain network in your wallet. If the Chain ID shows `56` and your balance displays the correct BNB amount, you're all set.

Once your wallet and network are ready, the next step is to [Get DFT Tokens](/docs/quickstart/get-dft).
