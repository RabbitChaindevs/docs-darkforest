# BSC سے جڑیں

ڈارک فاریسٹ **BNB Smart Chain (BSC)** پر تعینات ہے۔ گیم کے ساتھ تعامل کے لیے آپ کو اپنا والٹ BSC نیٹ ورک پر سوئچ کرنا ہوگا۔

## BSC کیا ہے؟

BNB Smart Chain Binance کی طرف سے شروع کردہ ایک بلاک چین نیٹ ورک ہے، جو پیش کرتا ہے:

- **کم لین دین فیس** — ہر عمل کی لاگت ایک سینٹ کے حصوں سے بھی کم
- **زیادہ تھروپٹ** — تیز رفتار گیم تعاملات کے لیے ڈیزائن کردہ
- **EVM مطابقت** — MetaMask اور دیگر والٹس کے ساتھ کام کرتا ہے

## خودکار BSC نیٹ ورک شامل کرنا

سب سے آسان طریقہ Chainlist استعمال کرنا ہے:

1. [Chainlist](https://chainlist.org) پر جائیں
2. "BNB Smart Chain" تلاش کریں
3. "Connect Wallet" پر کلک کریں اور اجازت دیں
4. "Add to MetaMask" پر کلک کریں
5. شامل کرنے کی تصدیق کریں

## دستی ترتیب

آپ BSC نیٹ ورک کو دستی طور پر بھی شامل کر سکتے ہیں:

### MetaMask میں BSC شامل کرنا

1. MetaMask کے اوپری دائیں کونے میں نیٹ ورک ڈراپ ڈاؤن پر کلک کریں
2. "Add Network" پر کلک کریں
3. درج ذیل تفصیلات پُر کریں:

| ترتیب | قیمت |
|---------|-------|
| **نیٹ ورک کا نام** | BNB Smart Chain |
| **RPC URL** | `https://bsc-dataseed.binance.org` |
| **چین ID** | `56` |
| **کرنسی علامت** | `BNB` |
| **بلاک ایکسپلورر** | `https://bscscan.com` |

4. "Save" پر کلک کریں

### بیک اپ RPC نوڈس

اگر مین RPC بھیڑ کا شکار ہو، تو آپ بیک اپ نوڈس استعمال کر سکتے ہیں:

| فراہم کنندہ | RPC URL |
|------------|---------|
| Binance آفیشل | `https://bsc-dataseed1.binance.org` |
| Ankr | `https://rpc.ankr.com/bsc` |
| PublicNode | `https://bsc.publicnode.com` |

## BNB حاصل کرنا

BSC پر ہر آپریشن کے لیے گیس فیس کے طور پر BNB درکار ہے۔ اسے حاصل کرنے کا طریقہ:

1. **مرکزی ایکسچینج** — Binance، OKX وغیرہ سے BNB خریدیں اور BSC نیٹ ورک پر نکالیں
2. **کراس چین برج** — برج کے ذریعے BNB کو دوسرے نیٹ ورکس سے BSC پر منتقل کریں

:::caution
نکالنے کے وقت، **یقینی بنائیں کہ آپ BSC نیٹ ورک (BEP-20) منتخب کر رہے ہیں** ورنہ آپ اپنے فنڈز کھو سکتے ہیں!
:::

## اپنے کنکشن کی تصدیق کریں

ترتیب کے بعد، اپنے والٹ میں BNB Smart Chain نیٹ ورک پر سوئچ کریں۔ اگر Chain ID `56` دکھائے اور آپ کا بیلنس صحیح BNB رقم ظاہر کرے، تو آپ تیار ہیں۔

ایک بار جب آپ کا والٹ اور نیٹ ورک تیار ہو جائیں، اگلا مرحلہ [DFT ٹوکن حاصل کرنا](/docs/quickstart/get-dft) ہے۔
