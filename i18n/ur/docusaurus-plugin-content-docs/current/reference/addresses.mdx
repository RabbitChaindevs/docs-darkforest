## کنٹریکٹ آرکیٹیکچر

```
DarkForest (DELEGATECALL داخلہ)
├── DELEGATECALL → DarkForestGameplay   (تہذیب بنانا، سسٹم اپ گریڈ)
├── DELEGATECALL → DarkForestBattle      (حملہ، ڈھال کی مرمت، ڈھال کی بحالی)
├── DELEGATECALL → DarkForestMovement    (خلائی چھلانگ، تعاقبی چھلانگ، سفر)
└── DELEGATECALL → DarkForestAdmin       (جمع، مرمت، تعمیر نو، ٹوٹم، نکالنا)

DarkForestStorage (مشترکہ اسٹوریج لے آؤٹ)
→ تمام سیٹلائٹ کنٹریکٹس مستقل لے آؤٹ کے لیے DarkForestStorage سے وراثت پاتے ہیں
```
