## Contract Architecture

```
DarkForest (DELEGATECALL entry)
├── DELEGATECALL → DarkForestGameplay   (create civ, upgrade systems)
├── DELEGATECALL → DarkForestBattle      (attack, repair shield, shield regen)
├── DELEGATECALL → DarkForestMovement    (space jump, tracking jump, cruise)
└── DELEGATECALL → DarkForestAdmin       (collect, repair, rebuild, totem, withdraw)

DarkForestStorage (shared storage layout)
→ All satellite contracts inherit DarkForestStorage for consistent layout
```
