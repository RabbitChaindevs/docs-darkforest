## कॉन्ट्रैक्ट आर्किटेक्चर

```
DarkForest (DELEGATECALL प्रवेश)
├── DELEGATECALL → DarkForestGameplay   (सभ्यता बनाना, सिस्टम अपग्रेड)
├── DELEGATECALL → DarkForestBattle      (हमला, ढाल मरम्मत, ढाल पुनर्जनन)
├── DELEGATECALL → DarkForestMovement    (स्पेस जंप, ट्रैकिंग जंप, क्रूज़)
└── DELEGATECALL → DarkForestAdmin       (संग्रह, मरम्मत, पुनर्निर्माण, टोटम, निकासी)

DarkForestStorage (साझा भंडारण लेआउट)
→ सभी उपग्रह कॉन्ट्रैक्ट संगत लेआउट के लिए DarkForestStorage से विरासत प्राप्त करते हैं
```
