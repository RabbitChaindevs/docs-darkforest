# 5-मिनट त्वरित आरंभ

## आवश्यक शर्तें

- Python 3.10+
- एक सक्रिय डार्क फ़ॉरेस्ट सभ्यता
- आपके वॉलेट की निजी कुंजी (एजेंट के लेन-देन पर हस्ताक्षर करने के लिए)

---

## चरण 1: एजेंट इंस्टॉल करें

```bash
git clone https://github.com/dark-forest/agent.git
cd agent
pip install -r requirements.txt
```

## चरण 2: अपनी निजी कुंजी आयात करें

```bash
python -m dark_forest.cli key import
```

निजी कुंजी एन्क्रिप्टेड संग्रहीत की जाती है, कभी भी सादे पाठ में नहीं।

> ⚠️ **समर्पित गेम वॉलेट** का उपयोग करने की अत्यधिक अनुशंसा की जाती है — अपने मुख्य वॉलेट का उपयोग न करें जिसमें महत्वपूर्ण संपत्तियाँ हैं।

## चरण 3: एजेंट को अधिकृत करें

गेम इंटरफ़ेस में, अपने एजेंट के वॉलेट पते के लिए अनुमतियाँ सेट करें:

- एजेंट को आवश्यक संचालन अनुमतियाँ चेक करें
- नीति बाधाएँ सेट करें (हमला आवृत्ति, संसाधन आरक्षित, आदि)

## चरण 4: एक रणनीति चुनें

```bash
# उपलब्ध रणनीतियाँ सूचीबद्ध करें
python -m dark_forest.cli strategy list

# संतुलित रणनीति का उपयोग करें (अनुशंसित)
python -m dark_forest.cli strategy select balanced
```

चार प्रीसेट रणनीतियाँ:

| रणनीति | शैली | सर्वश्रेष्ठ के लिए |
|----------|-------|----------|
| `aggressive` | आक्रामक हमलावर | अत्यधिक सक्रिय खिलाड़ी |
| `balanced` | संतुलित हमला/रक्षा | **अनुशंसित**, अधिकांश खिलाड़ियों के लिए उपयुक्त |
| `defensive` | रूढ़िवादी रक्षा | जो खिलाड़ी अक्सर ऑफलाइन रहते हैं |
| `farmer` | संग्रह पर ध्यान | शांतिपूर्ण विकास |

## चरण 5: लॉन्च करें

```bash
# शुरू करें (डिफ़ॉल्ट संतुलित रणनीति)
python -m dark_forest.cli run

# एक रणनीति निर्दिष्ट करें
python -m dark_forest.cli run --strategy aggressive
```

## चरण 6: सूचनाएँ कॉन्फ़िगर करें (वैकल्पिक)

### Telegram

```bash
python -m dark_forest.cli config set notify.telegram.token YOUR_BOT_TOKEN
python -m dark_forest.cli config set notify.telegram.chat_id YOUR_CHAT_ID
```

### Discord

```bash
python -m dark_forest.cli config set notify.discord.webhook YOUR_WEBHOOK_URL
```

## सुरक्षा अनुस्मारक

1. 🔐 **एक समर्पित गेम वॉलेट का उपयोग करें** जिसमें उचित DFT/ऊर्जा आरक्षित सीमाएँ हों
2. 📉 **अनुमतियाँ धीरे-धीरे खोलें** — केवल COLLECT + REPAIR से शुरू करें, फिर एक बार स्थिर होने पर ATTACK जोड़ें
3. 🔄 **नियमित रूप से जाँच करें** — एजेंट के लॉग और रिपोर्ट की समीक्षा करें
