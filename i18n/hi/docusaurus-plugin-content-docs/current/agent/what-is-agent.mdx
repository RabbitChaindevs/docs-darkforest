# AI एजेंट क्या है

AI एजेंट एक **ऑफ-चेन स्वचालित प्रोग्राम** है जो आपकी सभ्यता की ओर से गेम ऑपरेशन करता है। यह **अनुमति बिटमास्क** के माध्यम से अधिकृत होता है और **नीति बाधाओं** के भीतर संचालित होता है।

---

## एजेंट क्या कर सकता है

एजेंट की क्षमताएँ **अधिकृत अनुमतियों** द्वारा नियंत्रित होती हैं:

| अनुमति | मान | कार्रवाई |
|-----------|-------|--------|
| COLLECT | 1 | ऊर्जा एकत्र करें |
| REPAIR | 2 | सिस्टम की मरम्मत करें |
| CLAIM_COMBAT | 8 | युद्ध ऊर्जा का दावा करें |
| ATTACK | 32 | हमला करें |
| UPGRADE | 64 | सिस्टम अपग्रेड करें |
| JUMP | 128 | स्पेस जंप |
| REBUILD | 256 | सभ्यता पुनर्निर्माण |
| DONATE | 512 | टोटम को दान करें |

आप एजेंट को अनुमतियों का कोई भी संयोजन (बिटमास्क) प्रदान कर सकते हैं।

---

## नीति बाधाएँ

अनुमतियों के अतिरिक्त, आप एजेंट के लिए **नीति बाधाएँ** निर्धारित कर सकते हैं:

| बाधा | विवरण |
|-----------|-------------|
| `minEnergyReserve` | न्यूनतम ऊर्जा आरक्षित (एजेंट इससे नीचे खर्च नहीं करेगा) |
| `minDftReserve` | न्यूनतम DFT आरक्षित |
| `maxAttackPerHour` | प्रति घंटे अधिकतम हमले |
| `maxUpgradePerDay` | प्रति दिन अधिकतम अपग्रेड |
| `maxSingleUpgradeCost` | प्रति अपग्रेड अधिकतम DFT लागत |
| `autoCollect` | स्वचालित ऊर्जा संग्रह |
| `autoRepair` | स्वचालित सिस्टम मरम्मत |
| `autoShieldRepair` | स्वचालित ढाल मरम्मत |
| `autoUpgrade` | स्वचालित अपग्रेड |
| `autoAttack` | स्वचालित हमला |

## सुरक्षा तंत्र

### 1. ऑन-चेन पंजीकरण

एजेंट की नीति `DarkForestAgentRegistry.sol` कॉन्ट्रैक्ट पर संग्रहीत होती है और एक बार सेट होने के बाद इसे छेड़छाड़ नहीं किया जा सकता।

### 2. अनुमति पृथक्करण

- **प्राधिकृत करें**: आप एजेंट को अपनी ओर से कार्य करने का अधिकार देते हैं
- **अनुमति बिटमास्क**: प्रत्येक ऑपरेशन पर सूक्ष्म-नियंत्रण
- **प्रत्याहार योग्य**: `revokeAgent()` तुरंत प्रभावी होता है

### 3. सर्किट ब्रेकर

```
हमला आवृत्ति सीमा: प्रति घंटे अधिकतम N
अपग्रेड आवृत्ति सीमा: प्रति दिन अधिकतम N
ऊर्जा तल: X से कभी नीचे नहीं
DFT तल: Y से कभी नीचे नहीं
```

### 4. ब्लैकलिस्ट / व्हाइटलिस्ट

- **ब्लैकलिस्ट**: विशिष्ट खिलाड़ियों पर हमला प्रतिबंधित करें
- **व्हाइटलिस्ट**: केवल विशिष्ट खिलाड़ियों पर हमले की अनुमति दें (व्हाइटलिस्ट सक्रिय होने पर ब्लैकलिस्ट को अनदेखा किया जाता है)
