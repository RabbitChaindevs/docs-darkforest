# डार्क फ़ॉरेस्ट दस्तावेज़ीकरण

डार्क फ़ॉरेस्ट एक पूर्णतः ऑन-चेन 3D अंतरतारकीय सभ्यता रणनीति गेम (SLG) है जो **BNB स्मार्ट चेन** पर तैनात है। सभी गेम लॉजिक स्मार्ट कॉन्ट्रैक्ट पर चलते हैं, जिसमें मॉड्यूलर अपग्रेड के लिए DELEGATECALL आर्किटेक्चर है।
