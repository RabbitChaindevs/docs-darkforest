# ऊर्जा प्रणाली

डार्क फ़ॉरेस्ट में ऊर्जा सबसे मौलिक संसाधन है। लगभग हर ऑपरेशन — अपग्रेड, हमले, गति, और मरम्मत — के लिए ऊर्जा की आवश्यकता होती है।

---

## ऊर्जा अर्जित करना

### 1. ऊर्जा संग्राहक (निष्क्रिय आउटपुट)

```
संग्रह दर = 3 + 10 × √(संग्राहक स्तर - 1)   ऊर्जा/से
```

आपके द्वारा आमंत्रित प्रत्येक खिलाड़ी के लिए, दर और बढ़ जाती है।

**संग्राहक स्थायित्व**: संग्राहक प्रति सेकंड 1 स्थायित्व खोता है।

| संग्राहक स्तर | आउटपुट प्रति सेकंड | अधिकतम संग्राहक स्थायित्व |
|----------------|-------------------|--------------------------|
| Lv 1 | 3 | 1 दिन |
| Lv 10 | ~33 | 1 दिन + 18 घंटे |
| Lv 50 | ~73 | 1 दिन + 4 दिन |
| Lv 100 | ~103 | 1 दिन + 8.3 दिन |

> ⚠️ जब संग्राहक स्थायित्व समाप्त हो जाता है, उत्पादन रुक जाता है। संग्रह फिर से शुरू करने के लिए आपको इसकी मरम्मत करनी होगी।

### 2. युद्ध लूट

किसी अन्य सभ्यता के खिलाफ युद्ध जीतने के बाद, आप उनकी ऊर्जा का एक हिस्सा लूटते हैं:

```
लूट अनुपात = 5% + हथियार स्तर × 0.5%
```

लूटी गई ऊर्जा एक **लंबित पूल** (pendingCombatEnergy) में जाती है, जिसे आपको मैन्युअल रूप से दावा करना होगा।

### 3. आमंत्रण पुरस्कार

प्रत्येक नए खिलाड़ी के लिए जिसे आप सफलतापूर्वक आमंत्रित करते हैं (वे अपनी सभ्यता बनाते समय आपका पता दर्ज करते हैं):

- **आपको** 150 ऊर्जा मिलती है
- **आमंत्रित व्यक्ति** को 150 ऊर्जा मिलती है

### 4. ऊर्जा बाज़ार

देखें [ऊर्जा बाज़ार](/docs/economy/energy-market)।

## ऊर्जा खर्च करना

| उपयोग मामला | लागत |
|----------|------|
| **दैनिक रखरखाव** | `2,000 × कुल स्तर` / दिन |
| **सिस्टम अपग्रेड** | एंकर फ़ॉर्मूला (सभ्यता गाइड देखें) |
| **हमला** | `50,000 + 50,000 × हथियार स्तर` |
| **स्पेस जंप** | `200,000 + 200,000 × √(जंप गणना)`, सीमा 16.5M |
| **संग्राहक मरम्मत** | `1 ऊर्जा / सेकंड` |
| **हथियार मरम्मत** | `3 ऊर्जा / पॉइंट` |
| **ढाल मरम्मत** | `4 ऊर्जा / पॉइंट` |
| **इंजन मरम्मत** | `5 ऊर्जा / पॉइंट` |
| **ढाल पुनर्जनन** | `पुनर्जनन मात्रा × 1` (स्वचालित पुनर्जनन) |
| **ढाल मरम्मत (मैन्युअल)** | `मरम्मत मात्रा × 4` |
| **गठबंधन ढाल सहायता** | `मरम्मत मात्रा × 1` (सहयोगी द्वारा भुगतान) |
| **टोटम दान** | कस्टम राशि |
| **पुनर्निर्माण** | आधार 500,000, पुनर्जन्म गणना के साथ बढ़ता है |

## ऊर्जा प्रबंधन

### दैनिक रखरखाव

जब भी आप कोई कार्रवाई करते हैं तो रखरखाव समयानुपातिक रूप से काटा जाता है:

```
वर्तमान रखरखाव = 2,000 × कुल स्तर × (अंतिम कार्रवाई के बाद का समय / 86400)
```

### आरक्षित ऊर्जा

ऊर्जा बाज़ार पर बिक्री ऑर्डर में रखी गई ऊर्जा लॉक हो जाती है (`energyReserved`)। एक बार `energy - reserved` शून्य हो जाने पर रखरखाव काटा जाना बंद हो जाता है।

### टिप्स

1. **ऊर्जा > 1 महीने का रखरखाव रखें** — नियमित कार्यों से बार-बार कमी को रोकता है
2. **संग्राहक अपग्रेड को प्राथमिकता दें** — उच्च-स्तरीय संग्राहक निष्क्रिय आय की कुंजी है
3. **हमला करने से पहले लागत की गणना करें** — सुनिश्चित करें कि लूट निवेश के लायक है
4. **संग्राहक स्थायित्व की नियमित रूप से मरम्मत करें** — संग्रह में रुकावट से बचें
