# वॉलेट इंस्टॉल करें

डार्क फ़ॉरेस्ट खेलने से पहले, आपको अपनी डिजिटल संपत्तियों को प्रबंधित करने और ऑन-चेन लेन-देन पर हस्ताक्षर करने के लिए एक Web3 वॉलेट की आवश्यकता है।

## Web3 वॉलेट क्या है?

Web3 वॉलेट विकेंद्रीकृत एप्लिकेशन (dApps) की आपकी कुंजी है। यह आपको यह करने देता है:

- **डिजिटल संपत्तियों को स्टोर और प्रबंधित** करें जैसे DFT और BNB
- **लेन-देन पर हस्ताक्षर करें** — हर इन-गेम क्रिया (निर्माण, चलना, लड़ाई) के लिए आपके वॉलेट के हस्ताक्षर की आवश्यकता होती है
- **अपनी पहचान प्रमाणित करें** — डार्क फ़ॉरेस्ट में आपका वॉलेट पता ही आपकी पहचान है

## अनुशंसित वॉलेट

### MetaMask (अनुशंसित)

MetaMask सबसे लोकप्रिय Web3 वॉलेट है, जो ब्राउज़र एक्सटेंशन और मोबाइल ऐप के रूप में उपलब्ध है।

#### ब्राउज़र इंस्टॉलेशन

1. [MetaMask वेबसाइट](https://metamask.io) पर जाएँ
2. ब्राउज़र एक्सटेंशन प्राप्त करने के लिए "डाउनलोड" पर क्लिक करें
3. अपना ब्राउज़र चुनें (Chrome / Firefox / Edge / Brave)
4. "ब्राउज़र में जोड़ें" पर क्लिक करें
5. नया वॉलेट बनाने के लिए विज़ार्ड का अनुसरण करें

:::tip
इंस्टॉल करने के बाद, त्वरित पहुँच के लिए MetaMask एक्सटेंशन को अपने ब्राउज़र टूलबार पर पिन करें।
:::

#### वॉलेट बनाना

1. MetaMask आइकन पर क्लिक करें और "वॉलेट बनाएँ" चुनें
2. एक **मजबूत पासवर्ड** सेट करें (कम से कम 8 अक्षर, अक्षरों और संख्याओं का मिश्रण)
3. **अपने सीड फ्रेज़ को सुरक्षित रूप से बैकअप करें** — यह आपके वॉलेट को पुनर्प्राप्त करने का एकमात्र तरीका है!

:::danger सीड फ्रेज़ सुरक्षा
- **कभी भी** अपना सीड फ्रेज़ किसी के साथ साझा न करें
- **स्क्रीनशॉट न लें** या इसे क्लाउड में स्टोर न करें
- ऑफलाइन बैकअप के लिए भौतिक मीडिया (कागज और कलम) का उपयोग करें
- आपका सीड फ्रेज़ माँगने वाला कोई भी व्यक्ति स्कैमर है
:::

4. वॉलेट निर्माण पूरा करने के लिए अपने सीड फ्रेज़ की पुष्टि करें

### Trust Wallet

Trust Wallet बिनेंस का आधिकारिक मोबाइल वॉलेट है।

1. App Store या Google Play पर "Trust Wallet" खोजें और इंस्टॉल करें
2. ऐप खोलें और "नया वॉलेट बनाएँ" चुनें
3. अपने सीड फ्रेज़ का सुरक्षित रूप से बैकअप लें
4. वॉलेट के भीतर DApp ब्राउज़र सक्षम करें

## अगले कदम

एक बार आपका वॉलेट इंस्टॉल हो जाए, तो [BSC से कनेक्ट करें](/docs/quickstart/connect-bsc) पर आगे बढ़ें और डार्क फ़ॉरेस्ट में प्रवेश करने के लिए तैयार हो जाएँ।
