# BSC से कनेक्ट करें

डार्क फ़ॉरेस्ट **BNB स्मार्ट चेन (BSC)** पर तैनात है। गेम के साथ इंटरैक्ट करने के लिए आपको अपने वॉलेट को BSC नेटवर्क पर स्विच करना होगा।

## BSC क्या है?

BNB स्मार्ट चेन बिनेंस द्वारा लॉन्च किया गया एक ब्लॉकचेन नेटवर्क है, जो प्रदान करता है:

- **कम लेन-देन शुल्क** — प्रत्येक ऑपरेशन की लागत एक सेंट के अंश मात्र
- **उच्च थ्रूपुट** — उच्च-आवृत्ति गेम इंटरैक्शन के लिए निर्मित
- **EVM संगत** — MetaMask और अन्य मुख्यधारा वॉलेट के साथ काम करता है

## स्वचालित रूप से BSC नेटवर्क जोड़ें

सबसे आसान तरीका Chainlist का उपयोग करना है:

1. [Chainlist](https://chainlist.org) पर जाएँ
2. "BNB स्मार्ट चेन" खोजें
3. "वॉलेट कनेक्ट करें" पर क्लिक करें और अधिकृत करें
4. "MetaMask में जोड़ें" पर क्लिक करें
5. जोड़ने की पुष्टि करें

## मैन्युअल कॉन्फ़िगरेशन

आप BSC नेटवर्क को मैन्युअल रूप से भी जोड़ सकते हैं:

### MetaMask में BSC जोड़ना

1. MetaMask के ऊपरी-दाएँ कोने में नेटवर्क ड्रॉपडाउन पर क्लिक करें
2. "नेटवर्क जोड़ें" पर क्लिक करें
3. निम्नलिखित विवरण भरें:

| सेटिंग | मान |
|---------|-------|
| **नेटवर्क नाम** | BNB स्मार्ट चेन |
| **RPC URL** | `https://bsc-dataseed.binance.org` |
| **चेन ID** | `56` |
| **मुद्रा प्रतीक** | `BNB` |
| **ब्लॉक एक्सप्लोरर** | `https://bscscan.com` |

4. "सहेजें" पर क्लिक करें

### बैकअप RPC नोड्स

यदि मुख्य RPC भीड़भाड़ वाला है, तो आप बैकअप नोड्स का उपयोग कर सकते हैं:

| प्रदाता | RPC URL |
|----------|---------|
| बिनेंस आधिकारिक | `https://bsc-dataseed1.binance.org` |
| Ankr | `https://rpc.ankr.com/bsc` |
| PublicNode | `https://bsc.publicnode.com` |

## BNB प्राप्त करना

BSC पर प्रत्येक ऑपरेशन के लिए गैस शुल्क के रूप में BNB की आवश्यकता होती है। कुछ BNB प्राप्त करने का तरीका:

1. **केंद्रीकृत एक्सचेंज** — Binance, OKX आदि से BNB खरीदें और BSC नेटवर्क पर निकालें
2. **क्रॉस-चेन ब्रिज** — ब्रिज के माध्यम से अन्य नेटवर्क से BSC में BNB स्थानांतरित करें

:::caution
निकासी करते समय, **BSC नेटवर्क (BEP-20) चुनना सुनिश्चित करें** अन्यथा आप अपने फंड खो सकते हैं!
:::

## अपना कनेक्शन सत्यापित करें

कॉन्फ़िगरेशन के बाद, अपने वॉलेट में BNB स्मार्ट चेन नेटवर्क पर स्विच करें। यदि चेन ID `56` दिखाती है और आपका बैलेंस सही BNB राशि प्रदर्शित करता है, तो आप तैयार हैं।

एक बार आपका वॉलेट और नेटवर्क तैयार हो जाए, तो अगला कदम है [DFT टोकन प्राप्त करें](/docs/quickstart/get-dft)।
