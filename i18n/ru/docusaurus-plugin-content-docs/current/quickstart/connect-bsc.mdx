# Подключение к BSC

Dark Forest развёрнут на **BNB Smart Chain (BSC)**. Вам нужно переключить ваш кошелёк на сеть BSC для взаимодействия с игрой.

## Что такое BSC?

BNB Smart Chain — это блокчейн-сеть, запущенная Binance, предлагающая:

- **Низкие комиссии за транзакции** — каждая операция стоит доли цента
- **Высокую пропускную способность** — создана для высокочастотных игровых взаимодействий
- **Совместимость с EVM** — работает с MetaMask и другими популярными кошельками

## Автоматическое добавление сети BSC

Самый простой способ — использовать Chainlist:

1. Посетите [Chainlist](https://chainlist.org)
2. Найдите "BNB Smart Chain"
3. Нажмите "Connect Wallet" и подтвердите
4. Нажмите "Add to MetaMask"
5. Подтвердите добавление

## Ручная настройка

Вы также можете добавить сеть BSC вручную:

### Добавление BSC в MetaMask

1. Нажмите на выпадающее меню сети в правом верхнем углу MetaMask
2. Нажмите "Добавить сеть"
3. Заполните следующие данные:

| Параметр | Значение |
|----------|----------|
| **Название сети** | BNB Smart Chain |
| **RPC URL** | `https://bsc-dataseed.binance.org` |
| **ID цепи** | `56` |
| **Символ валюты** | `BNB` |
| **Обозреватель блоков** | `https://bscscan.com` |

4. Нажмите "Сохранить"

### Запасные RPC-узлы

Если основной RPC перегружен, вы можете использовать запасные узлы:

| Провайдер | RPC URL |
|-----------|---------|
| Binance Official | `https://bsc-dataseed1.binance.org` |
| Ankr | `https://rpc.ankr.com/bsc` |
| PublicNode | `https://bsc.publicnode.com` |

## Получение BNB

Каждая операция в BSC требует BNB для газовых сборов. Вот как получить BNB:

1. **Централизованная биржа** — купите BNB на Binance, OKX и т.д. и выведите в сеть BSC
2. **Кросс-чейн мост** — переведите BNB из других сетей в BSC через мост

:::caution
При выводе **обязательно выберите сеть BSC (BEP-20)**, иначе вы можете потерять средства!
:::

## Проверка подключения

После настройки переключитесь на сеть BNB Smart Chain в вашем кошельке. Если ID цепи показывает `56` и ваш баланс отображает правильное количество BNB, всё готово.

После того как ваш кошелёк и сеть готовы, следующим шагом будет [Получение токенов DFT](/docs/quickstart/get-dft).
