# Получение токенов DFT

**DFT (Dark Forest Token)** — это основной токен ERC-20 Dark Forest с общим лимитом эмиссии около **~4,2 триллиона (4 206 900 000 000)**.

**Использование DFT:**
- 🚀 Космические прыжки (10–1 000 DFT за прыжок)
- 🔧 Улучшение систем
- 🔄 Восстановление цивилизации
- 🏛️ Штрафы за выход из альянса
- ⚡ Торговля на рынке энергии

---

## Способ 1: Покупка на DEX

DFT торгуется на PancakeSwap.

### Шаги

1. Откройте [PancakeSwap](https://pancakeswap.finance/swap)
2. Подключите ваш кошелёк
3. В поле "Куда" введите адрес контракта DFT:

```
0x1266e922fe34459efda34e7ee5caf327fbf138d7
```

4. Введите сумму и нажмите **"Swap"**

> 💡 **Примечание**: LP-сделки на PancakeSwap облагаются **налогом в 2,5%** (1% комиссия разработчикам + 1,5% маркетинговая комиссия).

---

## Способ 2: Ежедневный майнинг (Бесплатно)

Получайте бесплатное количество DFT каждый день:

1. Откройте [darkforest.uk](https://darkforest.uk)
2. Подключите ваш кошелёк
3. Нажмите **"Ежедневное получение"**
4. Если `distribute()` ещё не был вызван за сегодня, вы можете запустить его сами
5. Подтвердите транзакцию (только газовая комиссия)

> ⚠️ **Получите в тот же день или потеряете.** DailyMinter не накапливает невостребованные DFT — остатки сжигаются при следующем ежедневном распределении.

## Просмотр баланса DFT

Чтобы добавить DFT в MetaMask:

1. Откройте MetaMask → **"Импорт токенов"**
2. Адрес контракта: `0x1266e922fe34459efda34e7ee5caf327fbf138d7`
3. Символ автоматически отобразится как **DFT**, с **18** десятичными знаками
4. Нажмите **"Добавить"**
