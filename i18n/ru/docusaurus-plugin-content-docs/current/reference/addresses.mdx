## Архитектура контрактов

```
DarkForest (вход DELEGATECALL)
├── DELEGATECALL → DarkForestGameplay   (создание цив., улучшение систем)
├── DELEGATECALL → DarkForestBattle      (атака, ремонт щита, реген. щита)
├── DELEGATECALL → DarkForestMovement    (космический прыжок, прыжок-преследование, круиз)
└── DELEGATECALL → DarkForestAdmin       (сбор, ремонт, восстановление, тотем, вывод)

DarkForestStorage (общая структура хранения)
→ Все спутниковые контракты наследуют DarkForestStorage для единообразной структуры
```
