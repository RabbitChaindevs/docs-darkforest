# Документация Dark Forest

Dark Forest — это полностью ончейновая 3D-стратегическая игра о межзвёздных цивилизациях (SLG), развёрнутая на **BNB Smart Chain**. Вся игровая логика работает через смарт-контракты с архитектурой DELEGATECALL для модульных обновлений.
