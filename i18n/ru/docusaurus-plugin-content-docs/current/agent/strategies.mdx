# Шаблоны стратегий

Стратегии агента определяются с помощью YAML-файлов конфигурации.

---

## Структура файла стратегии

```yaml
agent:
  name: "my-agent"
  mode: "auto"          # auto | manual | dry-run

limits:                 # Автоматический выключатель (ончейн-политика)
  min_energy_reserve: 100000
  min_dft_reserve: 10
  max_attacks_per_hour: 5
  max_upgrades_per_day: 3
  max_single_upgrade_cost: 100

combat:                 # Боевая стратегия
  min_target_level: 1
  max_target_level: 999
  weapon_level: 1
  only_attack_when_energy_above: 100000
  blacklist: []         # Адреса в чёрном списке
  whitelist: []         # Адреса в белом списке (атаковать только эти)

defense:                # Оборонительная стратегия
  auto_repair: true
  min_durability_before_repair: 20
  shield_regen_threshold: 50   # Начать реген. щита ниже этого %

collection:             # Стратегия сбора
  enabled: true
  interval_minutes: 60
```

## Четыре предустановленные стратегии

### 1. Агрессивная

```yaml
combat:
  max_attacks_per_hour: 10
  min_target_level: 0.5
  weapon_level: max

collection:
  enabled: false  # Нет сбора, только бой
```

- ✅ Подходит для: очень активных игроков, желающих боевых наград
- ❌ Риск: высокое потребление энергии, риск контратаки

### 2. Сбалансированная

```yaml
combat:
  max_attacks_per_hour: 5
  min_target_level: 0.8
  weapon_level: balanced

collection:
  enabled: true
  interval_minutes: 30

defense:
  auto_repair: true
  shield_regen_threshold: 60
```

- ✅ **Рекомендуется**: сбалансированная атака и защита, подходит большинству игроков

### 3. Оборонительная

```yaml
combat:
  max_attacks_per_hour: 1
  only_counterattack: true  # Только контратака

defense:
  auto_repair: true
  min_durability_before_repair: 50
  shield_regen_threshold: 80

collection:
  enabled: true
  interval_minutes: 15
```

- ✅ Подходит для: игроков, которые долго находятся офлайн и хотят избежать риска

### 4. Фермерская

```yaml
combat:
  enabled: false  # Нет боя

defense:
  auto_repair: true
  shield_regen_threshold: 90

collection:
  enabled: true
  interval_minutes: 5
```

- ✅ Подходит для: развития на ранних этапах и накопления ресурсов

## Пользовательская стратегия

```bash
# Скопируйте шаблон
cp templates/balanced.yaml my-strategy.yaml

# Отредактируйте параметры
vim my-strategy.yaml

# Используйте пользовательскую стратегию
python -m dark_forest.cli run --strategy ./my-strategy.yaml
```

---

## Параметры автоматического выключателя

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `max_attacks_per_hour` | Макс. атак в час | 10 |
| `min_energy_reserve` | Минимальный резерв энергии | 100 000 |
| `min_dft_reserve` | Минимальный резерв DFT | 10 |
| `max_upgrades_per_day` | Макс. улучшений в день | 5 |

> ⚠️ Параметры автоматического выключателя применяются ончейн. Агент не может их обойти.
