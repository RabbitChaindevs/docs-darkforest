# Что такое ИИ-агент

ИИ-агент — это **офчейн-автоматизированная программа**, выполняющая игровые операции от имени вашей цивилизации. Он авторизуется через **маску разрешений** и действует в рамках **политических ограничений**.

---

## Что может агент

Возможности агента контролируются **авторизованными разрешениями**:

| Разрешение | Значение | Действие |
|-----------|---------|----------|
| COLLECT | 1 | Сбор энергии |
| REPAIR | 2 | Ремонт систем |
| CLAIM_COMBAT | 8 | Получение боевой энергии |
| ATTACK | 32 | Атака |
| UPGRADE | 64 | Улучшение систем |
| JUMP | 128 | Космический прыжок |
| REBUILD | 256 | Восстановление цивилизации |
| DONATE | 512 | Пожертвование тотему |

Вы можете предоставить агенту любую комбинацию разрешений (битовая маска).

---

## Политические ограничения

В дополнение к разрешениям вы можете установить **политические ограничения** для агента:

| Ограничение | Описание |
|------------|----------|
| `minEnergyReserve` | Минимальный резерв энергии (агент не будет тратить ниже этого) |
| `minDftReserve` | Минимальный резерв DFT |
| `maxAttackPerHour` | Максимум атак в час |
| `maxUpgradePerDay` | Максимум улучшений в день |
| `maxSingleUpgradeCost` | Максимальная стоимость DFT за одно улучшение |
| `autoCollect` | Автосбор энергии |
| `autoRepair` | Авторемонт систем |
| `autoShieldRepair` | Авторемонт щита |
| `autoUpgrade` | Автоулучшение |
| `autoAttack` | Автоатака |

## Механизмы безопасности

### 1. Ончейн-регистрация

Политика агента хранится в контракте `DarkForestAgentRegistry.sol` и не может быть изменена после установки.

### 2. Разделение разрешений

- **Авторизация**: Вы авторизуете агента действовать от вашего имени
- **Маска разрешений**: Тонкий контроль над каждой операцией
- **Отзываемость**: `revokeAgent()` вступает в силу немедленно

### 3. Автоматические выключатели

```
Лимит частоты атак: макс N в час
Лимит частоты улучшений: макс N в день
Минимум энергии: никогда ниже X
Минимум DFT: никогда ниже Y
```

### 4. Чёрный список / Белый список

- **Чёрный список**: Запретить атаку конкретных игроков
- **Белый список**: Разрешить атаку только определённых игроков (чёрный список игнорируется, когда активен белый список)

## Агент vs ручная игра

| Сравнение | Вручную | ИИ-агент |
|-----------|---------|----------|
| Требование онлайн | Нужно быть часто онлайн | Работа 24/7 |
| Время реакции | Задержка | Мгновенно |
| Точность | Подвержено ошибкам | Строго следует политике |
| Безопасность | Приватный ключ в кошельке | Зашифрованное хранение + ончейн-ограничения |
