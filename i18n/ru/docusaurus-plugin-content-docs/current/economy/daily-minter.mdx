# Ежедневный майнинг

Контракт **DailyMinter** создаёт фиксированное количество DFT каждый день и распределяет его поровну среди активных цивилизаций. **Получите в тот же день или потеряете** — невостребованные суммы не накапливаются.

---

## Как получить

### Требования

- Вы должны создать цивилизацию **до полуночи UTC** в день получения
- `distribute()` должен быть вызван за этот день (любой может его запустить)
- Каждый адрес может получить только один раз в день

### Операция

```
claim()
```

Подтвердите транзакцию (только газовая комиссия).

---

## Процесс майнинга

### Шаг 1: Распределение

Любой может вызвать `distribute()`:

1. Контракт проверяет последнюю нераспределённую эпоху
2. Делает снимок текущего количества активных цивилизаций
3. Вычисляет `доля на игрока = дневная эмиссия / активные цивилизации`
4. Создаёт общее количество DFT в контракте
5. `Дневная эмиссия = 1 152 575 342 DFT` (константа)

### Шаг 2: Получение

Игроки вызывают `claim()`:

1. Проверяет, что вы создали цивилизацию до этой эпохи
2. Проверяет, что вы ещё не получили
3. Переводит вашу долю в ваш кошелёк

### Шаг 3: Сжигание остатка

При следующем `distribute()` любой DFT из предыдущей эпохи, который не был получен, **сжигается** (уменьшая циркулирующее предложение).

---

## Временная шкала

```
Эпоха 0      Эпоха 1      Эпоха 2
   |            |            |
UTC 00:00    UTC 00:00    UTC 00:00
   ↑создать цив ↑распределить ↑распределить
                ↑получить     ↑получить
                              ↑неполученное сожжено
```

- Эпохи разделены полуночью UTC
- Вы можете получить только **самую последнюю распределённую** эпоху
- Пропустите эпоху — и вы никогда не сможете её получить

---

## Ключевые параметры

| Параметр | Значение |
|----------|----------|
| Дневная эмиссия | **1 152 575 342 DFT** (константа) |
| Общий период распределения | **3 650 дней** (~10 лет) |
| Общий лимит эмиссии | **4 206 900 000 000 DFT** (~4,2 триллиона) |
| Окно получения | В пределах каждой эпохи (обычно 1 день) |

---

## Зачем получать ежедневно

1. 🆓 **Бесплатный DFT** — только газовая комиссия, доступен каждый день
2. 📉 **Неполученное сжигается** — пропустите, и оно исчезнет
3. 📈 **Меньше игроков = больше на человека** — если активных игроков становится меньше, доля каждого увеличивается

> 💡 Установите ежедневное напоминание около времени UTC для получения. Если `distribute()` ещё не был вызван за сегодня, вы можете запустить его сами — это стоит только газовой комиссии.
