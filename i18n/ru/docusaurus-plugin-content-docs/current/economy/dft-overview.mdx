# Обзор токена DFT

**DFT (Dark Forest Token)** — это основной токен экосистемы Dark Forest, реализованный как стандартный токен ERC-20.

---

## Основная информация

| Свойство | Значение |
|----------|----------|
| Адрес контракта | `0x1266e922fe34459efda34e7ee5caf327fbf138d7` |
| Название токена | DarkForest |
| Символ | DFT |
| Десятичные знаки | 18 |
| Общий лимит эмиссии | **4 206 900 000 000 DFT** (~4,2 триллиона) |
| Период распределения | ~10 лет (3 650 дней) |

---

## Использование DFT

| Применение | Описание |
|------------|----------|
| 🚀 **Космический прыжок** | `10 + 10 × √(количество прыжков)` DFT, макс 1 000 |
| 🔧 **Улучшение систем** | Стоимость на основе формулы якоря |
| 🔄 **Восстановление цивилизации** | `100 + 10 × количество возрождений` DFT, макс 2 000 |
| 🏛️ **Штраф за выход из альянса** | `100 + 10 × количество участников` DFT |
| ⚡ **Рынок энергии** | Покупка энергии через книгу ордеров с использованием DFT |

---

## Токеномика

### Предложение

- **Жёсткий потолок**: 4 206 900 000 000 DFT (~4,2 триллиона)
- **Ежедневная эмиссия**: `4 206 900 000 000 / 3 650 ≈ 1 152 575 342 DFT` (постоянная дневная эмиссия)
- **Ежедневное распределение**: через DailyMinter активным игрокам

### Налог на транзакции

При торговле DFT на DEX (PancakeSwap):

| Налог | Ставка | Назначение |
|-------|--------|------------|
| Комиссия разработки | 1% | Команда |
| Маркетинговая комиссия | 1,5% | Маркетинг |
| **Общий налог на транзакцию** | **2,5%** | |

Налог применяется только при торговле в **зарегистрированных пулах ликвидности (LP)**. Обычные переводы не облагаются налогом.

## Просмотр DFT в кошельке

Чтобы добавить DFT в MetaMask:

1. Откройте MetaMask → **"Импорт токенов"**
2. Адрес контракта: `0x1266e922fe34459efda34e7ee5caf327fbf138d7`
3. Символ автоматически отобразится как **DFT**
4. Нажмите **"Добавить"**

**Рекомендуемый следующий шаг: [Рынок энергии →](/docs/economy/energy-market)**
