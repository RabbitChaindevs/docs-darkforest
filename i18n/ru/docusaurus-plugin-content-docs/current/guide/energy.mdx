# Система энергии

Энергия — самый основной ресурс в Dark Forest. Почти все операции — улучшения, атаки, передвижение и ремонт — требуют энергии.

---

## Получение энергии

### 1. Сборщик энергии (Пассивная выработка)

```
Скорость сбора = 3 + 10 × √(уровень сборщика - 1)   эн./с
```

За каждого приглашённого вами игрока скорость дополнительно увеличивается.

**Прочность сборщика**: Сборщик теряет 1 прочность в секунду.

| Уровень сборщика | Выработка в секунду | Макс. прочность сборщика |
|-----------------|--------------------|-------------------------|
| Ур 1 | 3 | 1 день |
| Ур 10 | ~33 | 1 день + 18 часов |
| Ур 50 | ~73 | 1 день + 4 дня |
| Ур 100 | ~103 | 1 день + 8,3 дня |

> ⚠️ Когда прочность сборщика заканчивается, производство останавливается. Вам нужно отремонтировать его, чтобы возобновить сбор.

### 2. Боевой грабёж

После победы в бою над другой цивилизацией вы грабите часть её энергии:

```
Коэффициент грабежа = 5% + уровень оружия × 0,5%
```

Ограбленная энергия поступает в **ожидающий пул** (pendingCombatEnergy), который вы должны забрать вручную.

### 3. Награды за приглашения

За каждого успешно приглашённого нового игрока (он вводит ваш адрес при создании цивилизации):

- **Вы** получаете 150 энергии
- **Приглашённый** получает 150 энергии

### 4. Рынок энергии

См. [Рынок энергии](/docs/economy/energy-market).

## Трата энергии

| Использование | Стоимость |
|---------------|-----------|
| **Ежедневное обслуживание** | `2 000 × общий уровень` / день |
| **Улучшение систем** | Формула якоря (см. руководство по цивилизации) |
| **Атака** | `50 000 + 50 000 × уровень оружия` |
| **Космический прыжок** | `200 000 + 200 000 × √(количество прыжков)`, макс 16,5М |
| **Ремонт сборщика** | `1 энергия / секунда` |
| **Ремонт оружия** | `3 энергии / единица` |
| **Ремонт щита** | `4 энергии / единица` |
| **Ремонт двигателя** | `5 энергии / единица` |
| **Регенерация щита** | `количество регенерации × 1` (авторегенерация) |
| **Ремонт щита** | `количество ремонта × 4` (ручной ремонт) |
| **Помощь щита альянса** | `количество ремонта × 1` (оплачивается союзником) |
| **Пожертвование тотему** | Произвольная сумма |
| **Восстановление** | База 500 000, увеличивается с количеством возрождений |

## Управление энергией

### Ежедневное обслуживание

Обслуживание списывается пропорционально с течением времени при любом действии:

```
Текущее обслуживание = 2 000 × общий уровень × (время с последнего действия / 86400)
```

### Зарезервированная энергия

Энергия, размещённая в ордерах на продажу на рынке энергии, блокируется (`energyReserved`). Обслуживание перестаёт списываться, когда `энергия - зарезервированная` достигает нуля.

### Советы

1. **Держите энергию > 1 месяца обслуживания** — предотвращает частое истощение от рутинных действий
2. **В первую очередь улучшайте сборщик** — сборщик высокого уровня — ключ к пассивному доходу
3. **Рассчитывайте затраты перед атакой** — убедитесь, что грабёж стоит вложений
4. **Регулярно ремонтируйте прочность сборщика** — избегайте перерывов в сборе
