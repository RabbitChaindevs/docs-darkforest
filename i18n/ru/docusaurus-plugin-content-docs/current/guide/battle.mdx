# Атака и защита

## Условия атаки

### Кого можно атаковать?

- Цель должна существовать и не быть уничтоженной (не в состоянии руин)
- Цель не должна находиться в окне **24-часовой защиты нового игрока**
- Цель должна быть в пределах **радиуса сканирования вашего радара** (расстояние ≤ ваш радиус радара)
- Нельзя атаковать себя
- Нельзя атаковать членов своего альянса (нужно сначала выйти)
- Нельзя атаковать того, кто покинул ваш альянс в последние **24 часа** (взаимное охлаждение)

### Мягкий шлюз (Soft Gate)

Если условие атаки не выполнено, попытка **не расходует никаких ресурсов** — просто показывается причина отказа. Это позволяет свободно экспериментировать, не беспокоясь о потраченной энергии.

---

## Стоимость атаки

Каждая успешная атака ограничена **1 атакой в секунду** и потребляет:

| Ресурс | Стоимость |
|--------|-----------|
| **Энергия** | `50 000 + 50 000 × уровень оружия` |
| **Прочность оружия** | 1 единица (когда прочность оружия > 0) |
| **Токен атаки** | 1 токен |

---

## Система токенов атаки

Токены атаки используют **алгоритм «ведро токенов»** для ограничения частоты атак:

| Параметр | Формула | Описание |
|----------|---------|----------|
| **Макс. токенов** | `3 + уровень оружия / 10`, макс **10** | Автовосстановление со временем |
| **Интервал восстановления** | `(300 - уровень оружия × 10) / 100` сек, мин **1с** | 1 токен за интервал |
| **Стоимость** | 1 токен за атаку | |

> 💡 Чем выше уровень вашего оружия, тем быстрее восстанавливаются токены атаки и выше их максимум.

## Расчёт урона

### Атакующий

```
Базовая атака = 900 + 10 × уровень оружия атакующего²
Если прочность оружия = 0, атака уменьшается вдвое
Итоговая атака = Базовая атака × (1 + боевой бонус / 10000)
```

### Защищающийся

```
Базовая защита = 540 + 6 × уровень щита защищающегося²
Бонус защиты альянса = (союзники - 1) × 8 × (1 + уровень тотема × 0,5%)
Итоговая защита = Базовая защита × (1 + боевой бонус / 10000) + Бонус альянса
```

### Итоговый урон

```
Урон по щиту = Атака + 200 - Защита (если > 0)
Урон по здоровью = (Атака - Защита) × 2 (если > 0)

Если урон по щиту > оставшегося HP щита:
  избыток × 3 преобразуется в урон по здоровью
```

## Грабеж

После победы в бою вы грабите энергию цели:

```
Коэффициент грабежа = 5% + уровень оружия × 0,5%
Ограбленная энергия = текущая энергия цели × коэффициент грабежа
```

Ограбленная энергия поступает в **ожидающий пул боевой энергии**, который вы должны забрать вручную.

---

## Поражение и уничтожение

### Поражение

Когда здоровье цели достигает нуля:

1. **Вся оставшаяся энергия грабится** в ваш ожидающий пул
2. У цели **понижается** одна случайная система (на BSC: понижается самая высокоуровневая система; на цепях со случайностью маяка: случайный выбор из Сборщик/Оружие/Щит/Двигатель)
3. Если уровень пониженной системы всё ещё ≥ 1, цивилизация выживает, и её прочность восстанавливается до 50%
4. Если все системы 1-го уровня (дальнейшее понижение невозможно), цивилизация становится **руинами**
5. Грабеж за уничтожение добавляется **поверх** обычного грабежа за атаку

### Руины

Цивилизация в руинах:
- Удаляется из списка активных цивилизаций
- Не может быть атакована
- Может быть возрождена через функцию **Восстановления**

### Боевой бонус (Бонус возрождения)

Каждое восстановление даёт **+1200 б.п.** (12%) боевого бонуса, суммируясь до **30000 б.п.** (множитель ×3). И атака, и защита получают этот бонус.

## Регенерация щита

### Автоматическая регенерация

```
Количество регенерации = 50 + уровень щита²
Стоимость энергии = количество регенерации (соотношение 1:1)
```

### Ручной ремонт

```
Стоимость энергии = количество ремонта × 4
```

### Помощь альянса

```
Стоимость энергии = количество ремонта × 1 (оплачивается союзником)
Союзники могут ремонтировать щиты друг друга
```

## Стратегия защиты

1. **Держите резервы энергии > 7× ежедневное обслуживание** — предотвращает истощение энергии, которое остановит регенерацию щита
2. **Регулярно ремонтируйте прочность** — системы с нулевой прочностью крайне уязвимы
3. **Повышайте уровень щита** — HP и защита щита масштабируются с квадратом уровня
4. **Вступайте в альянс и улучшайте тотем** — бонусы защиты альянса значительны
