# Configuración Rápida en 5 Minutos

## Prerrequisitos

- Python 3.10+
- Una civilización activa de Dark Forest
- La clave privada de tu billetera (para que el agente firme transacciones)

---

## Paso 1: Instalar el Agent

```bash
git clone https://github.com/dark-forest/agent.git
cd agent
pip install -r requirements.txt
```

## Paso 2: Importar tu Clave Privada

```bash
python -m dark_forest.cli key import
```

La clave privada se almacena cifrada, nunca en texto plano.

> ⚠️ Se recomienda encarecidamente usar una **billetera de juego dedicada** — no uses tu billetera principal que tiene activos importantes.

## Paso 3: Autorizar el Agent

En la interfaz del juego, establece los permisos para la dirección de la billetera de tu agente:

- Marca los permisos de operación que necesita el agente
- Establece las restricciones de políticas (frecuencia de ataque, reservas de recursos, etc.)

## Paso 4: Elegir una Estrategia

```bash
# Listar estrategias disponibles
python -m dark_forest.cli strategy list

# Usar la estrategia equilibrada (recomendada)
python -m dark_forest.cli strategy select balanced
```

Cuatro estrategias predefinidas:

| Estrategia | Estilo | Mejor para |
|------------|--------|------------|
| `aggressive` | Atacante agresivo | Jugadores muy activos |
| `balanced` | Ataque/defensa equilibrados | **Recomendada**, adecuada para la mayoría |
| `defensive` | Defensa conservadora | Jugadores que suelen estar desconectados |
| `farmer` | Enfoque en recolección | Desarrollo pacífico |

## Paso 5: Iniciar

```bash
# Iniciar (estrategia equilibrada por defecto)
python -m dark_forest.cli run

# Especificar una estrategia
python -m dark_forest.cli run --strategy aggressive
```

## Paso 6: Configurar Notificaciones (Opcional)

### Telegram

```bash
python -m dark_forest.cli config set notify.telegram.token TU_TOKEN_DE_BOT
python -m dark_forest.cli config set notify.telegram.chat_id TU_CHAT_ID
```

### Discord

```bash
python -m dark_forest.cli config set notify.discord.webhook TU_URL_DE_WEBHOOK
```

## Recordatorios de Seguridad

1. 🔐 **Usa una billetera de juego dedicada** con límites razonables de reserva de DFT/energía
2. 📉 **Abre los permisos gradualmente** — comienza solo con COLLECT + REPAIR, luego añade ATTACK una vez que estés seguro de que es estable
3. 🔄 **Revisa regularmente** — consulta los registros e informes del agente
