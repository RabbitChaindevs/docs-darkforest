# Sistema de Tótem

El tótem es un sistema de mejora para toda la alianza. Los miembros donan energía para mejorarlo, otorgando bonificaciones defensivas a todos los miembros.

---

## Cómo Funciona

1. **Donar energía**: Los miembros de la alianza donan energía al pool del tótem
2. **Acumular hasta el umbral**: Una vez que la energía del tótem alcanza el costo de mejora, el líder activa la mejora
3. **Recibir la bonificación**: Todos los miembros obtienen una bonificación defensiva

---

## Donar Energía

### Operación

```
donateToTotem(allianceId, amount)
```

- Cada donación restablece tu temporizador de donación (caduca en 48 horas)
- Solo los miembros con donaciones válidas se benefician de la bonificación del tótem

### Requisitos de Donación

- Cada donación debe espaciarse por un intervalo mínimo (anti-spam)
- La donación caduca después de 48 horas (debes volver a donar para mantenerte activo)

---

## Mejorar el Tótem

### Fórmula de Costo

```
Próximo Costo de Mejora = 20.000.000 × N × (N + 5) / 10
Donde N = nivel actual del tótem + 1
```

| Nivel | Energía Requerida |
|-------|-------------------|
| Nv 0 → 1 | 12.000.000 |
| Nv 1 → 2 | 42.000.000 |
| Nv 2 → 3 | 96.000.000 |
| Nv 10 → 11 | 330.000.000 |

### Operación

Solo el **líder de la alianza** puede activar las mejoras:

```
upgradeTotem(allianceId)
```

La mejora consume energía del pool del tótem y aumenta el nivel del tótem en 1.

---

## Efecto del Tótem

```
Bonificación de Defensa de Alianza = (miembros totales - 1) × 8 × (1 + nivel del tótem × 0.5%)
```

| Nivel del Tótem | Coeficiente de Bonificación |
|-----------------|-----------------------------|
| Nv 0 | 0% (sin bonificación) |
| Nv 10 | +5% |
| Nv 20 | +10% |
| Nv 100 | +50% |

Ejemplo: una alianza de 50 miembros con un tótem de nivel 20:
```
Bonificación = (50-1) × 8 × (1 + 20 × 0.5%) = 392 × 1.1 = 431.2 de defensa
```

---

## Consejos de Estrategia

1. **Los miembros deben donar regularmente** — las donaciones pequeñas y frecuentes son más efectivas que una grande
2. **Los líderes deben mejorar rápidamente** — mejora tan pronto como el pool del tótem alcance el umbral
3. **Alto nivel de tótem + muchos aliados = ventaja defensiva significativa**
