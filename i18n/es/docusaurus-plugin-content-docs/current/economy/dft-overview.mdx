# Visión General del Token DFT

**DFT (Dark Forest Token)** es el token principal del ecosistema Dark Forest, implementado como un token ERC-20 estándar.

---

| Propiedad | Valor |
|-----------|-------|
| Dirección del Contrato | `0x1266e922fe34459efda34e7ee5caf327fbf138d7` |
| Nombre del Token | DarkForest |
| Símbolo | DFT |
| Decimales | 18 |
| Suministro Máximo Total | **4.206.900.000.000 DFT** (~4,2 billones) |
| Período de Distribución | ~10 años (3.650 días) |

## Tokenomics

### Suministro

- **Límite duro**: 4.206.900.000.000 DFT (~4,2 billones)
- **Acuñación diaria**: `4.206.900.000.000 / 3.650 ≈ 1.152.575.342 DFT` (emisión diaria constante)
- **Distribución diaria**: a través de DailyMinter para jugadores activos

### Impuesto de Transacción

Cuando DFT se negocia en un DEX (PancakeSwap):

| Impuesto | Tasa | Propósito |
|----------|------|-----------|
| Comisión de Desarrollo | 1% | Equipo |
| Comisión de Marketing | 1,5% | Marketing |
| **Impuesto Total de Transacción** | **2,5%** | |

El impuesto solo se aplica al comerciar en **pools de liquidez registrados** (LP). Las transferencias regulares no tienen impuestos.

## Ver DFT en tu Billetera

Para añadir DFT a MetaMask:

1. Abre MetaMask → **"Import Tokens"**
2. Dirección del contrato: `0x1266e922fe34459efda34e7ee5caf327fbf138d7`
3. El símbolo se mostrará automáticamente como **DFT**
4. Haz clic en **"Add"**
