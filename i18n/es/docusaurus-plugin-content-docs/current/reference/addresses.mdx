## Arquitectura de Contratos

```
DarkForest (entrada DELEGATECALL)
├── DELEGATECALL → DarkForestGameplay   (crear civ, mejorar sistemas)
├── DELEGATECALL → DarkForestBattle      (atacar, reparar escudo, regenerar escudo)
├── DELEGATECALL → DarkForestMovement    (salto espacial, salto de seguimiento, crucero)
└── DELEGATECALL → DarkForestAdmin       (recolectar, reparar, reconstruir, tótem, retirar)

DarkForestStorage (disposición de almacenamiento compartido)
→ Todos los contratos satélite heredan de DarkForestStorage para una disposición consistente
```
