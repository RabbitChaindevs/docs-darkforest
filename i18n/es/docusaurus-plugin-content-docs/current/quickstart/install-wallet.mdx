# Instalar una Billetera

Antes de poder jugar Dark Forest, necesitas una billetera Web3 para gestionar tus activos digitales y firmar transacciones en cadena.

## ¿Qué es una Billetera Web3?

Una billetera Web3 es tu llave para las aplicaciones descentralizadas (dApps). Te permite:

- **Almacenar y gestionar** activos digitales como DFT y BNB
- **Firmar transacciones** — cada acción en el juego (construir, moverse, luchar) requiere la firma de tu billetera
- **Autenticar tu identidad** — la dirección de tu billetera es tu identidad en Dark Forest

## Billeteras Recomendadas

### MetaMask (Recomendada)

MetaMask es la billetera Web3 más popular, disponible como extensión de navegador y aplicación móvil.

#### Instalación en Navegador

1. Visita el [sitio web de MetaMask](https://metamask.io)
2. Haz clic en "Descargar" para obtener la extensión del navegador
3. Elige tu navegador (Chrome / Firefox / Edge / Brave)
4. Haz clic en "Añadir al Navegador"
5. Sigue el asistente para crear una nueva billetera

:::tip
Después de instalar, fija la extensión de MetaMask en la barra de herramientas de tu navegador para acceso rápido.
:::

#### Crear una Billetera

1. Haz clic en el icono de MetaMask y selecciona "Crear una Billetera"
2. Establece una **contraseña segura** (al menos 8 caracteres, combinación de letras y números)
3. **Respaldar tu frase semilla de forma segura** — ¡esta es la única forma de recuperar tu billetera!

:::danger Seguridad de la Frase Semilla
- **Nunca** compartas tu frase semilla con nadie
- **No** tomes capturas de pantalla ni la almacenes en la nube
- Usa medios físicos (papel y bolígrafo) para respaldo offline
- Cualquiera que te pida tu frase semilla es un estafador
:::

4. Confirma tu frase semilla para completar la creación de la billetera

### Trust Wallet

Trust Wallet es la billetera móvil oficial de Binance.

1. Busca "Trust Wallet" en la App Store o Google Play e instálala
2. Abre la aplicación y selecciona "Crear una Nueva Billetera"
3. Respaldar tu frase semilla de forma segura
4. Habilita el navegador DApp dentro de la billetera
