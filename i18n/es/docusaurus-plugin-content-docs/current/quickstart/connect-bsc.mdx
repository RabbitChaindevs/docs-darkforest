# Conectar a BSC

Dark Forest está desplegado en **BNB Smart Chain (BSC)**. Necesitas cambiar tu billetera a la red BSC para interactuar con el juego.

## ¿Qué es BSC?

BNB Smart Chain es una red blockchain lanzada por Binance, que ofrece:

- **Tarifas de transacción bajas** — cada operación cuesta solo fracciones de un centavo
- **Alto rendimiento** — diseñada para interacciones de juego de alta frecuencia
- **Compatible con EVM** — funciona con MetaMask y otras billeteras populares

## Añadir Red BSC Automáticamente

La forma más fácil es usar Chainlist:

1. Visita [Chainlist](https://chainlist.org)
2. Busca "BNB Smart Chain"
3. Haz clic en "Connect Wallet" y autoriza
4. Haz clic en "Add to MetaMask"
5. Confirma la adición

## Configuración Manual

También puedes añadir la red BSC manualmente:

### Añadir BSC a MetaMask

1. Haz clic en el menú desplegable de redes en la esquina superior derecha de MetaMask
2. Haz clic en "Add Network"
3. Rellena los siguientes detalles:

| Configuración | Valor |
|--------------|-------|
| **Nombre de Red** | BNB Smart Chain |
| **URL RPC** | `https://bsc-dataseed.binance.org` |
| **Chain ID** | `56` |
| **Símbolo de Moneda** | `BNB` |
| **Explorador de Bloques** | `https://bscscan.com` |

4. Haz clic en "Save"

### Nodos RPC de Respaldo

Si el RPC principal está congestionado, puedes usar nodos de respaldo:

| Proveedor | URL RPC |
|-----------|---------|
| Binance Oficial | `https://bsc-dataseed1.binance.org` |
| Ankr | `https://rpc.ankr.com/bsc` |
| PublicNode | `https://bsc.publicnode.com` |

## Obtener BNB

Cada operación en BSC requiere BNB para las tarifas de gas. Así es como puedes obtener algo:

1. **Exchange centralizado** — compra BNB en Binance, OKX, etc. y retira a la red BSC
2. **Puente entre cadenas** — transfiere BNB desde otras redes a BSC mediante un puente

:::caution
Al retirar, **asegúrate de seleccionar la red BSC (BEP-20)** o podrías perder tus fondos.
:::

## Verificar tu Conexión

Después de la configuración, cambia a la red BNB Smart Chain en tu billetera. Si el Chain ID muestra `56` y tu saldo muestra la cantidad correcta de BNB, estás listo.

Una vez que tu billetera y red estén preparadas, el siguiente paso es [Obtener Tokens DFT](/docs/quickstart/get-dft).
