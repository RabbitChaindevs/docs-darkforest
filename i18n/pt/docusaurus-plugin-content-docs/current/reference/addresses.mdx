## Arquitetura de Contratos

```
DarkForest (entrada DELEGATECALL)
├── DELEGATECALL → DarkForestGameplay   (criar civ, melhorar sistemas)
├── DELEGATECALL → DarkForestBattle      (atacar, reparar escudo, regenerar escudo)
├── DELEGATECALL → DarkForestMovement    (salto espacial, salto de rastreamento, cruzeiro)
└── DELEGATECALL → DarkForestAdmin       (coletar, reparar, reconstruir, totem, retirar)

DarkForestStorage (disposição de armazenamento compartilhado)
→ Todos os contratos satélite herdam DarkForestStorage para layout consistente
```
