# Visão Geral do Token DFT

**DFT (Dark Forest Token)** é o token principal do ecossistema Dark Forest, implementado como um token ERC-20 padrão.

---

## Informações Básicas

| Propriedade | Valor |
|-------------|-------|
| Endereço do Contrato | `0x1266e922fe34459efda34e7ee5caf327fbf138d7` |
| Nome do Token | DarkForest |
| Símbolo | DFT |
| Decimais | 18 |
| Fornecimento Máximo Total | **4.206.900.000.000 DFT** (~4,2 trilhões) |
| Período de Distribuição | ~10 anos (3.650 dias) |

## Tokenomics

### Fornecimento

- **Limite rígido**: 4.206.900.000.000 DFT (~4,2 trilhões)
- **Cunhagem diária**: `4.206.900.000.000 / 3.650 ≈ 1.152.575.342 DFT` (emissão diária constante)
- **Distribuição diária**: via DailyMinter para jogadores ativos

### Imposto de Transação

Quando DFT é negociado em uma DEX (PancakeSwap):

| Imposto | Taxa | Propósito |
|---------|------|-----------|
| Taxa de Desenvolvimento | 1% | Equipe |
| Taxa de Marketing | 1,5% | Marketing |
| **Imposto Total de Transação** | **2,5%** | |

O imposto só se aplica ao negociar em **pools de liquidez registrados** (LP). Transferências regulares não são tributadas.

## Vendo DFT na sua Carteira

Para adicionar DFT ao MetaMask:

1. Abra MetaMask → **"Import Tokens"**
2. Endereço do contrato: `0x1266e922fe34459efda34e7ee5caf327fbf138d7`
3. O símbolo será exibido automaticamente como **DFT**
4. Clique em **"Add"**
