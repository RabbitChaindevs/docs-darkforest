# Mineração Diária

O contrato **DailyMinter** cunha uma quantidade fixa de DFT todos os dias e a distribui igualmente entre as civilizações ativas. **Reivindique no mesmo dia ou perca-o** — os valores não reivindicados não se acumulam.

---

## Como Reivindicar

### Requisitos

- Você deve ter criado uma civilização **antes da meia-noite UTC** do dia da reivindicação
- `distribute()` deve ter sido chamado para aquele dia (qualquer um pode acioná-lo)
- Cada endereço só pode reivindicar uma vez por dia

### Operação

```
claim()
```

Confirme a transação (apenas taxa de gas).

---

## Fluxo de Cunhagem

### Passo 1: Distribuir

Qualquer um pode chamar `distribute()`:

1. O contrato verifica a última época não distribuída
2. Tira um snapshot do número atual de civilizações ativas
3. Calcula `participação por jogador = emissão diária / civilizações ativas`
4. Cunha o DFT total no contrato
5. `Emissão diária = 1.152.575.342 DFT` (constante)

### Passo 2: Reivindicar

Os jogadores chamam `claim()`:

1. Verifica se você criou uma civilização antes desta época
2. Verifica se você já não reivindicou
3. Transfere sua participação para sua carteira

### Passo 3: Queimar o Remanescente

No próximo `distribute()`, qualquer DFT da época anterior que não foi reivindicado é **queimado** (reduzindo o fornecimento circulante).

---

## Cronograma

```
Época 0      Época 1      Época 2
   |            |            |
UTC 00:00    UTC 00:00    UTC 00:00
   ↑criar civ  ↑distribuir   ↑distribuir
               ↑reivindicar  ↑reivindicar
                              ↑não reivindicado queimado
```

- As épocas são divididas pela meia-noite UTC
- Você só pode reivindicar a época **distribuída mais recentemente**
- Se perder uma época, nunca poderá reivindicá-la

---

## Parâmetros Chave

| Parâmetro | Valor |
|-----------|-------|
| Emissão Diária | **1.152.575.342 DFT** (constante) |
| Período Total de Distribuição | **3.650 dias** (~10 anos) |
| Fornecimento Máximo Total | **4.206.900.000.000 DFT** (~4,2 trilhões) |
| Janela de Reivindicação | Dentro de cada época (normalmente 1 dia) |

---

## Por Que Reivindicar Diariamente

1. 🆓 **DFT Grátis** — só custa gas, disponível todos os dias
2. 📉 **Não reivindicado é queimado** — se perder, desaparece
3. 📈 **Menos jogadores = mais por pessoa** — se os jogadores ativos diminuírem, cada participação aumenta

> 💡 Defina um lembrete diário por volta da hora UTC para reivindicar. Se `distribute()` ainda não foi chamado hoje, você mesmo pode acioná-lo — só custa gas.
