# Sistema de Totem

O totem é um sistema de melhoria para toda a aliança. Os membros doam energia para melhorá-lo, concedendo bônus defensivos a todos os membros.

---

## Como Funciona

1. **Doar energia**: Membros da aliança doam energia para o pool do totem
2. **Acumular até o limite**: Quando a energia do totem atinge o custo de upgrade, o líder ativa o upgrade
3. **Receber o bônus**: Todos os membros ganham um bônus defensivo

---

## Doar Energia

### Operação

```
donateToTotem(allianceId, amount)
```

- Cada doação reinicia seu temporizador de doação (expira em 48 horas)
- Apenas membros com doações válidas se beneficiam do bônus do totem

### Requisitos de Doação

- Cada doação deve ser espaçada por um intervalo mínimo (anti-spam)
- A doação expira após 48 horas (deve doar novamente para permanecer ativo)

---

## Melhorar o Totem

### Fórmula de Custo

```
Próximo Custo de Upgrade = 20.000.000 × N × (N + 5) / 10
Onde N = nível atual do totem + 1
```

| Nível | Energia Necessária |
|-------|--------------------|
| Nv 0 → 1 | 12.000.000 |
| Nv 1 → 2 | 42.000.000 |
| Nv 2 → 3 | 96.000.000 |
| Nv 10 → 11 | 330.000.000 |

### Operação

Apenas o **líder da aliança** pode ativar os upgrades:

```
upgradeTotem(allianceId)
```

O upgrade consome energia do pool do totem e aumenta o nível do totem em 1.

---

## Efeito do Totem

```
Bônus de Defesa da Aliança = (membros totais - 1) × 8 × (1 + nível do totem × 0.5%)
```

| Nível do Totem | Coeficiente de Bônus |
|----------------|----------------------|
| Nv 0 | 0% (sem bônus) |
| Nv 10 | +5% |
| Nv 20 | +10% |
| Nv 100 | +50% |

Exemplo: uma aliança de 50 membros com um totem de nível 20:
```
Bônus = (50-1) × 8 × (1 + 20 × 0.5%) = 392 × 1.1 = 431.2 de defesa
```

---

## Dicas de Estratégia

1. **Os membros devem doar regularmente** — doações pequenas e frequentes são mais eficazes que uma grande
2. **Os líderes devem fazer upgrade rapidamente** — melhore assim que o pool do totem atingir o limite
3. **Alto nível de totem + muitos aliados = vantagem defensiva significativa**
