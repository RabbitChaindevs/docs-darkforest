# Sistema de Energia

A energia é o recurso mais fundamental no Dark Forest. Quase todas as operações — upgrades, ataques, movimento e reparos — requerem energia.

---

## Obtendo Energia

### 1. Coletor de Energia (Produção Passiva)

```
Taxa de Coleta = 3 + 10 × √(nível do coletor - 1)   energia/s
```

Para cada jogador que você convidar, a taxa aumenta ainda mais.

**Durabilidade do Coletor**: O coletor perde 1 de durabilidade por segundo.

| Nível do Coletor | Produção por Segundo | Durabilidade Máx. do Coletor |
|------------------|----------------------|-------------------------------|
| Nv 1 | 3 | 1 dia |
| Nv 10 | ~33 | 1 dia + 18 horas |
| Nv 50 | ~73 | 1 dia + 4 dias |
| Nv 100 | ~103 | 1 dia + 8.3 dias |

> ⚠️ Quando a durabilidade do coletor acaba, a produção para. Você precisa repará-lo para que a coleta seja retomada.

### 2. Saque de Combate

Após vencer uma batalha contra outra civilização, você saqueia uma parte da energia dela:

```
Taxa de Saque = 5% + nível de arma × 0.5%
```

A energia saqueada vai para um **pool pendente** (pendingCombatEnergy), que você deve reivindicar manualmente.

### 3. Recompensas por Convite

Para cada novo jogador que você convidar com sucesso (eles inserem seu endereço ao criar sua civilização):

- **Você** recebe 150 de energia
- **O convidado** recebe 150 de energia

### 4. Mercado de Energia

Veja [Mercado de Energia](/docs/economy/energy-market).

## Gastando Energia

| Caso de Uso | Custo |
|-------------|-------|
| **Manutenção Diária** | `2.000 × nível total` / dia |
| **Upgrades de Sistemas** | Fórmula Âncora (veja guia de Civilização) |
| **Ataque** | `50.000 + 50.000 × nível de arma` |
| **Salto Espacial** | `200.000 + 200.000 × √(número de saltos)`, teto 16.5M |
| **Reparar Coletor** | `1 energia / segundo` |
| **Reparar Arma** | `3 energia / ponto` |
| **Reparar Escudo** | `4 energia / ponto` |
| **Reparar Motor** | `5 energia / ponto` |
| **Regeneração de Escudo** | `quantidade de regeneração × 1` (auto regeneração) |
| **Reparo de Escudo** | `quantidade de reparo × 4` (reparo manual) |
| **Assistência de Escudo da Aliança** | `quantidade de reparo × 1` (pago pelo aliado) |
| **Doação ao Totem** | Quantidade personalizada |
| **Reconstruir** | Base 500.000, aumenta com o número de renascimentos |

## Gerenciamento de Energia

### Manutenção Diária

A manutenção é deduzida proporcionalmente ao longo do tempo sempre que você realiza qualquer ação:

```
Manutenção Atual = 2.000 × nível total × (tempo desde a última ação / 86400)
```

### Energia Reservada

A energia colocada em ordens de venda no mercado de energia é bloqueada (`energyReserved`). A manutenção para de ser deduzida quando `energia - reservada` chega a zero.

### Dicas

1. **Mantenha energia > 1 mês de manutenção** — evita o esgotamento frequente por ações rotineiras
2. **Priorize upgrades do coletor** — um coletor de alto nível é chave para a renda passiva
3. **Calcule os custos antes de atacar** — certifique-se de que o saque vale o investimento
4. **Repare a durabilidade do coletor regularmente** — evite interrupções na coleta
