# Documentação do Dark Forest

Dark Forest é um jogo de estratégia (SLG) de civilização interestelar 3D totalmente on-chain, implantado na **BNB Smart Chain**. Toda a lógica do jogo é executada em contratos inteligentes, com uma arquitetura DELEGATECALL para atualizações modulares.
