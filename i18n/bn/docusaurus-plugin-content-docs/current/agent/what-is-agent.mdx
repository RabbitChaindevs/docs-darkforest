# AI এজেন্ট কী

AI এজেন্ট একটি **অফ-চেইন স্বয়ংক্রিয় প্রোগ্রাম** যা আপনার সভ্যতার পক্ষে গেম অপারেশন সম্পাদন করে। এটি **অনুমতি বিটমাস্ক**-এর মাধ্যমে অনুমোদিত এবং **নীতি সীমাবদ্ধতার** মধ্যে কাজ করে।

---

## এজেন্ট কী করতে পারে

এজেন্টের ক্ষমতাগুলি **অনুমোদিত অনুমতি** দ্বারা নিয়ন্ত্রিত হয়:

| অনুমতি | মান | কাজ |
|-----------|-------|--------|
| COLLECT | 1 | শক্তি সংগ্রহ করুন |
| REPAIR | 2 | সিস্টেম মেরামত করুন |
| CLAIM_COMBAT | 8 | যুদ্ধ শক্তি দাবি করুন |
| ATTACK | 32 | আক্রমণ করুন |
| UPGRADE | 64 | সিস্টেম আপগ্রেড করুন |
| JUMP | 128 | স্পেস জাম্প |
| REBUILD | 256 | সভ্যতা পুনর্নির্মাণ |
| DONATE | 512 | টোটেমে দান করুন |

আপনি এজেন্টকে অনুমতির যেকোনো সংমিশ্রণ (বিটমাস্ক) প্রদান করতে পারেন।

---

## নীতি সীমাবদ্ধতা

অনুমতি ছাড়াও, আপনি এজেন্টের জন্য **নীতি সীমাবদ্ধতা** নির্ধারণ করতে পারেন:

| সীমাবদ্ধতা | বিবরণ |
|-----------|-------------|
| `minEnergyReserve` | ন্যূনতম শক্তি মজুদ (এজেন্ট এর নিচে খরচ করবে না) |
| `minDftReserve` | ন্যূনতম DFT মজুদ |
| `maxAttackPerHour` | প্রতি ঘন্টায় সর্বোচ্চ আক্রমণ |
| `maxUpgradePerDay` | প্রতিদিন সর্বোচ্চ আপগ্রেড |
| `maxSingleUpgradeCost` | প্রতি আপগ্রেডে সর্বোচ্চ DFT খরচ |
| `autoCollect` | স্বয়ংক্রিয় শক্তি সংগ্রহ |
| `autoRepair` | স্বয়ংক্রিয় সিস্টেম মেরামত |
| `autoShieldRepair` | স্বয়ংক্রিয় ঢাল মেরামত |
| `autoUpgrade` | স্বয়ংক্রিয় আপগ্রেড |
| `autoAttack` | স্বয়ংক্রিয় আক্রমণ |

## নিরাপত্তা প্রক্রিয়া

### 1. অন-চেইন নিবন্ধন

এজেন্টের নীতি `DarkForestAgentRegistry.sol` কন্ট্র্যাক্টে সংরক্ষিত থাকে এবং একবার সেট করলে তা নিয়ে ছলচল করা যায় না।

### 2. অনুমতি পৃথকীকরণ

- **অনুমোদন**: আপনি এজেন্টকে আপনার পক্ষে কাজ করার অনুমতি দেন
- **অনুমতি বিটমাস্ক**: প্রতিটি অপারেশনের উপর সূক্ষ্ম-নিয়ন্ত্রণ
- **প্রত্যাহারযোগ্য**: `revokeAgent()` তাৎক্ষণিকভাবে কার্যকর হয়

### 3. সার্কিট ব্রেকার

```
আক্রমণ ফ্রিকোয়েন্সি সীমা: প্রতি ঘন্টায় সর্বোচ্চ N
আপগ্রেড ফ্রিকোয়েন্সি সীমা: প্রতিদিন সর্বোচ্চ N
শক্তি ফ্লোর: X-এর নিচে কখনো নয়
DFT ফ্লোর: Y-এর নিচে কখনো নয়
```

### 4. ব্ল্যাকলিস্ট / হোয়াইটলিস্ট

- **ব্ল্যাকলিস্ট**: নির্দিষ্ট খেলোয়াড়দের আক্রমণ নিষিদ্ধ করুন
- **হোয়াইটলিস্ট**: শুধুমাত্র নির্দিষ্ট খেলোয়াড়দের আক্রমণের অনুমতি দিন (হোয়াইটলিস্ট সক্রিয় থাকলে ব্ল্যাকলিস্ট উপেক্ষিত)
