# ৫ মিনিটে দ্রুত শুরু

## পূর্বশর্ত

- Python 3.10+
- একটি সক্রিয় ডার্ক ফরেস্ট সভ্যতা
- আপনার ওয়ালেটের প্রাইভেট কী (এজেন্টের লেনদেনে স্বাক্ষরের জন্য)

---

## ধাপ ১: এজেন্ট ইনস্টল করুন

```bash
git clone https://github.com/dark-forest/agent.git
cd agent
pip install -r requirements.txt
```

## ধাপ ২: আপনার প্রাইভেট কী আমদানি করুন

```bash
python -m dark_forest.cli key import
```

প্রাইভেট কী এনক্রিপ্টেড সংরক্ষিত হয়, কখনো প্লেইন টেক্সটে নয়।

> ⚠️ একটি **ডেডিকেটেড গেম ওয়ালেট** ব্যবহার করার জন্য দৃঢ়ভাবে সুপারিশ করা হয় — আপনার মূল ওয়ালেট ব্যবহার করবেন না যাতে উল্লেখযোগ্য সম্পদ রয়েছে।

## ধাপ ৩: এজেন্টকে অনুমোদন করুন

গেম ইন্টারফেসে, আপনার এজেন্টের ওয়ালেট ঠিকানার জন্য অনুমতি সেট করুন:

- এজেন্টের প্রয়োজনীয় অপারেশন অনুমতি চেক করুন
- নীতি সীমাবদ্ধতা সেট করুন (আক্রমণ ফ্রিকোয়েন্সি, সম্পদ মজুদ, ইত্যাদি)

## ধাপ ৪: একটি কৌশল বেছে নিন

```bash
# উপলব্ধ কৌশল তালিকাভুক্ত করুন
python -m dark_forest.cli strategy list

# ভারসাম্যপূর্ণ কৌশল ব্যবহার করুন (প্রস্তাবিত)
python -m dark_forest.cli strategy select balanced
```

চারটি প্রিসেট কৌশল:

| কৌশল | শৈলী | সেরা জন্য |
|----------|-------|----------|
| `aggressive` | আক্রমণাত্মক | অত্যন্ত সক্রিয় খেলোয়াড় |
| `balanced` | ভারসাম্যপূর্ণ আক্রমণ/প্রতিরক্ষা | **প্রস্তাবিত**, অধিকাংশ খেলোয়াড়ের জন্য উপযোগী |
| `defensive` | রক্ষণশীল প্রতিরক্ষা | যারা প্রায়ই অফলাইনে থাকেন |
| `farmer` | সংগ্রহে মনোযোগ | শান্তিপূর্ণ উন্নয়ন |

## ধাপ ৫: চালু করুন

```bash
# শুরু করুন (ডিফল্ট ভারসাম্যপূর্ণ কৌশল)
python -m dark_forest.cli run

# একটি কৌশল নির্দিষ্ট করুন
python -m dark_forest.cli run --strategy aggressive
```

## ধাপ ৬: বিজ্ঞপ্তি কনফিগার করুন (ঐচ্ছিক)

### Telegram

```bash
python -m dark_forest.cli config set notify.telegram.token YOUR_BOT_TOKEN
python -m dark_forest.cli config set notify.telegram.chat_id YOUR_CHAT_ID
```

### Discord

```bash
python -m dark_forest.cli config set notify.discord.webhook YOUR_WEBHOOK_URL
```
