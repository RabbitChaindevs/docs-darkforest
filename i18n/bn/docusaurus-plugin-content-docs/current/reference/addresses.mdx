## কন্ট্র্যাক্ট আর্কিটেকচার

```
DarkForest (DELEGATECALL প্রবেশ)
├── DELEGATECALL → DarkForestGameplay   (সভ্যতা তৈরি, সিস্টেম আপগ্রেড)
├── DELEGATECALL → DarkForestBattle      (আক্রমণ, ঢাল মেরামত, ঢাল পুনর্জন্ম)
├── DELEGATECALL → DarkForestMovement    (স্পেস জাম্প, ট্র্যাকিং জাম্প, ক্রুজ)
└── DELEGATECALL → DarkForestAdmin       (সংগ্রহ, মেরামত, পুনর্নির্মাণ, টোটেম, উত্তোলন)

DarkForestStorage (ভাগ করা স্টোরেজ লেআউট)
→ সব স্যাটেলাইট কন্ট্র্যাক্ট সামঞ্জস্যপূর্ণ লেআউটের জন্য DarkForestStorage থেকে উত্তরাধিকার লাভ করে
```
