# ডার্ক ফরেস্ট ডকুমেন্টেশন

ডার্ক ফরেস্ট একটি সম্পূর্ণ অন-চেইন 3D আন্তঃনাক্ষত্রিক সভ্যতা কৌশল গেম (SLG) যা **BNB স্মার্ট চেইনে** ডিপ্লয় করা হয়েছে। সমস্ত গেম লজিক স্মার্ট কন্ট্র্যাক্টে চলে, মডুলার আপগ্রেডের জন্য DELEGATECALL আর্কিটেকচার সহ।
