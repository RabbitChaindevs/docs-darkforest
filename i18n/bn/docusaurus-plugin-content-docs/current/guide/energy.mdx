# শক্তি ব্যবস্থা

ডার্ক ফরেস্টে শক্তি হল সবচেয়ে মৌলিক সম্পদ। প্রায় প্রতিটি কাজ — আপগ্রেড, আক্রমণ, চলাচল এবং মেরামত — শক্তির প্রয়োজন।

---

## শক্তি অর্জন

### 1. শক্তি সংগ্রহকারী (নিষ্ক্রিয় আউটপুট)

```
সংগ্রহ হার = 3 + 10 × √(সংগ্রহকারী স্তর - 1)   শক্তি/সে
```

আপনার আমন্ত্রিত প্রতিটি খেলোয়াড়ের জন্য, হার আরও বৃদ্ধি পায়।

**সংগ্রহকারী স্থায়িত্ব**: সংগ্রহকারী প্রতি সেকেন্ডে 1 স্থায়িত্ব হারায়।

| সংগ্রহকারী স্তর | প্রতি সেকেন্ডে আউটপুট | সর্বোচ্চ সংগ্রহকারী স্থায়িত্ব |
|----------------|-------------------|--------------------------|
| Lv 1 | 3 | 1 দিন |
| Lv 10 | ~33 | 1 দিন + 18 ঘন্টা |
| Lv 50 | ~73 | 1 দিন + 4 দিন |
| Lv 100 | ~103 | 1 দিন + 8.3 দিন |

> ⚠️ যখন সংগ্রহকারীর স্থায়িত্ব শেষ হয়, উৎপাদন বন্ধ হয়ে যায়। সংগ্রহ পুনরায় শুরু করতে আপনাকে এটি মেরামত করতে হবে।

### 2. যুদ্ধ লুট

অন্য সভ্যতার বিরুদ্ধে যুদ্ধ জয়ের পর, আপনি তাদের শক্তির একটি অংশ লুট করেন:

```
লুটের অনুপাত = 5% + অস্ত্র স্তর × 0.5%
```

লুটকৃত শক্তি একটি **বিচারাধীন পুলে** (pendingCombatEnergy) যায়, যা আপনাকে ম্যানুয়ালি দাবি করতে হবে।

### 3. আমন্ত্রণ পুরস্কার

প্রত্যেক নতুন খেলোয়াড়ের জন্য যাকে আপনি সফলভাবে আমন্ত্রণ জানান (তারা তাদের সভ্যতা তৈরি করার সময় আপনার ঠিকানা প্রবেশ করান):

- **আপনি** 150 শক্তি পান
- **আমন্ত্রিত ব্যক্তি** 150 শক্তি পান

### 4. শক্তি বাজার

দেখুন [শক্তি বাজার](/docs/economy/energy-market)।

## শক্তি ব্যয়

| ব্যবহার | খরচ |
|----------|------|
| **দৈনিক রক্ষণাবেক্ষণ** | `2,000 × মোট স্তর` / দিন |
| **সিস্টেম আপগ্রেড** | অ্যাঙ্কর সূত্র (সভ্যতা গাইড দেখুন) |
| **আক্রমণ** | `50,000 + 50,000 × অস্ত্র স্তর` |
| **স্পেস জাম্প** | `200,000 + 200,000 × √(জাম্প গণনা)`, সীমা 16.5M |
| **সংগ্রহকারী মেরামত** | `1 শক্তি / সেকেন্ড` |
| **অস্ত্র মেরামত** | `3 শক্তি / পয়েন্ট` |
| **ঢাল মেরামত** | `4 শক্তি / পয়েন্ট` |
| **ইঞ্জিন মেরামত** | `5 শক্তি / পয়েন্ট` |
| **ঢাল পুনর্জন্ম** | `পুনর্জন্ম পরিমাণ × 1` (স্বয়ংক্রিয়) |
| **ঢাল মেরামত (ম্যানুয়াল)** | `মেরামত পরিমাণ × 4` |
| **জোট ঢাল সহায়তা** | `মেরামত পরিমাণ × 1` (সহযোগী দ্বারা) |
| **টোটেম দান** | কাস্টম পরিমাণ |
| **পুনর্নির্মাণ** | ভিত্তি 500,000, পুনর্জন্ম গণনার সাথে বৃদ্ধি পায় |

## শক্তি ব্যবস্থাপনা

### দৈনিক রক্ষণাবেক্ষণ

যখনই আপনি কোনো কাজ করেন তখন রক্ষণাবেক্ষণ সময়ানুপাতিকভাবে কাটা হয়:

```
বর্তমান রক্ষণাবেক্ষণ = 2,000 × মোট স্তর × (শেষ কাজের পর সময় / 86400)
```

### সংরক্ষিত শক্তি

শক্তি বাজারে বিক্রয় অর্ডারে রাখা শক্তি লক করা হয় (`energyReserved`)। একবার `energy - reserved` শূন্য হলে রক্ষণাবেক্ষণ কাটা বন্ধ হয়।

### টিপস

1. **শক্তি > 1 মাসের রক্ষণাবেক্ষণ রাখুন** — নিয়মিত কাজ থেকে ঘন ঘন হ্রাস রোধ করে
2. **সংগ্রহকারী আপগ্রেডকে অগ্রাধিকার দিন** — উচ্চ-স্তরের সংগ্রহকারী নিষ্ক্রিয় আয়ের চাবিকাঠি
3. **আক্রমণের আগে খরচ গণনা করুন** — নিশ্চিত করুন লুট বিনিয়োগের উপযুক্ত
4. **নিয়মিত সংগ্রহকারীর স্থায়িত্ব মেরামত করুন** — সংগ্রহে বাধা এড়ান
