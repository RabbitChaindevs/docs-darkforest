# DFT টোকেন ওভারভিউ

**DFT (ডার্ক ফরেস্ট টোকেন)** হল ডার্ক ফরেস্ট ইকোসিস্টেমের মূল টোকেন, যা একটি স্ট্যান্ডার্ড ERC-20 টোকেন হিসেবে প্রয়োগ করা হয়েছে।

---

## মৌলিক তথ্য

| বৈশিষ্ট্য | মান |
|----------|-------|
| কন্ট্র্যাক্ট ঠিকানা | `0x1266e922fe34459efda34e7ee5caf327fbf138d7` |
| টোকেন নাম | DarkForest |
| প্রতীক | DFT |
| দশমিক | 18 |
| মোট সরবরাহ সীমা | **4,206,900,000,000 DFT** (~৪.২ ট্রিলিয়ন) |
| বিতরণ সময়কাল | ~১০ বছর (৩,৬৫০ দিন) |

## DFT-এর ব্যবহার

| ব্যবহার | বিবরণ |
|----------|-------------|
| 🚀 **স্পেস জাম্প** | `10 + 10 × √(জাম্প গণনা)` DFT, সীমা ১,০০০ |
| 🔧 **সিস্টেম আপগ্রেড** | অ্যাঙ্কর সূত্র-ভিত্তিক আপগ্রেড খরচ |
| 🔄 **সভ্যতা পুনর্নির্মাণ** | `100 + 10 × পুনর্জন্ম গণনা` DFT, সীমা ২,০০০ |
| 🏛️ **জোট ত্যাগ জরিমানা** | `100 + 10 × সদস্য সংখ্যা` DFT |
| ⚡ **শক্তি বাজার** | DFT ব্যবহার করে অর্ডার বুকের মাধ্যমে শক্তি কেনা |

## টোকেনোমিক্স

### সরবরাহ

- **হার্ড ক্যাপ**: ৪,২০৬,৯০০,০০০,০০০ DFT (~৪.২ ট্রিলিয়ন)
- **দৈনিক মিন্ট**: `4,206,900,000,000 / 3,650 ≈ 1,152,575,342 DFT` (স্থির দৈনিক নির্গমন)
- **দৈনিক বিতরণ**: DailyMinter-এর মাধ্যমে সক্রিয় খেলোয়াড়দের কাছে

### লেনদেন কর

যখন DFT একটি DEX-এ (PancakeSwap) লেনদেন করা হয়:

| কর | হার | উদ্দেশ্য |
|-----|------|---------|
| উন্নয়ন ফি | 1% | টিম |
| মার্কেটিং ফি | 1.5% | মার্কেটিং |
| **মোট লেনদেন কর** | **২.৫%** | |

কর শুধুমাত্র **নিবন্ধিত লিকুইডিটি পুলে** (LP) লেনদেন করার সময় প্রযোজ্য। নিয়মিত স্থানান্তর করমুক্ত।

## আপনার ওয়ালেটে DFT দেখা

MetaMask-এ DFT যোগ করতে:

1. MetaMask খুলুন → **"টোকেন আমদানি করুন"**
2. কন্ট্র্যাক্ট ঠিকানা: `0x1266e922fe34459efda34e7ee5caf327fbf138d7`
3. প্রতীক স্বয়ংক্রিয়ভাবে **DFT** হিসাবে প্রদর্শিত হবে
4. **"যোগ করুন"** ক্লিক করুন
