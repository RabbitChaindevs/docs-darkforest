# BSC-তে সংযোগ করুন

ডার্ক ফরেস্ট **BNB স্মার্ট চেইন (BSC)**-এ ডিপ্লয় করা হয়েছে। গেমের সাথে ইন্টারঅ্যাক্ট করতে আপনাকে আপনার ওয়ালেট BSC নেটওয়ার্কে সুইচ করতে হবে।

## BSC কী?

BNB স্মার্ট চেইন হল বিন্যান্স দ্বারা চালু একটি ব্লকচেইন নেটওয়ার্ক, যা প্রদান করে:

- **কম লেনদেন ফি** — প্রতিটি অপারেশনের খরচ এক সেন্টের ভগ্নাংশ মাত্র
- **উচ্চ থ্রুপুট** — উচ্চ-ফ্রিকোয়েন্সি গেম ইন্টারঅ্যাকশনের জন্য তৈরি
- **EVM সামঞ্জস্যপূর্ণ** — MetaMask এবং অন্যান্য মূলধারার ওয়ালেটের সাথে কাজ করে

## স্বয়ংক্রিয়ভাবে BSC নেটওয়ার্ক যোগ করুন

সবচেয়ে সহজ উপায় হল Chainlist ব্যবহার করা:

1. [Chainlist](https://chainlist.org) ভিজিট করুন
2. "BNB স্মার্ট চেইন" অনুসন্ধান করুন
3. "ওয়ালেট সংযোগ করুন" ক্লিক করুন এবং অনুমোদন করুন
4. "MetaMask-এ যোগ করুন" ক্লিক করুন
5. যোগ করার বিষয়টি নিশ্চিত করুন

## ম্যানুয়াল কনফিগারেশন

আপনি ম্যানুয়ালি BSC নেটওয়ার্ক যোগ করতে পারেন:

### MetaMask-এ BSC যোগ করা

1. MetaMask-এর উপরের-ডান কোণায় নেটওয়ার্ক ড্রপডাউন ক্লিক করুন
2. "নেটওয়ার্ক যোগ করুন" ক্লিক করুন
3. নিম্নলিখিত বিবরণ পূরণ করুন:

| সেটিং | মান |
|---------|-------|
| **নেটওয়ার্ক নাম** | BNB স্মার্ট চেইন |
| **RPC URL** | `https://bsc-dataseed.binance.org` |
| **চেইন ID** | `56` |
| **মুদ্রা প্রতীক** | `BNB` |
| **ব্লক এক্সপ্লোরার** | `https://bscscan.com` |

4. "সংরক্ষণ করুন" ক্লিক করুন

### ব্যাকআপ RPC নোড

মূল RPC ভিড় থাকলে, আপনি ব্যাকআপ নোড ব্যবহার করতে পারেন:

| প্রদানকারী | RPC URL |
|----------|---------|
| বিন্যান্স অফিসিয়াল | `https://bsc-dataseed1.binance.org` |
| Ankr | `https://rpc.ankr.com/bsc` |
| PublicNode | `https://bsc.publicnode.com` |

## BNB পাওয়া

BSC-তে প্রতিটি অপারেশনের জন্য গ্যাস ফি হিসেবে BNB প্রয়োজন। কীভাবে কিছু পাবেন:

1. **কেন্দ্রীভূত এক্সচেঞ্জ** — Binance, OKX ইত্যাদি থেকে BNB কিনুন এবং BSC নেটওয়ার্কে উত্তোলন করুন
2. **ক্রস-চেইন ব্রিজ** — ব্রিজের মাধ্যমে অন্যান্য নেটওয়ার্ক থেকে BSC-তে BNB স্থানান্তর করুন

:::caution
উত্তোলনের সময়, **নিশ্চিত করুন যে আপনি BSC নেটওয়ার্ক (BEP-20) নির্বাচন করছেন** অন্যথায় আপনার তহবিল হারিয়ে যেতে পারে!
:::

## আপনার সংযোগ যাচাই করুন

কনফিগারেশনের পর, আপনার ওয়ালেটে BNB স্মার্ট চেইন নেটওয়ার্কে সুইচ করুন। যদি চেইন ID `56` দেখায় এবং আপনার ব্যালেন্স সঠিক BNB পরিমাণ প্রদর্শন করে, আপনি প্রস্তুত।

একবার আপনার ওয়ালেট এবং নেটওয়ার্ক প্রস্তুত হলে, পরবর্তী ধাপ হল [DFT টোকেন পান](/docs/quickstart/get-dft)।
