# ওয়ালেট ইনস্টল করুন

ডার্ক ফরেস্ট খেলার আগে, আপনার ডিজিটাল সম্পদ পরিচালনা এবং অন-চেইন লেনদেনে স্বাক্ষর করার জন্য একটি Web3 ওয়ালেট প্রয়োজন।

## Web3 ওয়ালেট কী?

Web3 ওয়ালেট হল বিকেন্দ্রীভূত অ্যাপ্লিকেশন (dApps)-এর চাবিকাঠি। এটি আপনাকে নিম্নলিখিত কাজ করতে দেয়:

- **ডিজিটাল সম্পদ সংরক্ষণ ও পরিচালনা** করুন যেমন DFT এবং BNB
- **লেনদেনে স্বাক্ষর করুন** — প্রতিটি ইন-গেম কার্যকলাপ (নির্মাণ, চলাচল, যুদ্ধ) আপনার ওয়ালেটের স্বাক্ষর প্রয়োজন
- **আপনার পরিচয় প্রমাণ করুন** — ডার্ক ফরেস্টে আপনার ওয়ালেট ঠিকানাই আপনার পরিচয়

## প্রস্তাবিত ওয়ালেট

### MetaMask (প্রস্তাবিত)

MetaMask হল সবচেয়ে জনপ্রিয় Web3 ওয়ালেট, যা ব্রাউজার এক্সটেনশন এবং মোবাইল অ্যাপ হিসেবে উপলব্ধ।

#### ব্রাউজার ইনস্টলেশন

1. [MetaMask ওয়েবসাইট](https://metamask.io) ভিজিট করুন
2. ব্রাউজার এক্সটেনশন পেতে "ডাউনলোড" ক্লিক করুন
3. আপনার ব্রাউজার বেছে নিন (Chrome / Firefox / Edge / Brave)
4. "ব্রাউজারে যোগ করুন" ক্লিক করুন
5. নতুন ওয়ালেট তৈরি করতে উইজার্ড অনুসরণ করুন

:::tip
ইনস্টল করার পর, দ্রুত অ্যাক্সেসের জন্য MetaMask এক্সটেনশনটি আপনার ব্রাউজার টুলবারে পিন করুন।
:::

#### ওয়ালেট তৈরি করা

1. MetaMask আইকন ক্লিক করুন এবং "ওয়ালেট তৈরি করুন" নির্বাচন করুন
2. একটি **শক্তিশালী পাসওয়ার্ড** সেট করুন (কমপক্ষে 8 অক্ষর, অক্ষর এবং সংখ্যার মিশ্রণ)
3. **আপনার সীড ফ্রেজ নিরাপদে ব্যাকআপ করুন** — এটি আপনার ওয়ালেট পুনরুদ্ধারের একমাত্র উপায়!

:::danger সীড ফ্রেজ নিরাপত্তা
- **কখনোই** আপনার সীড ফ্রেজ কারোর সাথে শেয়ার করবেন না
- **স্ক্রিনশট নেবেন না** বা ক্লাউডে সংরক্ষণ করবেন না
- অফলাইন ব্যাকআপের জন্য ফিজিক্যাল মিডিয়া (কাগজ ও কলম) ব্যবহার করুন
- আপনার সীড ফ্রেজ চাওয়া যে কেউ স্ক্যামার
:::

4. ওয়ালেট তৈরি সম্পূর্ণ করতে আপনার সীড ফ্রেজ নিশ্চিত করুন

### Trust Wallet

Trust Wallet হল বিন্যান্সের অফিসিয়াল মোবাইল ওয়ালেট।

1. App Store বা Google Play-তে "Trust Wallet" খুঁজুন এবং ইনস্টল করুন
2. অ্যাপ খুলুন এবং "নতুন ওয়ালেট তৈরি করুন" নির্বাচন করুন
3. আপনার সীড ফ্রেজ নিরাপদে ব্যাকআপ করুন
4. ওয়ালেটের মধ্যে DApp ব্রাউজার সক্রিয় করুন

## পরবর্তী পদক্ষেপ

একবার আপনার ওয়ালেট ইনস্টল হয়ে গেলে, [BSC-তে সংযোগ করুন](/docs/quickstart/connect-bsc) এ এগিয়ে যান এবং ডার্ক ফরেস্টে প্রবেশের জন্য প্রস্তুত হন।
