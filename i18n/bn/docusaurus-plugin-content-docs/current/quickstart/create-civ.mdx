# আপনার প্রথম সভ্যতা তৈরি করুন

## পূর্বশর্ত

- MetaMask ইনস্টল এবং BSC মেইননেটে সংযুক্ত
- আপনার ওয়ালেটে **0.01 ~ 0.05 BNB** (প্রবেশ ফি, যা ন্যূনতম থেকে সর্বোচ্চ পর্যন্ত সময়ের সাথে সাথে রৈখিকভাবে বৃদ্ধি পায়)
- কমপক্ষে **1 DFT** (পরবর্তী আপগ্রেডের জন্য)

## তৈরির ধাপ

### ধাপ ১: গেম অ্যাক্সেস করুন

**[darkforest.uk](https://darkforest.uk)** খুলুন এবং উপরের-ডান কোণায় **"ওয়ালেট সংযোগ করুন"** ক্লিক করুন।

### ধাপ ২: আপনার সভ্যতার নাম দিন

সভ্যতার নাম **1–32 বাইট** হতে পারে, যা অক্ষর, সংখ্যা এবং প্রতীক সমর্থন করে।

### ধাপ ৩: আমন্ত্রণ কোড লিখুন (ঐচ্ছিক)

যদি আপনার কাছে আমন্ত্রণকারীর ওয়ালেট ঠিকানা থাকে, আপনি এটি এখানে লিখতে পারেন। আপনি এবং আমন্ত্রণকারী উভয়েই পুরস্কার হিসেবে **150 শক্তি** পাবেন।

### ধাপ ৪: তৈরি নিশ্চিত করুন

**"সভ্যতা তৈরি করুন"** ক্লিক করুন — MetaMask একটি লেনদেন নিশ্চিতকরণ পপ আপ করবে। লেনদেনে অন্তর্ভুক্ত:

- **প্রবেশ ফি**: 0.01 BNB (ন্যূনতম, 365 দিনে রৈখিকভাবে 0.05 BNB-তে বৃদ্ধি পায়)
- **গ্যাস ফি**: অল্প পরিমাণ BNB

নিশ্চিত করুন এবং লেনদেন অন-চেইন চূড়ান্ত হওয়ার অপেক্ষা করুন (প্রায় 3–5 সেকেন্ড)।

## প্রাথমিক অবস্থা

সফলভাবে তৈরি করার পর, আপনার সভ্যতা শুরু হয়:

| বৈশিষ্ট্য | প্রাথমিক মান |
|-----------|---------------|
| শক্তি | **2,000** |
| স্বাস্থ্য | **3,000** |
| স্ক্যান রেঞ্জ | **1,000** |
| সব সিস্টেম স্তর | **Lv 1** (সংগ্রাহক, অস্ত্র, ঢাল, রাডার, ইঞ্জিন) |
| ঢাল HP | **3,615** (3600 + 15 × 1²) |
| অবস্থান | নির্ধারক স্থানাঙ্ক (আপনার ঠিকানা এবং একটি নন্স থেকে প্রাপ্ত) |

## স্থানাঙ্ক নির্ধারণ

আপনার সভ্যতাকে একটি **নির্ধারক স্থানাঙ্ক** `(x, y, z)` বরাদ্দ করা হয়, যা আপনার ওয়ালেট ঠিকানা, তৈরি করার সময় এবং মোট সভ্যতার সংখ্যার হ্যাশের মাধ্যমে গণনা করা হয়। প্রতিটি সভ্যতার স্থানাঙ্ক অনন্য।

## নতুন খেলোয়াড় সুরক্ষা
