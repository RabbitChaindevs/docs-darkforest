# DFT টোকেন পান

**DFT (ডার্ক ফরেস্ট টোকেন)** হল ডার্ক ফরেস্টের মূল ERC-20 টোকেন, যার মোট সরবরাহ সীমা **~৪.২ ট্রিলিয়ন (4,206,900,000,000)**।

**DFT ব্যবহার:**
- 🚀 স্পেস জাম্প (প্রতি জাম্পে 10–1,000 DFT)
- 🔧 সিস্টেম আপগ্রেড
- 🔄 সভ্যতা পুনর্নির্মাণ
- 🏛️ জোট ত্যাগ জরিমানা
- ⚡ শক্তি বাজার লেনদেন

---

## পদ্ধতি ১: DEX-এ কিনুন

DFT PancakeSwap-এ তালিকাভুক্ত।

### ধাপ

1. [PancakeSwap](https://pancakeswap.finance/swap) খুলুন
2. আপনার ওয়ালেট সংযোগ করুন
3. "প্রাপ্তি" ফিল্ডে, DFT কন্ট্র্যাক্ট ঠিকানা লিখুন:

```
0x1266e922fe34459efda34e7ee5caf327fbf138d7
```

4. পরিমাণ লিখুন এবং **"সোয়াপ"** ক্লিক করুন

> 💡 **নোট**: PancakeSwap-এ LP লেনদেনে **২.৫% লেনদেন কর** প্রযোজ্য (১% ডেভেলপমেন্ট ফি + ১.৫% মার্কেটিং ফি)।

---

## পদ্ধতি ২: দৈনিক মিন্ট (বিনামূল্যে)

প্রতিদিন বিনামূল্যে DFT দাবি করুন:

1. [darkforest.uk](https://darkforest.uk) খুলুন
2. আপনার ওয়ালেট সংযোগ করুন
3. **"দৈনিক দাবি"** ক্লিক করুন
4. যদি সেদিন `distribute()` এখনও কল না করা হয়ে থাকে, আপনি নিজেই এটি ট্রিগার করতে পারেন
5. লেনদেন নিশ্চিত করুন (শুধুমাত্র গ্যাস ফি)

> ⚠️ **একই দিনে দাবি করুন নতুবা হারান।** DailyMinter দাবি না করা DFT জমা করে না — পরের দিনের বিতরণে অবশিষ্টাংশ পুড়িয়ে দেওয়া হয়।

## পদ্ধতি ৩: সাম্প্রদায়িক ইভেন্ট

মাঝে মাঝে এয়ারড্রপ এবং সাম্প্রদায়িক পুরস্কার ইভেন্টের জন্য অফিসিয়াল Discord / Telegram অনুসরণ করুন।

## আপনার DFT ব্যালেন্স দেখুন

MetaMask-এ DFT যোগ করতে:

1. MetaMask খুলুন → **"টোকেন আমদানি করুন"**
2. কন্ট্র্যাক্ট ঠিকানা: `0x1266e922fe34459efda34e7ee5caf327fbf138d7`
3. প্রতীক স্বয়ংক্রিয়ভাবে **DFT** হিসাবে প্রদর্শিত হবে, **18** দশমিক সহ
4. **"যোগ করুন"** ক্লিক করুন
