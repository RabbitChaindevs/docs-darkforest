# エネルギーシステム

エネルギーは Dark Forest で最も基本的なリソースです。アップグレード、攻撃、移動、修理など、ほとんどすべての操作にエネルギーが必要です。

---

## エネルギーの獲得

### 1. エネルギーコレクター（受動的出力）

```
収集速度 = 3 + 10 × √(コレクターレベル - 1)   エネルギー/秒
```

招待したプレイヤーごとに、収集速度がさらに向上します。

**コレクター耐久度**：コレクターは毎秒1の耐久度を消費します。

| コレクターレベル | 毎秒の出力 | 最大コレクター耐久度 |
|----------------|-------------------|--------------------------|
| Lv 1 | 3 | 1日 |
| Lv 10 | ~33 | 1日 + 18時間 |
| Lv 50 | ~73 | 1日 + 4日 |
| Lv 100 | ~103 | 1日 + 8.3日 |

> ⚠️ コレクターの耐久度がなくなると生産が停止します。収集を再開するには修理が必要です。

### 2. 戦闘略奪

他の文明との戦闘に勝利すると、相手のエネルギーの一部を略奪します：

```
略奪率 = 5% + 武器レベル × 0.5%
```

略奪したエネルギーは **保留中プール**（pendingCombatEnergy）に入り、手動で請求する必要があります。

### 3. 招待報酬

新規プレイヤーを招待し（相手が文明作成時にあなたのアドレスを入力）、成功するごとに：

- **あなた** は150エネルギーを受け取る
- **招待されたプレイヤー** は150エネルギーを受け取る

### 4. エネルギーマーケット

[エネルギーマーケット](/docs/economy/energy-market) を参照してください。

## エネルギーの消費

| 用途 | コスト |
|----------|------|
| **毎日の維持費** | `2,000 × 総レベル` / 日 |
| **システムアップグレード** | アンカー方式（文明ガイド参照） |
| **攻撃** | `50,000 + 50,000 × 武器レベル` |
| **宇宙ジャンプ** | `200,000 + 200,000 × √(ジャンプ回数)`、上限16.5M |
| **コレクター修理** | `1 エネルギー / 秒` |
| **武器修理** | `3 エネルギー / ポイント` |
| **シールド修理** | `4 エネルギー / ポイント` |
| **エンジン修理** | `5 エネルギー / ポイント` |
| **シールド回復** | `回復量 × 1`（自動回復） |
| **シールド修理（手動）** | `修理量 × 4`（手動修理） |
| **同盟シールド支援** | `修理量 × 1`（味方が負担） |
| **トーテム寄付** | 任意の金額 |
| **再建** | 基本500,000、再生回数に応じて増加 |

## エネルギー管理

### 毎日の維持費

維持費はアクションを実行するたびに時間に比例して差し引かれます：

```
現在の維持費 = 2,000 × 総レベル × (最終アクションからの経過時間 / 86400)
```

### 予約エネルギー

エネルギーマーケットの売り注文に設定されたエネルギーはロックされます（`energyReserved`）。`energy - reserved` がゼロになると維持費の差し引きが停止します。

### ヒント

1. **1ヶ月分以上の維持費を確保** — 日常的なアクションによる頻繁な枯渇を防止
2. **コレクターのアップグレードを優先** — 高レベルのコレクターが受動的収入の鍵
3. **攻撃前にコストを計算** — 略奪が投資に見合うか確認
4. **コレクター耐久度を定期的に修理** — 収集の中断を防止
