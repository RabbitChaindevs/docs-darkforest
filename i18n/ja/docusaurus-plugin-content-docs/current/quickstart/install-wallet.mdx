# ウォレットのインストール

Dark Forest をプレイするには、デジタル資産を管理し、オンチェーン取引に署名するための Web3 ウォレットが必要です。

## Web3 ウォレットとは

Web3 ウォレットは、分散型アプリケーション（dApps）への鍵となるツールです。以下のことが可能です：

- **DFT や BNB などのデジタル資産の保存と管理**
- **取引への署名** — ゲーム内のすべてのアクション（建築、移動、戦闘）にはウォレットの署名が必要です
- **本人確認** — ウォレットアドレスが Dark Forest におけるあなたの identity です

## 推奨ウォレット

### MetaMask（推奨）

MetaMask は最も人気のある Web3 ウォレットで、ブラウザ拡張機能とモバイルアプリとして利用できます。

#### ブラウザインストール

1. [MetaMask ウェブサイト](https://metamask.io) にアクセス
2. 「Download」をクリックしてブラウザ拡張機能を取得
3. ブラウザを選択（Chrome / Firefox / Edge / Brave）
4. 「Add to Browser」をクリック
5. ウィザードに従って新しいウォレットを作成

:::tip
インストール後、MetaMask 拡張機能をブラウザのツールバーにピン留めすると素早くアクセスできます。
:::

#### ウォレットの作成

1. MetaMask アイコンをクリックし、「Create a Wallet」を選択
2. **強力なパスワード**を設定（8文字以上、英数字混在）
3. **シードフレーズを安全にバックアップ** — これがウォレットを復元する唯一の方法です！

:::danger シードフレーズのセキュリティ
- シードフレーズを**絶対に**他人と共有しないでください
- スクリーンショットを撮ったり、クラウドに保存したりしないでください
- オフラインバックアップには物理メディア（紙とペン）を使用してください
- シードフレーズを尋ねる人物は詐欺師です
:::

4. シードフレーズを確認してウォレット作成を完了

### Trust Wallet

Trust Wallet は Binance 公式のモバイルウォレットです。

1. App Store または Google Play で「Trust Wallet」を検索してインストール
2. アプリを開き「Create a New Wallet」を選択
3. シードフレーズを安全にバックアップ
4. ウォレット内の DApp ブラウザを有効化

## 次のステップ

ウォレットのインストールが完了したら、[BSC への接続](/docs/quickstart/connect-bsc) に進み、Dark Forest に入場する準備をしましょう。
