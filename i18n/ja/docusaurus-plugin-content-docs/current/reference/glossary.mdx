# 用語集

| 用語 | 説明 |
|------|-------------|
| **文明** | プレイヤーのゲーム内 identity。独自のレベル、属性、座標を持つ |
| **エネルギー** | アップグレード、攻撃、移動、修理に使用されるゲーム内基本リソース |
| **DFT** | Dark Forest Token。ジャンプ、アップグレード、同盟脱退罰金などに使用される ERC-20 トークン |
| **座標** | 3D空間における文明の位置 `(x, y, z)`。決定論的アルゴリズムで割り当てられる |
| **総レベル** | 全システムレベルの合計。毎日の維持費に影響 |
| **スキャン範囲** | 探知半径：`1000 + 150×lv + 5×lv²`。攻撃可能な対象や追跡対象を決定 |
| **攻撃力** | `900 + 10 × level²`。武器システムの中核属性 |
| **防御力** | `540 + 6 × level²`。シールドシステムの中核属性 |
| **シールドHP** | `3600 + 15 × level²`。受けるダメージを吸収 |
| **エンジン速度** | `10 + 5 × level`。巡航速度 |
| **収集速度** | `3 + 10 × √(level - 1)` エネルギー/秒 |
| **耐久度** | 各システムの稼働状態。時間または使用により消費される |
| **毎日の維持費** | `2,000 × 総レベル / 日`。時間に比例して差し引かれる |
| **エントリー費** | 文明作成時に支払う BNB。0.01〜0.05 BNB で線形成長 |
| **アンカー方式** | 日次出力とアンカー日数に基づいてアップグレードコストを計算する方式 |
| **アンカー日数** | レベル1〜100：1〜30日。100〜1000：30〜60日。1000以上：60日（上限） |
| **巡航** | 標準的な移動方法 — 目標座標に向かって一定速度で移動 |
| **宇宙ジャンプ** | 新しい座標への瞬間テレポート。エネルギー+DFTを消費し、シールドが10%に低下 |
| **追跡ジャンプ** | ターゲットの場所への瞬間ジャンプ。レーダーLv20以上と範囲内のターゲットが必要 |
| **攻撃トークン** | 攻撃頻度を制限するトークンバケットアルゴリズム。最大3〜10トークン |
| **略奪率** | `5% + 武器レベル × 0.5%`。勝利時に略奪できるターゲットエネルギーの割合 |
| **新規プレイヤー保護** | 作成後24時間は攻撃不可 |
| **遺跡** | 敗北後にこれ以上ダウングレードできない文明。再建で復活可能 |
| **再建 / 再生** | 遺跡状態の文明を復活。エネルギー+DFTを消費し、戦闘ブーストを付与 |
| **戦闘ブースト** | 再建ごとに+12%、最大300%（30000 bps） |
| **同盟** | プレイヤーの協力組織。最大100名 |
| **脱退クールダウン** | 同盟脱退後の24時間の相互不可侵期間 |
| **トーテム** | 同盟全体のバフシステム。エネルギーを寄付してアップグレードし防御ボーナスを得る |
| **エポック** | DailyMinter の時間単位。UTC 0:00 から次の UTC 0:00 まで |
| **DELEGATECALL** | メインコントラクトがロジックをサテライトコントラクトに委任するアーキテクチャパターン |
| **ソフトゲート** | 攻撃条件を満たさない場合、リソースを消費せずに理由を表示 |
