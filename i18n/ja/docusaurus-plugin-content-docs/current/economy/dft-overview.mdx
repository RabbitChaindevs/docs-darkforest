# DFT トークン概要

**DFT（Dark Forest Token）** は Dark Forest エコシステムのコアトークンであり、標準 ERC-20 トークンとして実装されています。

---

## 基本情報

| プロパティ | 値 |
|----------|-------|
| コントラクトアドレス | `0x1266e922fe34459efda34e7ee5caf327fbf138d7` |
| トークン名 | DarkForest |
| シンボル | DFT |
| 小数点 | 18 |
| 総供給上限 | **4,206,900,000,000 DFT**（約4.2兆） |
| 配布期間 | 約10年（3,650日） |

---

## DFT の用途

| 用途 | 説明 |
|----------|-------------|
| 🚀 **宇宙ジャンプ** | `10 + 10 × √(ジャンプ回数)` DFT、上限1,000 |
| 🔧 **システムアップグレード** | アンカー方式に基づくアップグレードコスト |
| 🔄 **文明の再建** | `100 + 10 × 再生回数` DFT、上限2,000 |
| 🏛️ **同盟脱退罰金** | `100 + 10 × メンバー数` DFT |
| ⚡ **エネルギーマーケット** | オーダーブックでDFTを使用してエネルギーを購入 |

---

## トークノミクス

### 供給量

- **ハードキャップ**：4,206,900,000,000 DFT（約4.2兆）
- **毎日のミント**：`4,206,900,000,000 / 3,650 ≈ 1,152,575,342 DFT`（一定の日次排出量）
- **毎日の配布**：DailyMinter を通じてアクティブプレイヤーに配布

### 取引税

DFT が DEX（PancakeSwap）で取引される場合：

| 税 | 税率 | 目的 |
|-----|------|---------|
| 開発費 | 1% | チーム |
| マーケティング費 | 1.5% | マーケティング |
| **取引税合計** | **2.5%** | |

税金は **登録された流動性プール（LP）** での取引にのみ適用されます。通常の転送は非課税です。

## ウォレットで DFT を表示する

MetaMask に DFT を追加するには：

1. MetaMask を開く → **「Import Tokens」**
2. コントラクトアドレス：`0x1266e922fe34459efda34e7ee5caf327fbf138d7`
3. シンボルが自動的に **DFT** と表示
4. **「Add」** をクリック

**推奨される次のステップ：[エネルギーマーケット →](/docs/economy/energy-market)**
