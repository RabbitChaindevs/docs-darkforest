# デイリーミント

**DailyMinter** コントラクトは毎日一定量の DFT をミントし、アクティブな文明に均等に分配します。**その日のうちに請求しないと失われます** — 未請求の金額は累積されません。

---

## 請求方法

### 必要条件

- UTC のその日の 0:00 **より前に** 文明を作成していること
- その日の `distribute()` が呼び出されていること（誰でもトリガー可能）
- 各アドレスは1日に1回のみ請求可能

### 操作

```
claim()
```

トランザクションを確認（ガス代のみ）。

---

## ミンティングフロー

### ステップ1：配布

誰でも `distribute()` を呼び出せます：

1. コントラクトが最後に未配布のエポックをチェック
2. 現在のアクティブな文明数をスナップショット
3. `一人当たりのシェア = 日次排出量 / アクティブな文明数` を計算
4. 総 DFT をコントラクトにミント
5. `日次排出量 = 1,152,575,342 DFT`（一定）

### ステップ2：請求

プレイヤーが `claim()` を呼び出します：

1. このエポックより前に文明を作成したことを確認
2. まだ請求していないことを確認
3. あなたのシェアをウォレットに転送

### ステップ3：残りのバーン

次の `distribute()` 時に、前のエポックで請求されなかった DFT は **バーン** されます（循環供給量が減少）。

---

## タイムライン

```
エポック 0     エポック 1     エポック 2
   |            |            |
UTC 00:00    UTC 00:00    UTC 00:00
   ↑文明作成    ↑配布         ↑配布
               ↑請求         ↑請求
                             ↑未請求分バーン
```

- エポックは UTC の0:00 で区切られます
- **最後に配布された** エポックのみ請求可能
- エポックを逃すと永遠に請求できません

---

## 主要パラメータ

| パラメータ | 値 |
|-----------|-------|
| 日次排出量 | **1,152,575,342 DFT**（一定） |
| 総配布期間 | **3,650日**（約10年） |
| 総供給上限 | **4,206,900,000,000 DFT**（約4.2兆） |
| 請求可能期間 | 各エポック内（通常1日） |

---

## 毎日請求する理由

1. 🆓 **無料の DFT** — ガス代のみで、毎日利用可能
2. 📉 **未請求分はバーン** — 逃すと失われます
3. 📈 **プレイヤーが少ないほど一人当たり増加** — アクティブプレイヤーが減ると、各人のシェアが増加

> 💡 UTC 時間に毎日請求するリマインダーを設定しましょう。その日の `distribute()` がまだ呼び出されていない場合、自分でトリガーできます — ガス代のみかかります。
