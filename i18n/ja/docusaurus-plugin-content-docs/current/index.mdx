# Dark Forest ドキュメント

Dark Forest は、**BNB Smart Chain** にデプロイされた完全オンチェーン3D星际文明ストラテジーゲーム（SLG）です。すべてのゲームロジックはスマートコントラクト上で動作し、DELEGATECALL アーキテクチャによるモジュール式アップグレードを採用しています。
