AI 에이전트란?

AI 에이전트는 사용자의 문명을 대신하여 게임 작업을 수행하는 오프체인 자동화 프로그램입니다. 권한 비트마스크를 통해 승인되며 정책 제약 조건 내에서 작동합니다.

에이전트가 할 수 있는 일​

에이전트의 기능은 승인된 권한에 의해 제어됩니다:

권한값액션
COLLECT1에너지 수집
REPAIR2시스템 수리
CLAIM_COMBAT8전투 에너지 청구
ATTACK32공격
UPGRADE64시스템 업그레이드
JUMP128스페이스 점프
REBUILD256문명 재건
DONATE512토템 기부

원하는 권한 조합을 에이전트에 부여할 수 있습니다 (비트마스크).

정책 제약 조건​

권한 외에도 에이전트에 정책 제약 조건을 설정할 수 있습니다:

제약 조건설명
minEnergyReserve최소 에너지 보유량 (이 이하로는 사용 불가)
minDftReserve최소 DFT 보유량
maxAttackPerHour시간당 최대 공격 횟수
maxUpgradePerDay일일 최대 업그레이드 횟수
maxSingleUpgradeCost업그레이드당 최대 DFT 비용
autoCollect자동 에너지 수집
autoRepair자동 시스템 수리
autoShieldRepair자동 방어막 수리
autoUpgrade자동 업그레이드
autoAttack자동 공격

보안 메커니즘​

1. 온체인 등록​

에이전트의 정책은 DarkForestAgentRegistry.sol 컨트랙트에 저장되며, 한 번 설정되면 변조할 수 없습니다.

2. 권한 분리​

  • 승인: 에이전트가 사용자를 대신해 행동하도록 승인
  • 권한 비트마스크: 각 작업에 대한 세분화된 제어
  • 철회 가능: revokeAgent()가 즉시 적용됨

3. 서킷 브레이커​

공격 빈도 제한: 시간당 최대 N회
업그레이드 빈도 제한: 일일 최대 N회
에너지 최소치: X 미만으로 떨어지지 않음
DFT 최소치: Y 미만으로 떨어지지 않음

4. 블랙리스트 / 화이트리스트​

  • 블랙리스트: 특정 플레이어 공격 금지
  • 화이트리스트: 특정 플레이어만 공격 허용 (화이트리스트 활성 시 블랙리스트 무시)
