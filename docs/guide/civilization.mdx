# 文明建设与升级

你的文明由 **六大系统** 组成，每个系统可独立升级。系统等级影响你的采集效率、战斗力和生存能力。

---

## 六大系统

| 系统 ID | 名称 | 作用 | 核心公式 |
|---------|------|------|----------|
| 0 | **能量采集** | 每秒自动产出能源 | `3 + 10×√(等级-1)` 能源/秒 |
| 1 | **武器系统** | 攻击力 | `900 + 10×等级²` |
| 2 | **护盾系统** | 防御与护盾 HP | HP: `3600 + 15×等级²`，防御: `540 + 6×等级²` |
| 3 | **雷达系统** | 扫描范围 | `1000 + 150×等级 + 5×等级²` |
| 4 | **引擎系统** | 移动速度 | `10 + 5×等级` |

---

## 升级消耗

升级消耗基于 **锚定公式（Anchor Formula）**，根据系统类型和当前等级计算：

### 等级区间

| 等级区间 | 锚定天数（折算成本） |
|----------|-------------------|
| 1 - 100 | 1 天 → 30 天的能源/DFT 产出 |
| 100 - 1000 | 30 天 → 60 天的产出 |
| 1000+ | 60 天的产出（封顶） |

### 各类系统的升级乘数

| 系统 | 能源/DFT 乘数 |
|------|-------------|
| 采集系统 (0) | 1x |
| 武器系统 (1) | 4x |
| 护盾系统 (2) | 4x |
| 雷达系统 (3) | 8x |
| 引擎系统 (4) | 16x |

### 升级消耗公式

```
升级消耗 = 锚定天数 × 每日产出 × 系统乘数
```

其中「每日产出」又和你的文明等级相关，等级越高每日产出越高，形成**正反馈循环**。

---

## 每日维护费（Upkeep）

文明每天需要消耗能源作为维护费用：

```
每日维护费 = 2,000 × 总等级
```

维护费按时间比例从你的能源中扣除，在你执行任何操作时自动结算（`_deductUpkeep`）。

> ⚠️ 能源如果不足以支付维护费，你的系统仍会正常运行，但能源会变为负数，需要及时补充。

---

## 耐久度系统

每个系统都有耐久度，随时间或使用消耗：

| 系统 | 基础耐久 | 每级增加 |
|------|---------|---------|
| 采集器 | 86,400 秒（1天） | +7,200 秒（2小时） |
| 武器 | 60 | +15 |
| 护盾 | 40 | +10 |
| 引擎 | 30 | +6 |

### 耐久度消耗

- **采集器**：随时间衰减，每过 1 秒消耗 1 耐久
- **武器**：每次攻击消耗 1 耐久（耐久为 0 时攻击力减半）
- **护盾**：每次被攻击消耗 1 耐久
- **引擎**：每次跳跃消耗 1 耐久

### 修复消耗

| 系统 | 修复成本 |
|------|---------|
| 采集器 | 1 能源/秒 |
| 武器 | 3 能源/点 |
| 护盾 | 4 能源/点 |
| 引擎 | 5 能源/点 |

---

## 升级策略建议

1. **优先升采集器**：能源是一切的基础，高采集等级让你更快积累资源
2. **武器和护盾均衡发展**：武器 Lv 影响攻击力 + 掠夺比例，护盾 Lv 影响 HP 和防御
3. **雷达决定攻击范围**：你只能攻击雷达范围内的目标
4. **引擎影响移动速度**：探索和追击都依赖引擎
