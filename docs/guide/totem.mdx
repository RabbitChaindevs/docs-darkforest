# 图腾系统

图腾是联盟级的增益系统，通过联盟成员捐赠能源来升级，为**全体成员**提供防御加成（无论个人是否捐过）。

---

## 如何运作

1. **捐赠能源**：联盟成员向图腾池捐赠能源
2. **积累到阈值**：图腾能量达到升级成本后，盟主触发升级
3. **获得加成**：全体成员获得防御加成

---

## 捐赠能源

### 操作

```
donateToTotem(allianceId, amount)
```

- 每次捐赠会重置你的捐赠计时器（48 小时超时）
- 只有捐赠有效的成员才能享受图腾加成

### 捐赠条件

- 每次捐赠需间隔一定时间（防刷）
- 捐赠 48 小时后标记为失效（需要重新捐赠保持活跃）

---

## 升级图腾

### 成本公式

```
下一次升级成本 = 20,000,000 × N × (N + 5) / 10
其中 N = 当前图腾等级 + 1
```

| 等级 | 升级所需能量 |
|------|------------|
| Lv 0 → 1 | 12,000,000 |
| Lv 1 → 2 | 42,000,000 |
| Lv 2 → 3 | 96,000,000 |
| Lv 10 → 11 | 330,000,000 |

### 操作

只有**盟主**可以触发升级：

```
upgradeTotem(allianceId)
```

升级消耗图腾池中的能量，图腾等级 +1。

---

## 图腾效果

```
联盟防御加成 = (盟友在线数 - 1) × 8 × (1 + 图腾等级 × 0.5%)
```

| 图腾等级 | 加成系数 |
|---------|---------|
| Lv 0 | +0%（不提升） |
| Lv 10 | +5% |
| Lv 20 | +10% |
| Lv 100 | +50% |

例如：一个 50 人的联盟，图腾等级 20：

```
联盟防御加成 = (50-1) × 8 × (1 + 20 × 0.5%) = 49 × 8 × 1.1 = 431.2 防御
```

此加成对联盟**全体成员**生效，无论个人是否捐赠过图腾。
加成 = (50-1) × 8 × (1 + 20 × 0.5%) = 392 × 1.1 = 431.2 防御
```

## 策略建议

1. **联盟成员定期捐赠**：少量多次比一次性大量捐赠更有效
2. **盟主及时升级**：能量积攒到阈值立即升级
3. **高图腾等级 + 多盟友 = 显著防御优势**
