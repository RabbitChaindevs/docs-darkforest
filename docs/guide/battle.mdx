# 攻击与防御

## 攻击条件

### 你可以攻击谁？

- 目标必须存在且未被摧毁（非废墟状态）
- 目标不能在 **24 小时新手保护期** 内
- 目标必须在你的**雷达扫描范围**内（距离 ≤ 你的雷达范围）
- 不能攻击自己
- 不能攻击同一联盟的成员（需先退出联盟）
- 不能攻击最近**24 小时内**从你联盟退出的成员（相互冷却）

### 软限制（Soft Gate）

上述条件不满足时，攻击**不会消耗任何资源**，只会提示原因。这让你可以自由尝试攻击而不必担心浪费能源。

---

## 攻击消耗

每次成功攻击消耗：

| 项目 | 消耗 |
|------|------|
| **能源** | `50,000 + 50,000 × 武器等级` |
| **武器耐久** | 1 点（武器耐久 > 0 时） |
| **攻击代币** | 1 个 |

---

## 攻击代币系统

攻击代币使用 **令牌桶算法** 限制攻击频率：

| 参数 | 公式 | 说明 |
|------|------|------|
| **最大代币数** | `3 + 武器等级/10`，上限 **10** | 随时间自动恢复 |
| **恢复间隔** | `300 - 武器等级 × 10` 毫秒，最小 **100ms** | 每间隔恢复 1 个 |
| **消耗** | 每次攻击消耗 1 个 | |

> 💡 武器等级越高，攻击代币恢复越快，上限也越高。

## 伤害计算

### 攻击方

```
基础攻击力 = 900 + 10 × 攻击方武器等级²
如果武器耐久 = 0，攻击力减半
最终攻击力 = 基础攻击力 × (1 + 战斗加成/10000)
```

### 防御方

```
基础防御力 = 540 + 6 × 防御方护盾等级²
联盟防御加成 = (盟友数-1) × 8 × (1 + 图腾等级 × 0.5%)
最终防御力 = 基础防御力 × (1 + 战斗加成/10000) + 联盟加成
```

### 最终伤害

```
护盾伤害 = 攻击力 + 200 - 防御力（如果 > 0）
生命伤害 = (攻击力 - 防御力) × 2（如果 > 0）

如果护盾伤害 > 护盾剩余 HP：
  溢出部分 × 3 转为生命伤害
```

## 掠夺（Plunder）

获胜后，你可以从目标掠夺能源：

```
掠夺比例 = 5% + 武器等级 × 0.5%
掠夺能源 = 目标当前能源 × 掠夺比例
```

掠夺的能源进入你的 **待领取战斗能源** 池，需要手动领取。

## 击败与摧毁

### 击败

当目标生命值归零时：

1. **掠夺全部剩余能源**到你的待领取池
2. 目标被**降级**一个随机系统（根据链上随机数选择采集/武器/护盾/引擎之一）
3. 如降级后系统等级仍 ≥ 1，文明存活，耐久恢复至 50%
4. 如所有系统等级都是 1（无法再降级），文明变成**废墟**

### 废墟

变为废墟的文明：
- 从活跃文明列表中移除
- 不能被攻击
- 可以通过**重建**功能复活

### 战斗加成（Rebirth Boost）

每次重生（Rebuild）获得 **+1200 bps**（12%）的战斗加成，最高叠加至 **30000 bps**（3 倍）。
攻击和防御同时受益。

## 护盾回复

### 自动回复

```
回复量 = 50 + 护盾等级²
消耗能源 = 回复量（1:1 比例）
```

### 手动修复

```
消耗能源 = 修复量 × 4
```

### 联盟协助

```
消耗能源 = 修复量 × 1（盟友支付）
可以为同一联盟的盟友修复护盾
```

---

## 防御策略

1. **保持能源储备 > 每日维护费 × 7**：避免能源耗尽后护盾无法回复
2. **定时修复耐久**：武器/护盾/引擎耐久归零后非常脆弱
3. **升级护盾等级**：护盾 HP 和防御力都随等级平方增长
4. **加入联盟并获得图腾等级**：联盟防御加成显著
