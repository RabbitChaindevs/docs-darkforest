# Dark Forest 文档

Dark Forest 是一款部署在 **BNB Smart Chain** 上的完全链上 3D 星际文明策略游戏（SLG）。所有游戏逻辑由智能合约执行，通过 DELEGATECALL 架构实现模块化升级。
