# 常见问题

## 入门

### 创建文明需要多少钱？

入场费从 **0.01 BNB** 开始，随游戏上线天数线性增长至 **0.05 BNB**（365 天后达到最大值）。此外还需要少量的 BNB 作为 Gas 费。

### DFT 有什么用？

DFT 用于空间跳跃、系统升级、重建文明、退出联盟罚款以及在能源市场购买能源。

### 新手保护期多久？

**24 小时**。期间其他玩家不能攻击你。

## 游戏玩法

### 为什么我不能攻击某个文明？

以下是可能的软限制原因（不消耗任何资源）：

- 目标在新手保护期内（< 24 小时）
- 你和目标在同一联盟
- 你和目标之间有退出冷却（24 小时内）
- 目标不在你的雷达范围内
- 你就是目标自己

### 能源不够用了怎么办？

- 升级**能量采集系统**提高被动产出
- 攻击其他文明**掠夺**能源
- 在**能源市场**用 DFT 购买
- **邀请新玩家**获得奖励
- 减少不必要的操作（如非必要的护盾回复）

### 空间跳跃后护盾为什么变成零？

空间跳跃会对舰体造成损伤，**护盾 HP 重置为最大值的 10%**。跳跃后请小心：

1. 跳跃后立即修复护盾
2. 不要在战斗区域附近跳跃
3. 联盟成员可以在跳跃后协助修复

### 耐久度归零会怎样？

| 系统 | 耐久归零后果 |
|------|-------------|
| 采集器 | 停止产出能源 |
| 武器 | 攻击力减半 |
| 护盾 | 护盾失效（不提供防御） |
| 引擎 | 无法巡航和跳跃 |

## 经济

### DFT 的总供应量是多少？

**约 4.2 万亿（42,069 亿）DFT**，这是硬顶。每日铸造约 1.15 万亿 DFT，按活跃玩家数量分配。

### DFT 交易有手续费吗？

只有在 DEX（如 PancakeSwap）上交易时才收 **2.5% 的总手续费**（1% 开发 + 1.5% 营销）。普通转账不收手续费。

### 每日铸造的 DFT 可以累积吗？

**不能。** 每个 epoch 的 DFT 必须在该 epoch 内领取。当天不领，次日的 `distribute()` 会销毁剩余 DFT。

## 联盟

### 退出联盟有惩罚吗？

有，需要支付 **100 DFT + 10 DFT × 当前成员数**。这笔 DFT 会分配给联盟内的其他成员。

### 退出联盟后多久可以再次攻击？

**24 小时冷却期**。在此期间你和原联盟之间互不攻击。

### 联盟最多可以有多少人？

**100 人。**

## 重建（Rebirth）

### 重建需要什么条件？

你的文明必须是**废墟状态**（被击败且所有系统已降至 Lv1）。

### 重建后新手保护会重置吗？

会。重建后 `creationTime` 重置为当前时间，因此**新手保护的 24 小时也会重置**。

### 生命值上限是多少？

**20,000**。初始生命值 3,000，战斗中生命值不会自动恢复，只能通过重建重置为 10,000。

### 重建消耗多少？

- **能源**：`500,000 × (10000 + 重生次数 × 1200) / 10000`，上限 10,000,000（20倍）
- **DFT**：`100 + 10 × 重生次数`，上限 2,000 DFT

### 重建有什么好处？

每次重建获得 **+12%（1200 bps）** 的战斗加成，攻击和防御同时受益。最高叠加至 **300%（30000 bps）**。

## AI Agent

### Agent 安全吗？

Agent 系统有多层安全机制：

1. **权限位掩码**：精确控制每个操作
2. **链上策略**：熔断参数不可绕过
3. **可随时撤销**：`revokeAgent()` 立即失效
4. **加密存储**：私钥本地加密

建议使用**独立游戏钱包**运行 Agent。

### Agent 能自动攻击吗？

可以，前提是你授予了 `ATTACK` 权限并设置了 `autoAttack: true`。还可以设置黑名单/白名单控制攻击目标。
