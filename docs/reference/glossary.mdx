# 术语表

| 术语 | 说明 |
|------|------|
| **文明（Civilization）** | 玩家的游戏身份，拥有独立的等级、属性和坐标 |
| **能源（Energy）** | 游戏基础资源，用于升级、攻击、移动、修复等所有操作 |
| **DFT** | Dark Forest Token，ERC-20 代币，用于跳跃、升级、退出联盟罚款等 |
| **坐标（Coordinates）** | 文明在 3D 空间中的位置 `(x, y, z)`，确定性算法分配 |
| **总等级** | 文明所有系统等级之和，影响每日维护费 |
| **雷达范围（Scan Range）** | 你的探测半径 `1000 + 150×lv + 5×lv²`，决定了你可以攻击/追踪谁 |
| **攻击力** | `900 + 10×等级²`，武器系统的核心属性 |
| **防御力** | `540 + 6×等级²`，护盾系统的核心属性 |
| **护盾 HP** | `3600 + 15×等级²`，抵消攻击伤害 |
| **引擎速度** | `10 + 5×等级`，巡航速度 |
| **采集速率** | `3 + 10×√(等级-1)` 能源/秒 |
| **耐久度（Durability）** | 各系统的工作状态，随时间或使用消耗 |
| **每日维护费（Upkeep）** | `2,000 × 总等级 / 天`，随时间比例自动扣除 |
| **入场费（Entry Fee）** | 创建文明时支付的 BNB，0.01~0.05 BNB 线性增长 |
| **锚定公式（Anchor Formula）** | 升级消耗的计算公式，基于每日产出和锚定天数 |
| **锚定天数** | 等级 1-100: 1-30 天，100-1000: 30-60 天，1000+: 60 天 |
| **巡航（Cruise）** | 常规移动方式，向目标坐标匀速移动 |
| **空间跳跃（Space Jump）** | 瞬间移动到新坐标，消耗能源+DFT，护盾降至 10% |
| **追踪跳跃（Tracking Jump）** | 瞬间跳到目标位置，需雷达 Lv20+且目标在范围内 |
| **攻击代币（Attack Token）** | 令牌桶算法限制攻击频率，最大 3-10 个 |
| **掠夺比例（Plunder Ratio）** | `5% + 武器等级 × 0.5%`，胜利后掠夺目标能源 |
| **新手保护** | 创建后 24 小时内不可被攻击 |
| **废墟（Ruins）** | 文明被击败后无法降级时变成废墟，可通过重建复活 |
| **重建（Rebuild/Rebirth）** | 废墟文明复活，消耗能源+DFT，获得战斗加成 |
| **战斗加成（Combat Boost）** | 每次重建 +12%，最高 300%（30000 bps） |
| **联盟（Alliance）** | 玩家组成的协作组织，最多 100 人 |
| **退出冷却（Leave Cooldown）** | 退出联盟后 24 小时内互不攻击 |
| **图腾（Totem）** | 联盟级增益系统，捐赠能源升级，提供防御加成 |
| **Epoch** | DailyMinter 的时间单位，从 UTC 午夜到次日午夜 |
| **DELEGATECALL** | 合约架构模式，主合约将逻辑调用委托给卫星合约 |
| **软限制（Soft Gate）** | 攻击检查不通过时仅提示原因，不消耗资源 |
