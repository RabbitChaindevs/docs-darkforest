## 合约架构

```
DarkForest（DELEGATECALL 入口）
├── DELEGATECALL → DarkForestGameplay   （创建文明、升级系统）
├── DELEGATECALL → DarkForestBattle      （攻击、修复护盾、护盾回复）
├── DELEGATECALL → DarkForestMovement    （空间跳跃、追踪跳跃、巡航）
└── DELEGATECALL → DarkForestAdmin       （采集、修复、重建、图腾、提现）

DarkForestStorage（共享存储布局）
→ 所有卫星合约继承 DarkForestStorage 以保证存储布局一致
```
