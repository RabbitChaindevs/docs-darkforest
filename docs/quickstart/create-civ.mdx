# 创建你的第一个文明

## 前置条件

- MetaMask 钱包已安装并连接 BSC 主网
- 钱包里有 **0.01 ~ 0.05 BNB**（作为入场费，费用随游戏启动天数从最低到最高线性增长）
- 至少 **1 DFT**（用于后续升级）

## 创建步骤

### 第 1 步：访问游戏

打开 **[darkforest.uk](https://darkforest.uk)**，点击右上角 **「连接钱包」**。

### 第 2 步：命名你的文明

文明名称为 **1-32 字节**，支持中英文、数字和符号。

### 第 3 步：输入邀请码（可选）

如果你有邀请人的钱包地址，可以输入。邀请人和你各获得 **150 能源** 奖励。

### 第 4 步：确认创建

点击 **「创建文明」**，MetaMask 弹出交易确认。交易包含：

- **入场费**：0.01 BNB（最小值，随时间线性增长至 0.05 BNB，365 天达到最大值）
- **Gas 费**：少量 BNB

确认后等待上链（约 3-5 秒）。

## 初始状态

文明创建成功后，你将获得：

| 属性 | 初始值 |
|------|--------|
| 能源 | **2,000** |
| 生命值 | **3,000** |
| 雷达范围 | **1,000** |
| 所有系统等级 | **Lv 1**（采集、武器、护盾、雷达、引擎） |
| 护盾 HP | **3,615**（3600 + 15×1²） |
| 位置 | 确定性坐标（由你的地址和随机数共同决定） |

## 坐标分配

你的文明会被分配到一个**确定性坐标** `(x, y, z)`，由你的钱包地址、创建时间和全局文明总数通过哈希计算得出。每个文明的坐标是唯一的。

## 新手保护

> 🛡️ 新创建的文明有 **24 小时** 新手保护期。期间其他玩家无法攻击你。
