# 获取 DFT 代币

**DFT（Dark Forest Token）** 是 Dark Forest 的 ERC-20 核心代币，总供应上限 **420.69 万亿枚**。

**DFT 用途：**
- 🚀 空间跳跃（每次 10-1,000 DFT）
- 🔧 系统升级
- 🔄 重建文明
- 🏛️ 退出联盟罚款
- ⚡ 能源市场交易

---

## 方式一：DEX 购买

DFT 已上线 PancakeSwap。

### 操作步骤

1. 打开 [PancakeSwap](https://pancakeswap.finance/swap)
2. 连接你的钱包
3. 在「兑换到」栏输入 DFT 合约地址：

```
0x1266e922fe34459efda34e7ee5caf327fbf138d7
```

4. 输入数量，点击 **「Swap」**

> 💡 **注意**：PancakeSwap 等 LP 交易会收取 **2.5% 交易税**（1% 开发费 + 1.5% 营销费）。

---

## 方式二：每日铸造（免费）

每天可以免费领取一定数量的 DFT：

1. 打开 [darkforest.uk](https://darkforest.uk)
2. 连接钱包
3. 点击 **「每日领取」**
4. 如果当日尚未分发（`distribute()` 未被调用），你也可以自己调用
5. 确认交易（只需 Gas 费）

> ⚠️ **当天不领就没了**。DailyMinter 不累积未领取的 DFT，次日的分发会销毁剩余部分。

---

## 方式三：社区活动

关注官方 Discord / Telegram，有时会有空投和社区奖励活动。

## 查看 DFT 余额

在 MetaMask 中添加 DFT 代币：

1. 打开 MetaMask → **「导入代币」**
2. 合约地址：`0x1266e922fe34459efda34e7ee5caf327fbf138d7`
3. 符号自动显示为 **DFT**，精度 **18**
4. 点击 **「添加」**
