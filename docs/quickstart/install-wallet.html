安装钱包

在开始游玩 Dark Forest 之前，你需要一个 Web3 钱包来管理你的数字资产和进行链上操作。

什么是 Web3 钱包？​

Web3 钱包是通往去中心化应用（dApp）的钥匙。它用于：

  • 存储和管理 DFT、BNB 等加密资产
  • 签名交易 在游戏中的每一步操作（建造、移动、战斗）都需要钱包签名确认
  • 身份认证 作为你在 Dark Forest 中的身份标识

推荐钱包​

MetaMask（推荐）​

MetaMask 是最流行的 Web3 钱包，支持桌面浏览器和移动设备。

浏览器安装​

  1. 访问 MetaMask 官网
  2. 点击 "Download" 下载浏览器扩展
  3. 选择你的浏览器（Chrome / Firefox / Edge / Brave）
  4. 点击 "添加到浏览器"
  5. 按照向导创建新钱包
提示

安装完成后，请将 MetaMask 扩展固定在浏览器工具栏，方便快速访问。

创建钱包​

  1. 点击 MetaMask 图标，选择 "创建钱包"
  2. 设置一个强密码（至少 8 位，包含字母和数字）
  3. 安全备份助记词 — 这是恢复钱包的唯一方式！
助记词安全
  • 永远不要将助记词透露给任何人
  • 不要截图或存储在云端
  • 建议使用物理介质（如纸笔）离线备份
  • 任何索要助记词的人都是骗子
  1. 确认助记词后，钱包即创建完成

Trust Wallet​

Trust Wallet 是 Binance 官方支持的移动端钱包。

  1. 在 App Store 或 Google Play 搜索 "Trust Wallet" 并安装
  2. 打开应用，选择 "创建新钱包"
  3. 安全备份助记词
  4. 创建完成后，在钱包内启用 DApp 浏览器

后续步骤​

安装好钱包后，继续学习 连接 BSC 网络，为进入 Dark Forest 做好准备。
