# DFT 代币概览

**DFT（Dark Forest Token）** 是 Dark Forest 生态系统的核心代币，标准 ERC-20 代币。

---

## 基本信息

| 属性 | 值 |
|------|-----|
| 合约地址 | `0x1266e922fe34459efda34e7ee5caf327fbf138d7` |
| 代币名称 | DarkForest |
| 代币符号 | DFT |
| 精度 | 18 位小数 |
| 总供应上限 | **420,690,000,000,000 DFT**（420.69 万亿） |
| 发行周期 | ~10 年（3,650 天） |

---

## DFT 用途

| 用途 | 说明 |
|------|------|
| 🚀 **空间跳跃** | `10 + 10×√(跳跃次数)` DFT，上限 1,000 |
| 🔧 **系统升级** | 锚定公式计算的升级成本 |
| 🔄 **重建文明** | `100 + 10×重生次数` DFT，上限 2,000 |
| 🏛️ **退出联盟罚款** | `100 + 10×成员数` DFT |
| ⚡ **能源市场** | 在订单簿中用 DFT 购买能源 |

---

## 代币经济

### 供应量

- **硬顶**：420.69 万亿 DFT
- **每日铸造**：`总供应量 / 3,650`
- **每日分发**：通过 DailyMinter 分发给活跃玩家

### 交易税

在 DEX（PancakeSwap）上交易 DFT 时：

| 税项 | 比例 | 用途 |
|------|------|------|
| 开发费 | 1% | 团队 |
| 营销费 | 1.5% | 市场运营 |
| **总交易税** | **2.5%** | |

只有在**注册的流动性池**（LP）中交易才收税，普通转账不收税。

## 查看方式

在 MetaMask 中添加 DFT 代币：

1. 打开 MetaMask → **「导入代币」**
2. 合约地址：`0x1266e922fe34459efda34e7ee5caf327fbf138d7`
3. 符号自动显示为 **DFT**
4. 点击 **「添加」**
