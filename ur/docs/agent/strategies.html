حکمت عملی ٹیمپلیٹس

ایجنٹ کی حکمت عملیاں YAML کنفیگریشن فائلوں کا استعمال کرتے ہوئے بیان کی جاتی ہیں۔

حکمت عملی فائل کا ڈھانچہ​

agent:
  name: "my-agent"
  mode: "auto"          # auto | manual | dry-run

limits:                 # سرکٹ بریکر (آن چین پالیسی)
  min_energy_reserve: 100000
  min_dft_reserve: 10
  max_attacks_per_hour: 5
  max_upgrades_per_day: 3
  max_single_upgrade_cost: 100

combat:                 # جنگی حکمت عملی
  min_target_level: 1
  max_target_level: 999
  weapon_level: 1
  only_attack_when_energy_above: 100000
  blacklist: []         # بلیک لسٹ شدہ ایڈریسز
  whitelist: []         # وائٹ لسٹ شدہ ایڈریسز (صرف ان پر حملہ کریں)

defense:                # دفاعی حکمت عملی
  auto_repair: true
  min_durability_before_repair: 20
  shield_regen_threshold: 50   # اس % سے نیچے ڈھال کی بحالی شروع کریں

collection:             # جمع کرنے کی حکمت عملی
  enabled: true
  interval_minutes: 60

چار پری سیٹ حکمت عملیاں​

1. جارحانہ (Aggressive)​

combat:
  max_attacks_per_hour: 10
  min_target_level: 0.5
  weapon_level: max

collection:
  enabled: false  # کوئی جمع نہیں، خالص جنگ
  • ✅ موزوں برائے: انتہائی فعال کھلاڑی جو جنگی انعامات چاہتے ہیں
  • ❌ خطرہ: زیادہ توانائی کی کھپت، جوابی حملے کا خطرہ

2. متوازن (Balanced)​

combat:
  max_attacks_per_hour: 5
  min_target_level: 0.8
  weapon_level: balanced

collection:
  enabled: true
  interval_minutes: 30

defense:
  auto_repair: true
  shield_regen_threshold: 60
  • ✅ تجویز کردہ: متوازن حملہ اور دفاع، زیادہ تر کھلاڑیوں کے لیے موزوں

3. دفاعی (Defensive)​

combat:
  max_attacks_per_hour: 1
  only_counterattack: true  # صرف جوابی حملہ

defense:
  auto_repair: true
  min_durability_before_repair: 50
  shield_regen_threshold: 80

collection:
  enabled: true
  interval_minutes: 15
  • ✅ موزوں برائے: وہ کھلاڑی جو طویل عرصے تک آف لائن رہتے ہیں اور خطرے سے بچنا چاہتے ہیں

4. کسان (Farmer)​

combat:
  enabled: false  # کوئی جنگ نہیں

defense:
  auto_repair: true
  shield_regen_threshold: 90

collection:
  enabled: true
  interval_minutes: 5
  • ✅ موزوں برائے: ابتدائی مرحلے کی ترقی اور وسائل جمع کرنا

کسٹم حکمت عملی​

# ٹیمپلیٹ کاپی کریں
cp templates/balanced.yaml my-strategy.yaml

# پیرامیٹرز میں ترمیم کریں
vim my-strategy.yaml

# کسٹم حکمت عملی استعمال کریں
python -m dark_forest.cli run --strategy ./my-strategy.yaml

سرکٹ بریکر پیرامیٹرز​

پیرامیٹرتفصیلڈیفالٹ
max_attacks_per_hourفی گھنٹہ زیادہ سے زیادہ حملے10
min_energy_reserveکم از کم توانائی کا ذخیرہ100,000
min_dft_reserveکم از کم DFT کا ذخیرہ10
max_upgrades_per_dayفی دن زیادہ سے زیادہ اپ گریڈ5

⚠️ سرکٹ بریکر پیرامیٹرز آن چین نافذ کیے جاتے ہیں۔ ایجنٹ انہیں نظرانداز نہیں کر سکتا۔
