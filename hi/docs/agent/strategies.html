रणनीति टेम्पलेट

एजेंट रणनीतियाँ YAML कॉन्फ़िगरेशन फ़ाइलों का उपयोग करके परिभाषित की जाती हैं।

रणनीति फ़ाइल संरचना​

agent:
  name: "my-agent"
  mode: "auto"          # auto | manual | dry-run

limits:                 # सर्किट ब्रेकर (ऑन-चेन नीति)
  min_energy_reserve: 100000
  min_dft_reserve: 10
  max_attacks_per_hour: 5
  max_upgrades_per_day: 3
  max_single_upgrade_cost: 100

combat:                 # युद्ध रणनीति
  min_target_level: 1
  max_target_level: 999
  weapon_level: 1
  only_attack_when_energy_above: 100000
  blacklist: []         # ब्लैकलिस्टेड पते
  whitelist: []         # व्हाइटलिस्टेड पते (केवल इन पर हमला करें)

defense:                # रक्षा रणनीति
  auto_repair: true
  min_durability_before_repair: 20
  shield_regen_threshold: 50   # इस % से नीचे ढाल पुनर्जनन शुरू करें

collection:             # संग्रह रणनीति
  enabled: true
  interval_minutes: 60

चार प्रीसेट रणनीतियाँ​

1. आक्रामक (Aggressive)​

combat:
  max_attacks_per_hour: 10
  min_target_level: 0.5
  weapon_level: max

collection:
  enabled: false  # कोई संग्रह नहीं, पूरी तरह युद्ध
  • ✅ अच्छा है: अत्यधिक सक्रिय खिलाड़ियों के लिए जो युद्ध पुरस्कार चाहते हैं
  • ❌ जोखिम: उच्च ऊर्जा खपत, जवाबी हमले का जोखिम

2. संतुलित (Balanced)​

combat:
  max_attacks_per_hour: 5
  min_target_level: 0.8
  weapon_level: balanced

collection:
  enabled: true
  interval_minutes: 30

defense:
  auto_repair: true
  shield_regen_threshold: 60
  • ✅ अनुशंसित: संतुलित हमला और रक्षा, अधिकांश खिलाड़ियों के लिए उपयुक्त

3. रक्षात्मक (Defensive)​

combat:
  max_attacks_per_hour: 1
  only_counterattack: true  # केवल जवाबी हमला

defense:
  auto_repair: true
  min_durability_before_repair: 50
  shield_regen_threshold: 80

collection:
  enabled: true
  interval_minutes: 15
  • ✅ अच्छा है: उन खिलाड़ियों के लिए जो लंबे समय तक ऑफलाइन रहते हैं और जोखिम से बचना चाहते हैं

4. किसान (Farmer)​

combat:
  enabled: false  # कोई युद्ध नहीं

defense:
  auto_repair: true
  shield_regen_threshold: 90

collection:
  enabled: true
  interval_minutes: 5
  • ✅ अच्छा है: शुरुआती-गेम विकास और संसाधन संचय के लिए

कस्टम रणनीति​

# टेम्पलेट कॉपी करें
cp templates/balanced.yaml my-strategy.yaml

# पैरामीटर संपादित करें
vim my-strategy.yaml

# कस्टम रणनीति का उपयोग करें
python -m dark_forest.cli run --strategy ./my-strategy.yaml

सर्किट ब्रेकर पैरामीटर​

पैरामीटरविवरणडिफ़ॉल्ट
max_attacks_per_hourप्रति घंटे अधिकतम हमले10
min_energy_reserveन्यूनतम ऊर्जा आरक्षित100,000
min_dft_reserveन्यूनतम DFT आरक्षित10
max_upgrades_per_dayप्रति दिन अधिकतम अपग्रेड5

⚠️ सर्किट ब्रेकर पैरामीटर ऑन-चेन लागू किए जाते हैं। एजेंट उन्हें बायपास नहीं कर सकता।
