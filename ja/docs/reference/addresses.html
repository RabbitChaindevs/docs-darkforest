コントラクトアーキテクチャ​

DarkForest（DELEGATECALL エントリー）
├── DELEGATECALL → DarkForestGameplay   （文明作成、システムアップグレード）
├── DELEGATECALL → DarkForestBattle      （攻撃、シールド修理、シールド回復）
├── DELEGATECALL → DarkForestMovement    （宇宙ジャンプ、追跡ジャンプ、巡航）
└── DELEGATECALL → DarkForestAdmin       （収集、修理、再建、トーテム、引き出し）

DarkForestStorage（共有ストレージレイアウト）
→ すべてのサテライトコントラクトは一貫したレイアウトのため DarkForestStorage を継承
